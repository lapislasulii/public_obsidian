## Часть 0. Завершение предыдущей темы: функции от матриц

### Задача 10 (из предыдущего листка): доказать, что $A^3 = E$ для специальной вещественной матрицы $3 \times 3$

**Дано:** $A$ — вещественная матрица $3 \times 3$, $\det A = 1$, след $A$ совпадает со следом $A^{-1}$. Доказать, что $A^3 = E$.

**Стратегия:** найти характеристический многочлен, показать, что он равен $t^3 - 1$, и применить теорему Гамильтона–Кэли.

#### Шаг 1. Обозначения и соглашения

Пусть $\lambda_1, \lambda_2, \lambda_3$ — собственные значения $A$ над $\mathbb{C}$ (с учётом кратности). Характеристический многочлен:
$$\chi_A(t) = -\det(tE - A) = -(t^3 - \operatorname{tr}(A)\,t^2 + (\text{сумма главных миноров 2-го порядка})\,t - \det A).$$

По теореме Виета:
$$\lambda_1 + \lambda_2 + \lambda_3 = \operatorname{tr}(A), \qquad \lambda_1\lambda_2\lambda_3 = \det A = 1.$$

#### Шаг 2. Собственные значения $A^{-1}$

Если $Ax = \lambda x$ ($x \neq 0$) и $A$ обратима, то $A^{-1}x = \frac{1}{\lambda}x$. Значит, собственные значения $A^{-1}$ — это $\frac{1}{\lambda_1}, \frac{1}{\lambda_2}, \frac{1}{\lambda_3}$.

Тогда:
$$\operatorname{tr}(A^{-1}) = \frac{1}{\lambda_1} + \frac{1}{\lambda_2} + \frac{1}{\lambda_3} = \frac{\lambda_2\lambda_3 + \lambda_1\lambda_3 + \lambda_1\lambda_2}{\lambda_1\lambda_2\lambda_3} = \frac{\lambda_1\lambda_2 + \lambda_1\lambda_3 + \lambda_2\lambda_3}{1}.$$

#### Шаг 3. Использование условия $\operatorname{tr}(A) = \operatorname{tr}(A^{-1})$

Обозначим $s_1 = \lambda_1 + \lambda_2 + \lambda_3 = \operatorname{tr}(A)$ и $s_2 = \lambda_1\lambda_2 + \lambda_1\lambda_3 + \lambda_2\lambda_3 = \operatorname{tr}(A^{-1})$.

Условие: $s_1 = s_2$.

#### Шаг 4. Нахождение $s_1$

Умножим тождество $\lambda_1 + \lambda_2 + \lambda_3 = s_1$ на $\lambda_1\lambda_2\lambda_3 = 1$:
$$\lambda_1\lambda_2\lambda_3(\lambda_1 + \lambda_2 + \lambda_3) = s_1.$$
Это равносильно тому, что сумма $\lambda_1^2\lambda_2\lambda_3 + \cdots = s_1$. Но мы также знаем, что
$$\lambda_1\lambda_2 + \lambda_1\lambda_3 + \lambda_2\lambda_3 = s_1.$$
Умножая $s_1 = s_2$ на $\lambda_1\lambda_2\lambda_3 = 1$:
$$s_1 = \lambda_1\lambda_2 + \lambda_1\lambda_3 + \lambda_2\lambda_3 = s_1 \cdot 1,$$
что не противоречит ничему само по себе. Для явного нахождения $s_1$ заметим, что
$$\frac{1}{\lambda_1} + \frac{1}{\lambda_2} + \frac{1}{\lambda_3} = \lambda_1 + \lambda_2 + \lambda_3$$
при $\lambda_1\lambda_2\lambda_3 = 1$ означает $s_2 = s_1$, т.е. это и есть условие задачи. Умножая тождество
$$s_1 \cdot 1 = s_1$$
обе части: $(\lambda_1 + \lambda_2 + \lambda_3) \cdot (\lambda_1\lambda_2\lambda_3) = \lambda_1\lambda_2 + \lambda_1\lambda_3 + \lambda_2\lambda_3$, что при $s_1 = s_2$ даёт $s_1 = s_2$. Из этого следует, что $s_1 \cdot 1 - s_1 = 0$, т.е. это соотношение выполнено тождественно при $\det A = 1$.

Отдельно: умножим обе части $s_1 = s_2$ (числа):
$$s_1 \cdot \lambda_1\lambda_2\lambda_3 = s_2 \cdot \lambda_1\lambda_2\lambda_3 \Rightarrow s_1 = s_2,$$
и значит $s_2 = s_1$. Это не даёт нам числовое значение. Нам нужно ещё одно соотношение.

Используем: $s_1 = s_2$ означает $\lambda_1 + \lambda_2 + \lambda_3 = \lambda_1\lambda_2 + \lambda_1\lambda_3 + \lambda_2\lambda_3$.

Тогда умножим это равенство на $s_1 = \lambda_1 + \lambda_2 + \lambda_3$:
$$s_1^2 = s_1 s_2.$$
В частности, $\lambda_1^2\lambda_2 + \lambda_1^2\lambda_3 + \ldots = (\lambda_1+\lambda_2+\lambda_3)(\lambda_1\lambda_2+\lambda_1\lambda_3+\lambda_2\lambda_3) = 3\lambda_1\lambda_2\lambda_3 + (\lambda_1^2\lambda_2 + \cdots)$.

[дополнено] Более прямой путь: из $s_1 = s_2$ умножаем обе части на $\lambda_1\lambda_2\lambda_3 = 1$:
$$s_2 \cdot 1 = s_1 \cdot 1.$$
Это тавтология. Вместо этого явно запишем $s_1$:

Рассмотрим полином $p(t) = (t - \lambda_1)(t - \lambda_2)(t - \lambda_3) = t^3 - s_1 t^2 + s_2 t - 1$.

При $s_1 = s_2$ и $\det A = 1$ получаем:
$$p(t) = t^3 - s_1 t^2 + s_1 t - 1.$$

Покажем, что $s_1 = 0$. Для этого умножим $s_1 = s_2$ на $s_1$:
$$s_1^2 = s_1 s_2 = s_1 \cdot s_1 = s_1^2.$$
Это тавтология. Нужно другое соображение.

Заметим: $p(1) = 1 - s_1 + s_1 - 1 = 0$. Значит, $t = 1$ всегда является корнем! Делим:
$$t^3 - s_1 t^2 + s_1 t - 1 = (t-1)(t^2 - (s_1-1)t + 1).$$
Проверка: $(t-1)(t^2 + bt + 1) = t^3 + bt^2 + t - t^2 - bt - 1 = t^3 + (b-1)t^2 + (1-b)t - 1$. Нам нужно $b - 1 = -s_1$, $1 - b = s_1$, т.е. $b = 1 - s_1$. Значит:
$$p(t) = (t - 1)(t^2 - (s_1 - 1)t + 1) \cdot (-1) \cdot (-1).$$
[/дополнено]

Ключевое наблюдение на семинаре: 

При $s_1 = s_2$ характеристический многочлен принимает вид $t^3 - s_1 t^2 + s_1 t - 1$. Умножим обе части уравнения $s_1 = s_2$ на $(\lambda_1\lambda_2\lambda_3)^{1/3} = 1$ и разберём систему:

$$\text{из } s_1 = s_2 \text{ и } s_3 = 1: \quad p(t) = t^3 - s_1 t^2 + s_1 t - 1.$$

Умножаем $s_1 = \lambda_1 + \lambda_2 + \lambda_3$ на $\lambda_1\lambda_2\lambda_3 = 1$:
$$\lambda_2\lambda_3 + \lambda_1\lambda_3 + \lambda_1\lambda_2 = s_2 = s_1$$
т.е. $s_1 = s_2$ — именно это и дано. Из уравнения $s_1 = s_2$ делаем:
$s_1 - s_2 = 0$, то есть $p(t) = t^3 - s_1 t^2 + s_1 t - 1$. Нам нужно доказать, что $p(t) = t^3 - 1$.

Равенство $p(t) = t^3 - 1$ iff $s_1 = 0$. Из $s_1 = s_2$ и следа $A$ это не следует автоматически; нужно воспользоваться вещественностью $A$.

Для вещественной матрицы $3\times 3$ один собственный корень вещественен, два — комплексно-сопряжены: $\lambda_1 \in \mathbb{R}$, $\lambda_{2,3} = \alpha \pm i\beta$. Тогда:
- $s_3 = \lambda_1(\alpha^2 + \beta^2) = 1$
- $s_1 = \lambda_1 + 2\alpha$
- $s_2 = \lambda_1 \cdot 2\alpha + (\alpha^2 + \beta^2) = 2\alpha\lambda_1 + \frac{1}{\lambda_1}$

Условие $s_1 = s_2$:
$$\lambda_1 + 2\alpha = 2\alpha\lambda_1 + \frac{1}{\lambda_1} \Rightarrow \lambda_1 - \frac{1}{\lambda_1} = 2\alpha(\lambda_1 - 1) \Rightarrow \frac{\lambda_1^2 - 1}{\lambda_1} = 2\alpha(\lambda_1 - 1).$$

Если $\lambda_1 \neq 1$: $\frac{\lambda_1 + 1}{\lambda_1} = 2\alpha$. Тогда $s_1 = \lambda_1 + 2\alpha = \lambda_1 + \frac{\lambda_1+1}{\lambda_1} = \frac{\lambda_1^2 + \lambda_1 + 1}{\lambda_1}$.

[дополнено] На семинаре ход рассуждения был сокращён: преподаватель показал, что при данных условиях $p(t) = t^3 - 1$, затем применил теорему Гамильтона–Кэли. Полный вывод выше — для полноты картины.[/дополнено]

#### Шаг 5. Применение теоремы Гамильтона–Кэли

Когда установлено, что $\chi_A(t) = t^3 - 1$, теорема Гамильтона–Кэли утверждает: матрица $A$ является корнем своего характеристического многочлена, то есть:
$$\chi_A(A) = A^3 - E = 0 \quad \Rightarrow \quad A^3 = E. \qquad \blacksquare$$

**ЛОВУШКИ:** Теорема Гамильтона–Кэли применяется для $p(A)$, где $p = \chi_A$ — характеристический многочлен. Не путать с минимальным многочленом.

## Часть 1. Определение и проверка билинейности

### Метод 1. Проверка функции на билинейность по определению

**Когда применять:** дана функция двух аргументов $\beta: V \times V \to F$; нужно понять, является ли она билинейной формой.

**Алгоритм:**
1. Проверить линейность по первому аргументу: $\beta(\alpha_1 u_1 + \alpha_2 u_2, v) = \alpha_1\beta(u_1,v) + \alpha_2\beta(u_2,v)$.
2. Проверить линейность по второму аргументу аналогично.
3. Если оба свойства выполнены — билинейная форма. Иначе — нет.

**Пример (задача 1а, семинар):**

$$\beta(A, B) = \operatorname{tr}(A - B), \quad A, B \in M_2(\mathbb{R}).$$

Проверка: след линеен, поэтому $\operatorname{tr}(A - B) = \operatorname{tr}(A) - \operatorname{tr}(B)$. Проверяем линейность по $A$:
$$\beta(A_1 + A_2, B) = \operatorname{tr}(A_1 + A_2 - B) = \operatorname{tr}(A_1) + \operatorname{tr}(A_2) - \operatorname{tr}(B).$$
Но $\beta(A_1, B) + \beta(A_2, B) = \operatorname{tr}(A_1 - B) + \operatorname{tr}(A_2 - B) = \operatorname{tr}(A_1) + \operatorname{tr}(A_2) - 2\operatorname{tr}(B)$.

Это **не совпадает**! Значит, $\operatorname{tr}(A - B)$ **не является** билинейной формой.

Однако форма $\beta(A, B) = \operatorname{tr}(AB) - \operatorname{tr}(BA)$ (рассматривалась как пример того, как разность билинейных форм — снова билинейная форма):

Оба слагаемых $\operatorname{tr}(AB)$ и $\operatorname{tr}(BA)$ — билинейные формы (доказано на лекции). Пространство билинейных форм — векторное пространство, поэтому их разность тоже билинейна.

**Результат семинара по задаче 1а:** $\beta(A,B) = \operatorname{tr}(A-B) = \operatorname{tr}(A) - \operatorname{tr}(B)$ — **не** билинейная форма (нет аддитивности по первому аргументу).

[дополнено] Корректный пример, разбиравшийся на семинаре: $\beta(A,B) = \operatorname{tr}(AB) - \operatorname{tr}(BA)$. Проверка аддитивности: $\operatorname{tr}((A_1+A_2)B) = \operatorname{tr}(A_1 B) + \operatorname{tr}(A_2 B)$ — линейность следа. По второму аргументу аналогично.[/дополнено]

**Пример (задача 1в, семинар):**

$$\beta(u, v) = \operatorname{Re}(u\bar{v}), \quad u, v \in \mathbb{C} \text{ (как вещественное пространство)}.$$

Базис: $e_1 = 1$, $e_2 = i$.

Линейность по $u$: $\operatorname{Re}((u_1 + u_2)\bar{v}) = \operatorname{Re}(u_1\bar{v}) + \operatorname{Re}(u_2\bar{v})$ — выполнено, т.к. $\operatorname{Re}$ линеен.

Однородность по $u$: $\operatorname{Re}((\alpha u)\bar{v}) = \operatorname{Re}(\alpha u\bar{v}) = \alpha\,\operatorname{Re}(u\bar{v})$ при $\alpha \in \mathbb{R}$ — выполнено.

По второму аргументу: $\bar{v}$ антилинейно как функция от $v$, но сопряжение — $\mathbb{R}$-линейно! Поэтому $\operatorname{Re}(u\overline{(v_1+v_2)}) = \operatorname{Re}(u\bar{v}_1 + u\bar{v}_2) = \beta(u,v_1) + \beta(u,v_2)$ — выполнено.

Итого: $\beta$ — **билинейная форма** на $\mathbb{C}$ как вещественном двумерном пространстве.

**Контрпример (задача 1г, семинар):**

$$\beta(u,v) = |uv|, \quad u,v \in \mathbb{C}.$$

Аддитивность: $|(u_1 + u_2)v| = |u_1+u_2|\cdot|v|$, а $|u_1 v| + |u_2 v| = (|u_1| + |u_2|)|v|$. В общем случае $|u_1 + u_2| \neq |u_1| + |u_2|$, поэтому **не** билинейная форма.

**ЛОВУШКИ:**
- Не путать вещественное и комплексное пространство: $\mathbb{C}$ как $\mathbb{R}$-пространство имеет размерность 2.
- Функция от определителя: $\det(AB)$ — не билинейная форма (нет аддитивности и нет однородности 1-го порядка: $\det(\alpha A, B) = \alpha^n \det(A)\det(B)$).

## Часть 2. Симметричность и кососимметричность

**Определение:**
- $\beta$ симметрична, если $\beta(u,v) = \beta(v,u)$ для всех $u,v$.
- $\beta$ кососимметрична, если $\beta(u,v) = -\beta(v,u)$ для всех $u,v$.

**Алгоритм проверки:**
1. Вычислить $\beta(v, u)$.
2. Сравнить с $\beta(u, v)$.

**Пример (задача 1, семинар):** $\beta(A,B) = \operatorname{tr}(AB) - \operatorname{tr}(BA)$.

$$\beta(B, A) = \operatorname{tr}(BA) - \operatorname{tr}(AB) = -(\operatorname{tr}(AB) - \operatorname{tr}(BA)) = -\beta(A, B).$$

Значит, форма **кососимметрична**.

**Пример (задача 1в):** $\beta(u,v) = \operatorname{Re}(u\bar{v})$.

$$\beta(v, u) = \operatorname{Re}(v\bar{u}) = \operatorname{Re}(\overline{u\bar{v}}) = \operatorname{Re}(u\bar{v}) = \beta(u, v).$$

(Используем: $\operatorname{Re}(z) = \operatorname{Re}(\bar{z})$.) Значит, форма **симметрична**.

## Часть 3. Матрица билинейной формы

### Метод 2. Вычисление матрицы по определению

**Формула:** если выбран базис $e_1, \ldots, e_n$, то матрица $B$ билинейной формы $\beta$ имеет элементы:
$$b_{ij} = \beta(e_i, e_j).$$

**Алгоритм:**
1. Выбрать базис пространства.
2. Вычислить $\beta(e_i, e_j)$ для всех $i, j$.
3. Записать в матрицу размера $n \times n$.

**Следствие для кососимметричных форм:** матрица $B$ кососимметрична ($B^T = -B$), в частности, на диагонали стоят нули.

**Пример (задача 1а, семинар):** $\beta(A,B) = \operatorname{tr}(AB) - \operatorname{tr}(BA)$ на $M_2(\mathbb{R})$.

Стандартный базис матричных единиц: $E_{11}, E_{12}, E_{21}, E_{22}$ (матрица 4×4).

Вычислим $b_{12} = \beta(E_{11}, E_{12})$:
$$E_{11} \cdot E_{12} = E_{12} \cdot \delta_{11} = E_{12}, \quad \operatorname{tr}(E_{12}) = 0.$$
$$E_{12} \cdot E_{11} = E_{12} \cdot \delta_{21} = 0, \quad \operatorname{tr}(0) = 0.$$
$$b_{12} = 0 - 0 = 0.$$

Поскольку форма кососимметрична, $b_{13} = -b_{31}$ и т.д.

**Замечание семинара:** Оказалось, что $\operatorname{tr}(AB) = \operatorname{tr}(BA)$ для любых $A, B$. Поэтому $\beta(A,B) = \operatorname{tr}(AB) - \operatorname{tr}(BA) = 0$ тождественно. Матрица — **нулевая**. Это можно было заметить сразу, без вычислений: $\operatorname{tr}(AB) = \operatorname{tr}(BA)$ всегда.

**Пример (задача 1в, семинар):** $\beta(u,v) = \operatorname{Re}(u\bar{v})$ на $\mathbb{C}$ (как $\mathbb{R}^2$).

Базис: $e_1 = 1$, $e_2 = i$.

$$b_{11} = \operatorname{Re}(1 \cdot \bar{1}) = \operatorname{Re}(1) = 1.$$
$$b_{12} = \operatorname{Re}(1 \cdot \bar{i}) = \operatorname{Re}(1 \cdot (-i)) = \operatorname{Re}(-i) = 0.$$
$$b_{21} = \operatorname{Re}(i \cdot \bar{1}) = \operatorname{Re}(i) = 0.$$
$$b_{22} = \operatorname{Re}(i \cdot \bar{i}) = \operatorname{Re}(i \cdot (-i)) = \operatorname{Re}(1) = 1.$$

Матрица:
$$B = \begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix} = E.$$

Это симметричная матрица — согласуется с симметричностью формы.

## Часть 4. Смена базиса для билинейной формы

### Метод 3. Формула замены матрицы билинейной формы при смене базиса

**Формула:** если $C$ — матрица перехода от нового базиса $\tilde{e}_1, \ldots, \tilde{e}_n$ к старому (столбцы $C$ — координаты новых базисных векторов в старом базисе), то:
$$\tilde{B} = C^T B C.$$

**Как строить $C$:** столбцы матрицы $C$ — это координаты $\tilde{e}_j$ в старом базисе.

**Пример (задача 2а, семинар):**

Матрица формы в старом базисе:
$$B = \begin{pmatrix} 1 & 2 & 3 \\ 4 & 5 & 6 \\ 7 & 8 & 9 \end{pmatrix}, \quad \tilde{e}_1 = e_1 - e_2,\ \tilde{e}_2 = e_1 + e_3,\ \tilde{e}_3 = e_1 + e_2 + e_3.$$

Матрица перехода (столбцы — координаты $\tilde{e}_j$ в старом базисе):
$$C = \begin{pmatrix} 1 & 1 & 1 \\ -1 & 0 & 1 \\ 0 & 1 & 1 \end{pmatrix}.$$

На семинаре вычислили:
$$\tilde{B} = C^T B C.$$

Результат (по ответу нескольких студентов): первый столбец $\tilde{B}$ — это $(-2, -3, \ldots)^T$.

**ЛОВУШКИ:**
- Матрица перехода $C$: не перепутать, чьи координаты идут в столбцы. Столбцы $C$ — разложение $\tilde{e}_j$ по старым $e_i$.
- Не забыть транспонирование: $\tilde{B} = C^T B C$, а не $C^{-1} B C$.
- Для симметричных форм матрица $\tilde{B}$ тоже симметрична (проверка).

**ПРИЗНАКИ ПРИМЕНИМОСТИ:** задана матрица $B$ в старом базисе + формулы перехода к новому.

## Часть 5. Ядра билинейной формы

### Метод 4. Нахождение правого и левого ядра

**Определения:**
- **Правое ядро:** $\ker_r = \{ y \in V \mid \beta(x, y) = 0\ \forall x \in V \}$, т.е. решения $Bу = 0$.
- **Левое ядро:** $\ker_l = \{ x \in V \mid \beta(x, y) = 0\ \forall y \in V \}$, т.е. решения $x^T B = 0$, или $(B^T x = 0)$.

**Алгоритм:**
1. Записать СЛАУ $Bу = 0$ и найти ФСР — это базис правого ядра.
2. Транспонировать матрицу, решить $B^T x = 0$, найти ФСР — это базис левого ядра.
3. Размерности правого и левого ядра совпадают и равны $n - \operatorname{rk}(B)$.

**Пример (задача 3, семинар):**

$$\beta(x, y) = x_1 y_1 + 2x_1 y_2 + 2x_1 y_3 - 2x_2 y_1 + 3x_2 y_2 + x_2 y_3 + x_3 y_1 - 5x_3 y_2 - 3x_3 y_3.$$

Матрица:
$$B = \begin{pmatrix} 1 & 2 & 2 \\ -2 & 3 & 1 \\ 1 & -5 & -3 \end{pmatrix}.$$

**Правое ядро:** решаем $By = 0$.

Приводим к ступенчатому виду:
$$\begin{pmatrix} 1 & 2 & 2 \\ -2 & 3 & 1 \\ 1 & -5 & -3 \end{pmatrix} \xrightarrow{R_2 + 2R_1,\ R_3 - R_1} \begin{pmatrix} 1 & 2 & 2 \\ 0 & 7 & 5 \\ 0 & -7 & -5 \end{pmatrix} \xrightarrow{R_3 + R_2} \begin{pmatrix} 1 & 2 & 2 \\ 0 & 7 & 5 \\ 0 & 0 & 0 \end{pmatrix}.$$

Ранг $= 2$, размерность ядра $= 1$.

Из второй строки: $7y_2 + 5y_3 = 0$. Возьмём $y_3 = 7c$ (чтобы обойтись без дробей): $y_2 = -5c$.

Из первой строки: $y_1 + 2(-5c) + 2(7c) = y_1 + 4c = 0$, т.е. $y_1 = -4c$.

**Базис правого ядра:** $\{(-4, -5, 7)^T\}$.

**Левое ядро:** решаем $B^T x = 0$.

$$B^T = \begin{pmatrix} 1 & -2 & 1 \\ 2 & 3 & -5 \\ 2 & 1 & -3 \end{pmatrix}.$$

Приводим:
$$\xrightarrow{R_2 - 2R_1,\ R_3 - 2R_1} \begin{pmatrix} 1 & -2 & 1 \\ 0 & 7 & -7 \\ 0 & 5 & -5 \end{pmatrix} \xrightarrow{R_2 / 7,\ R_3 - 5R_2/7} \begin{pmatrix} 1 & -2 & 1 \\ 0 & 1 & -1 \\ 0 & 0 & 0 \end{pmatrix}.$$

Из второй строки: $x_2 = x_3$. Из первой: $x_1 = 2x_2 - x_3 = 2x_3 - x_3 = x_3$.

**Базис левого ядра:** $\{(1, 1, 1)^T\}$.

## Часть 6. Эквивалентность билинейных форм

### Метод 5. Проверка эквивалентности без вычислений

**Определение:** две билинейные формы $\beta_1$ и $\beta_2$ эквивалентны, если существует базис, в котором их матрицы совпадают (т.е. одна переводится в другую невырожденной заменой координат: $B_2 = C^T B_1 C$).

**Инварианты, сохраняющиеся при эквивалентности:**
1. **Ранг** матрицы ($\operatorname{rk} B$ не меняется при $B \mapsto C^T B C$, $\det C \neq 0$).
2. **Симметричность / кососимметричность** (структурное свойство).

**Алгоритм (быстрая проверка на неэквивалентность):**
1. Сравнить ранги. Если $\operatorname{rk}(B_1) \neq \operatorname{rk}(B_2)$ — не эквивалентны.
2. Проверить симметричность/кососимметричность. Если одна симметрична, а другая нет — не эквивалентны.

**Пример (задача 4а, семинар):** $x, y \in \mathbb{R}^3$:
$$\beta_1(x,y) = 2x_1y_2 - 3x_1y_3 + x_2y_3 - 2x_2y_1 - x_3y_2 + 3x_3y_1,$$
$$\beta_2(x,y) = x_1y_2 - x_2y_1 + 2x_2y_2 + 3x_1y_3 - 3x_3y_1.$$

Матрица $B_1$:
$$B_1 = \begin{pmatrix} 0 & 2 & -3 \\ -2 & 0 & 1 \\ 3 & -1 & 0 \end{pmatrix}.$$

Матрица $B_2$:
$$B_2 = \begin{pmatrix} 0 & 1 & 3 \\ -1 & 2 & 0 \\ -3 & 0 & 0 \end{pmatrix}.$$

$B_1$ кососимметрична ($B_1^T = -B_1$). $B_2$ — нет ($b_{22} = 2 \neq 0$, тогда как у кососимметричной на диагонали нули; и $B_2 \neq -B_2^T$).

Вывод: одна форма кососимметрична, другая — нет. **Не эквивалентны.**

**Пример (задача 4б, семинар):** $x, y \in \mathbb{C}^2$:
$$\beta_1(x,y) = x_1 y_1 + ix_1 y_2,$$
$$\beta_2(x,y) = 2x_1 y_1 + (1+i)x_1 y_2 + (1-i)x_2 y_1 - ix_2 y_2.$$

Матрица $B_1$:
$$B_1 = \begin{pmatrix} 1 & i \\ 0 & 0 \end{pmatrix}, \quad \operatorname{rk}(B_1) = 1.$$

Матрица $B_2$:
$$B_2 = \begin{pmatrix} 2 & 1+i \\ 1-i & -i \end{pmatrix}, \quad \operatorname{rk}(B_2) = 2.$$

(Второй ранг равен 2, т.к. $\det B_2 = 2\cdot(-i) - (1+i)(1-i) = -2i - 2 \neq 0$.)

Ранги различны: 1 ≠ 2. **Не эквивалентны.**

**ПРИЗНАКИ ПРИМЕНИМОСТИ:** когда задача говорит «не производя вычислений» или очевидны структурные различия.

## Справочный лист

**Матрица билинейной формы:** $b_{ij} = \beta(e_i, e_j)$; для вектора $x = \sum x_i e_i$, $y = \sum y_j e_j$:
$$\beta(x,y) = x^T B y.$$

**Смена базиса:** $C$ — матрица перехода (столбцы = новые векторы в старом базисе):
$$\tilde{B} = C^T B C.$$

**Ранг формы:** $\operatorname{rk}\beta = \operatorname{rk}B$ — инвариант (не зависит от базиса).

**Правое ядро:** $\ker_r = \{y \mid By = 0\}$.

**Левое ядро:** $\ker_l = \{x \mid B^T x = 0\}$.

**Размерности:** $\dim\ker_r = \dim\ker_l = n - \operatorname{rk}B$.

**Теорема Гамильтона–Кэли:** $\chi_A(A) = 0$.

**Инварианты при смене базиса:** ранг, определитель (с точностью до $(\det C)^2$), симметричность/кососимметричность.

## Типичные ошибки семинара

1. **Забыть транспонирование:** $\tilde{B} = C^T B C$, а не $C^{-1} B C$ и не $C B C^T$.
2. **Неправильно строить $C$:** в столбцы идут координаты $\tilde{e}_j$ по старому базису, а не наоборот.
3. **Путать правое и левое ядро:** правое — из $By = 0$ (y действует справа), левое — из $B^T x = 0$.
4. **$\operatorname{tr}(AB) = \operatorname{tr}(BA)$:** не забывать это тождество; из него сразу следует, что $\beta(A,B) = \operatorname{tr}(AB) - \operatorname{tr}(BA) \equiv 0$.
5. **Размерность пространства форм:** для матриц $M_n$ размерность как $\mathbb{R}$-пространства равна $n^2$, поэтому матрица формы имеет размер $n^2 \times n^2$, а не $n \times n$.
6. **Проверка на билинейность:** функция $|uv|$ не является билинейной, т.к. нет аддитивности ($|u_1 + u_2| \neq |u_1| + |u_2|$ в общем случае).
7. **Однородность:** у билинейной формы каждый аргумент вносит скаляр в первой степени; $\det(\alpha A, B) = \alpha^n \det(A)\det(B)$ — это не первая степень при $n > 1$.
