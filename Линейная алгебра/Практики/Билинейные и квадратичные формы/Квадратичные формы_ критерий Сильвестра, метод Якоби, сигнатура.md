# Квадратичные формы: критерий Сильвестра, метод Якоби, сигнатура

## Обзор семинара

Семинар посвящён работе с вещественными квадратичными формами. Разбирались следующие классы задач: приведение квадратичной формы к каноническому виду методом Якоби (через угловые миноры); нахождение значений параметра, при которых форма является положительно или отрицательно определённой (критерий Сильвестра); теоретические задачи на доказательство неравенства для определителя положительно определённой матрицы; контрпримеры к утверждениям об определённости форм; задачи на вычисление сигнатуры квадратичной формы на пространстве матриц и пространстве многочленов.

## Методы и подходы

### 1. Метод Якоби (приведение к каноническому виду через угловые миноры)

**Когда применять:** квадратичная форма на вещественном пространстве, все угловые миноры $\Delta_1, \Delta_2, \ldots, \Delta_n$ матрицы формы ненулевые.

**Алгоритм:**

1. Выписать симметричную матрицу $B$ квадратичной формы $q(x) = \sum_{i,j} b_{ij} x_i x_j$, где $b_{ij} = b_{ji}$.
2. Вычислить угловые миноры:
$$\Delta_0 = 1, \quad \Delta_k = \det B_k,$$
где $B_k$ — левый верхний блок размера $k \times k$.
3. Канонический вид:
$$q = \sum_{k=1}^{n} \frac{\Delta_k}{\Delta_{k-1}}\, y_k^2.$$
4. Коэффициенты $\lambda_k = \dfrac{\Delta_k}{\Delta_{k-1}}$ — диагональные элементы в каноническом виде.

**Пример (Задача 11а):** $q = x_1^2 + 2x_2^2 - x_3^2 + 2x_1x_2$.

Матрица формы:
$$B = \begin{pmatrix} 1 & 1 & 0 \\ 1 & 2 & 0 \\ 0 & 0 & -1 \end{pmatrix}$$

Угловые миноры:
$$\Delta_1 = 1, \quad \Delta_2 = \det\begin{pmatrix}1&1\\1&2\end{pmatrix} = 2 - 1 = 1, \quad \Delta_3 = \det B.$$

Так как матрица блочно-треугольная (правый нижний блок $(-1)$ не связан с остальными):
$$\Delta_3 = \Delta_2 \cdot (-1) = -1.$$

Коэффициенты канонического вида:
$$\lambda_1 = \frac{\Delta_1}{\Delta_0} = 1, \quad \lambda_2 = \frac{\Delta_2}{\Delta_1} = 1, \quad \lambda_3 = \frac{\Delta_3}{\Delta_2} = -1.$$

Канонический вид: $q = y_1^2 + y_2^2 - y_3^2$. Сигнатура $(2,\,1)$.

**ЛОВУШКИ:**
- Не перепутать: $\Delta_0 = 1$ (не $0$), чтобы формула $\lambda_k = \Delta_k / \Delta_{k-1}$ работала при $k=1$.
- Если какой-то $\Delta_k = 0$, метод Якоби неприменим напрямую — нужно переупорядочивать переменные или использовать метод выделения полного квадрата.
- Матрица квадратичной формы симметрична: элемент $b_{ij}$ равен половине коэффициента при $x_i x_j$ (при $i \neq j$).

**ПРИЗНАКИ ПРИМЕНИМОСТИ:** Форма записана явно или дана её матрица; все угловые миноры ненулевые.

### 2. Критерий Сильвестра (определённость квадратичной формы)

**Когда применять:** нужно выяснить, является ли форма положительно или отрицательно определённой, в том числе при наличии параметра.

**Алгоритм для положительной определённости:**

1. Выписать симметричную матрицу $B$ формы.
2. Вычислить все угловые миноры $\Delta_1, \Delta_2, \ldots, \Delta_n$.
3. Форма положительно определена тогда и только тогда, когда все $\Delta_k > 0$.

**Алгоритм для отрицательной определённости:**

1. Вычислить угловые миноры $\Delta_1, \ldots, \Delta_n$.
2. Форма отрицательно определена тогда и только тогда, когда знаки чередуются, начиная с отрицательного:
$$\Delta_1 < 0, \quad \Delta_2 > 0, \quad \Delta_3 < 0, \quad \ldots$$
то есть $(-1)^k \Delta_k > 0$ для всех $k$.

**Пример (Задача 12а):** $q = 5x_1^2 + x_2^2 + \lambda x_3^2 + 4x_1x_2 - 2x_1x_3 - 2x_2x_3$.

Матрица формы:
$$B = \begin{pmatrix} 5 & 2 & -1 \\ 2 & 1 & -1 \\ -1 & -1 & \lambda \end{pmatrix}$$

Угловые миноры:
$$\Delta_1 = 5 > 0 \quad \checkmark$$
$$\Delta_2 = \det\begin{pmatrix}5&2\\2&1\end{pmatrix} = 5 - 4 = 1 > 0 \quad \checkmark$$

Для $\Delta_3$ раскладываем по третьей строке (или напрямую):
$$\Delta_3 = \det B = 5(1 \cdot \lambda - (-1)(-1)) - 2(2\lambda - (-1)(-1)) + (-1)(2(-1) - 1(-1))$$
$$= 5(\lambda - 1) - 2(2\lambda - 1) + (-1)(-2+1)$$
$$= 5\lambda - 5 - 4\lambda + 2 + 1 = \lambda - 2.$$

Условие положительной определённости: $\Delta_3 > 0$, то есть $\lambda > 2$.

**Пример (Задача 13б):** $q = \lambda x_1^2 - 2x_2^2 - 3x_3^2 + 2x_1x_2 - 2x_1x_3 + 2x_2x_3$.

Матрица:
$$B = \begin{pmatrix} \lambda & 1 & -1 \\ 1 & -2 & 1 \\ -1 & 1 & -3 \end{pmatrix}$$

Угловые миноры:
$$\Delta_1 = \lambda, \quad \Delta_2 = \det\begin{pmatrix}\lambda & 1 \\ 1 & -2\end{pmatrix} = -2\lambda - 1.$$

Условие отрицательной определённости:
$$\Delta_1 < 0 \Rightarrow \lambda < 0,$$
$$\Delta_2 > 0 \Rightarrow -2\lambda - 1 > 0 \Rightarrow \lambda < -\tfrac{1}{2}.$$

Вычислим $\Delta_3$. Раскладываем:
$$\Delta_3 = \lambda((-2)(-3) - 1 \cdot 1) - 1(1 \cdot (-3) - 1 \cdot (-1)) + (-1)(1 \cdot 1 - (-2)(-1))$$
$$= \lambda(6-1) - 1(-3+1) + (-1)(1-2)$$
$$= 5\lambda + 2 + 1 = 5\lambda + 3.$$

Условие $\Delta_3 < 0$: $5\lambda + 3 < 0 \Rightarrow \lambda < -\tfrac{3}{5}$.

Объединяем все три условия (самое жёсткое — $\lambda < -\tfrac{3}{5}$... но пересечение с $\lambda < -\tfrac{1}{2}$):

Пересечение: $\lambda < 0$, $\lambda < -\tfrac{1}{2}$, $\lambda < -\tfrac{3}{5}$.

Ответ: $\lambda < -\dfrac{3}{5}$.

Уточнение из транскрипта: преподаватель называл ответ $\lambda < -20$ для одного из вариантов задачи 13 — конкретные коэффициенты могут различаться в разных вариантах. Алгоритм во всех случаях одинаковый.

**ЛОВУШКИ:**
- При отрицательной определённости чередование начинается с $\Delta_1 < 0$, а не с $\Delta_1 > 0$.
- Знаки коэффициентов при $x_i x_j$ в матрице: коэффициент $b_{ij}$ при $i\neq j$ равен половине коэффициента при $x_i x_j$ в форме.
- Если форма содержит $\lambda$, неравенство для $\Delta_k$ может быть нелинейным по $\lambda$ — решать аккуратно.

**ПРИЗНАКИ ПРИМЕНИМОСТИ:** Нужно определить знак формы (положительная/отрицательная определённость) или найти параметр, при котором форма определённого знака.

### 3. Нахождение сигнатуры без приведения к каноническому виду

**Когда применять:** матрица формы имеет специальную структуру, позволяющую вычислить сигнатуру напрямую; или задача требует лишь сигнатуру, а не явную замену переменных.

**Идея:** сигнатура $(r_+, r_-)$ не меняется при конгруэнтных преобразованиях $B \mapsto C^T B C$. Если матрица формы приводится к диагональному виду с коэффициентами $\lambda_1, \ldots, \lambda_n$, то $r_+$ = количество положительных $\lambda_k$, $r_-$ = количество отрицательных.

**Метод для задачи 17* ($q(X) = \mathrm{tr}(X^2)$ на $M_n(\mathbb{R})$):**

Пространство $M_n(\mathbb{R})$ имеет размерность $n^2$. Выбираем базис из стандартных матричных единиц $E_{ij}$ в удобном порядке:

- Сначала все диагональные единицы $E_{11}, E_{22}, \ldots, E_{nn}$.
- Затем пары $(E_{ij} + E_{ji})/\sqrt{2}$ и $(E_{ij} - E_{ji})/\sqrt{2}$ для $i < j$.

Вычислим значения формы на базисных элементах:

На $E_{kk}$: $\mathrm{tr}(E_{kk}^2) = \mathrm{tr}(E_{kk}) = 1 > 0$.

На $E_{ij}$ при $i \neq j$: $\mathrm{tr}(E_{ij}^2) = \mathrm{tr}(E_{ij} E_{ij})$. Но $E_{ij} E_{ij} = 0$ (так как $j \neq i$), значит $\mathrm{tr}(E_{ij}^2) = 0$.

Поэтому матрица формы в базисе из $E_{ij}$ не диагональна для недиагональных единиц. Рассмотрим блоки, соответствующие паре $(E_{ij}, E_{ji})$:

Биилинейная форма $\beta(X,Y) = \mathrm{tr}(XY)$ (симметризованная). На паре $(E_{ij}, E_{ji})$:
$$\beta(E_{ij}, E_{ij}) = \mathrm{tr}(E_{ij}^2) = 0, \quad \beta(E_{ij}, E_{ji}) = \mathrm{tr}(E_{ij}E_{ji}) = \mathrm{tr}(E_{ii}) = 1.$$

Матрица ограничения на $\langle E_{ij}, E_{ji} \rangle$:
$$\begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}.$$

Замена: $F_1 = (E_{ij}+E_{ji})/\sqrt{2}$, $F_2 = (E_{ij}-E_{ji})/\sqrt{2}$.
$$q(F_1) = \mathrm{tr}\!\left(\frac{(E_{ij}+E_{ji})^2}{2}\right) = \frac{1}{2}\mathrm{tr}(E_{ij}E_{ji} + E_{ji}E_{ij}) = \frac{1}{2}(\mathrm{tr}(E_{ii})+\mathrm{tr}(E_{jj})) = 1.$$
$$q(F_2) = \mathrm{tr}\!\left(\frac{(E_{ij}-E_{ji})^2}{2}\right) = \frac{1}{2}\mathrm{tr}(-(E_{ij}E_{ji}+E_{ji}E_{ij})) = -1.$$

Итого: каждая пара $(i,j)$ с $i < j$ даёт один положительный и один отрицательный коэффициент. Пар таких $\binom{n}{2} = \dfrac{n(n-1)}{2}$.

Диагональные элементы дают $n$ положительных коэффициентов.

Сигнатура:
$$r_+ = n + \frac{n(n-1)}{2} = \frac{n(n+1)}{2}, \quad r_- = \frac{n(n-1)}{2}.$$

Проверка: $r_+ + r_- = n^2 = \dim M_n(\mathbb{R})$ — совпадает с размерностью пространства.

**Пример для задачи 18* ($q(f) = f(1) \cdot f(2)$ на $\mathbb{R}[x]_n$):**

Пространство $\mathbb{R}[x]_n$ — многочлены степени $\leq n-1$, размерность $n$.

Соответствующая билинейная форма: $\beta(f,g) = \frac{1}{2}(q(f+g)-q(f)-q(g)) = \frac{1}{2}((f+g)(1)(f+g)(2) - f(1)f(2) - g(1)g(2))$
$= \frac{1}{2}(f(1)g(2) + f(2)g(1))$.

Введём функционалы $F_1(f) = f(1)$ и $F_2(f) = f(2)$. Тогда:
$$q(f) = F_1(f) \cdot F_2(f).$$

Замена: $A = \dfrac{F_1+F_2}{2}$, $B = \dfrac{F_1-F_2}{2}$. Тогда $F_1 = A+B$, $F_2 = A-B$, и:
$$q(f) = (A(f)+B(f))(A(f)-B(f)) = A(f)^2 - B(f)^2.$$

Это канонический вид с одним положительным и одним отрицательным коэффициентом — при условии, что $A$ и $B$ линейно независимы как функционалы, что верно (так как $f(1)\neq f(2)$ в общем случае).

Ядро формы: $A(f) = 0$ и $B(f) = 0$ влечёт $f(1) = 0$ и $f(2) = 0$, то есть ядро — подпространство многочленов, обращающихся в нуль в обоих точках. Его размерность $n - 2$.

Сигнатура: $r_+ = 1$, $r_- = 1$, ранг $= 2$.

**ЛОВУШКИ:**
- На пространстве $M_n(\mathbb{R})$ не путать: $\mathrm{tr}(X^2)$ — это квадратичная форма, а не просто след.
- $E_{ij}^2 = 0$ при $i \neq j$, поэтому матрица формы в базисе матричных единиц не диагональна.

**ПРИЗНАКИ ПРИМЕНИМОСТИ:** Пространство имеет специальную структуру (матрицы, многочлены, функции), позволяющую выбрать базис, в котором форма принимает удобный вид.

### 4. Рассуждения о знаке угловых миноров и сигнатуре (Задача 19*)

**Постановка:** матрица $Q$ квадратичной формы $q$ на трёхмерном пространстве, $\Delta_1, \Delta_2, \Delta_3$ — угловые миноры. Какими могут быть индексы инерции?

**Алгоритм рассуждений:**

1. Если $\Delta_3 \neq 0$, то форма невырождена, значит $r_+ + r_- = 3$.
2. Перебираем возможные пары $(r_+, r_-)$ с суммой 3: $(0,3)$, $(1,2)$, $(2,1)$, $(3,0)$.
3. Исключаем пары, противоречащие знакам угловых миноров.

**Случай а) $\Delta_1 > 0$, $\Delta_2 = 0$, $\Delta_3 < 0$:**

$\Delta_3 < 0$ означает $\det Q < 0$, то есть произведение коэффициентов канонического вида $\lambda_1\lambda_2\lambda_3 < 0$. При $r_+ = 3$ произведение было бы положительным. При $r_- = 3$ произведение тоже было бы $<0$, но тогда критерий отрицательной определённости требует $\Delta_1 < 0$ — противоречие. Поэтому $(0,3)$ и $(3,0)$ исключаются.

Остаются $(1,2)$ и $(2,1)$. При $(2,1)$: произведение $\lambda_1\lambda_2\lambda_3 = (+)(+)(-) < 0$ — согласуется с $\Delta_3 < 0$. При $(1,2)$: произведение $(+)(-)(-) > 0$ — не согласуется. 

Ответ: $(r_+, r_-) = (2,1)$.

**Случай б) $\Delta_1 = 0$, $\Delta_2 = 0$, $\Delta_3 > 0$:**

$\Delta_1 = 0$ означает, что форма вырождена на первом шаге. $\Delta_3 > 0$ означает $\det Q > 0$.

Ранг формы может быть меньше 3. Рассматриваем $r_+ + r_- \leq 3$, при этом $\det Q = \Delta_3 > 0$. Если форма невырождена ($\text{rk}\, Q = 3$), то $\lambda_1\lambda_2\lambda_3 > 0$ при $(3,0)$ или $(1,2)$. Случай $(3,0)$: по критерию Сильвестра все $\Delta_k > 0$, но $\Delta_1 = 0$ — противоречие. Случай $(1,2)$: $\Delta_3 > 0$ согласуется.

Однако нулевые $\Delta_1, \Delta_2$ допускают и вырожденные случаи. Дополнительный анализ требует конкретного примера или более тонких аргументов.

**ЛОВУШКИ:**
- $\Delta_k = 0$ не означает вырожденность формы (вырожденность — это $\det Q = \Delta_n = 0$).
- Связь $\det Q = \lambda_1 \cdots \lambda_n$ верна при конгруэнтном приведении, но определитель меняется при конгруэнтном преобразовании как $\det(C^T B C) = (\det C)^2 \det B$, то есть знак $\det B$ сохраняется.

### 5. Доказательство неравенства для определителя (Задача 14)

**Утверждение:** если $B$ — матрица положительно определённой квадратичной формы, то $\det B \leq b_{11} b_{22} \cdots b_{nn}$.

**Доказательство (набросок из семинара):**

Положительно определённая форма симметрична и все угловые миноры положительны (критерий Сильвестра). По методу Якоби форма приводится к каноническому виду:
$$q = \sum_{k=1}^{n} \lambda_k y_k^2, \quad \lambda_k = \frac{\Delta_k}{\Delta_{k-1}} > 0.$$

Преобразование выполняется унитреугольной матрицей $C$ (с единицами на диагонали, $\det C = 1$):
$$C^T B C = \mathrm{diag}(\lambda_1, \ldots, \lambda_n).$$

Следовательно: $\det B = \lambda_1 \cdots \lambda_n$.

При каждом шаге метода Якоби диагональный элемент $\tilde{b}_{kk}$ после исключения удовлетворяет $\tilde{b}_{kk} \leq b_{kk}$ (исходный диагональный элемент уменьшается, так как вычитается неотрицательный квадрат). Формально: $\lambda_k = \tilde{b}_{kk}^{(k-1)} \leq b_{kk}$.

Поэтому $\det B = \prod_{k=1}^{n} \lambda_k \leq \prod_{k=1}^{n} b_{kk}$.

### 6. Контрпример к утверждению об определённости на прямой сумме (Задача 15)

**Утверждение, которое проверяется:** если $V = U \oplus W$ и $q|_U > 0$, $q|_W > 0$, следует ли $q > 0$ на $V$?

**Ответ: нет.**

**Контрпример из семинара:** возьмём $V = \mathbb{R}^2 = U \oplus W$, где $U = \langle e_1 \rangle$, $W = \langle e_2 \rangle$ (одномерные подпространства). Определим квадратичную форму с матрицей:
$$B = \begin{pmatrix} 1 & -3 \\ -3 & 1 \end{pmatrix}.$$

На $U$: $q(e_1) = 1 > 0$.
На $W$: $q(e_2) = 1 > 0$.

Проверим определённость на $V$: $\Delta_2 = 1 - 9 = -8 < 0$. Форма не является положительно определённой (и даже неопределённа).

Смысл: ограничения на подпространства прямой суммы контролируют форму лишь на координатных осях, а перекрёстные члены могут сделать форму отрицательной.

### 7. Контрпример к утверждению об угловых минорах (Задача 16)

**Утверждение, которое проверяется:** если все угловые миноры $\Delta_k \geq 0$, то форма неотрицательно определена?

**Ответ: нет.**

**Контрпример:** матрица $2 \times 2$:
$$B = \begin{pmatrix} 0 & 0 \\ 0 & -1 \end{pmatrix}.$$

Угловые миноры: $\Delta_1 = 0 \geq 0$, $\Delta_2 = 0 \cdot (-1) - 0 = 0 \geq 0$.

Форма: $q(x_1, x_2) = -x_2^2 \leq 0$ — это не неотрицательно определённая форма (принимает отрицательные значения).

Вывод: если хотя бы один угловой минор равен нулю, из критерия Сильвестра ничего нельзя заключить о знаке формы.

## Разбор задач

### Задача 11а: метод Якоби

$q = x_1^2 + 2x_2^2 - x_3^2 + 2x_1x_2$.

Матрица:
$$B = \begin{pmatrix} 1 & 1 & 0 \\ 1 & 2 & 0 \\ 0 & 0 & -1 \end{pmatrix}$$

$\Delta_1 = 1$, $\Delta_2 = 2 - 1 = 1$, $\Delta_3 = (-1)\cdot\Delta_2 = -1$ (блочная структура).

Коэффициенты: $\lambda_1 = 1$, $\lambda_2 = 1$, $\lambda_3 = -1$.

Канонический вид: $q = y_1^2 + y_2^2 - y_3^2$.

---

### Задача 12а: при каких $\lambda$ форма положительно определена

$q = 5x_1^2 + x_2^2 + \lambda x_3^2 + 4x_1x_2 - 2x_1x_3 - 2x_2x_3$.

Матрица:
$$B = \begin{pmatrix} 5 & 2 & -1 \\ 2 & 1 & -1 \\ -1 & -1 & \lambda \end{pmatrix}$$

$\Delta_1 = 5 > 0$.

$\Delta_2 = 5 \cdot 1 - 2 \cdot 2 = 1 > 0$.

$\Delta_3$: раскладываем по первой строке:
$$\Delta_3 = 5(1\cdot\lambda - (-1)(-1)) - 2(2\lambda - (-1)(-1)) + (-1)(2(-1) - 1(-1))$$
$$= 5(\lambda-1) - 2(2\lambda-1) + (-1)(-2+1)$$
$$= 5\lambda - 5 - 4\lambda + 2 + 1 = \lambda - 2.$$

Условие: $\lambda - 2 > 0$, то есть $\boxed{\lambda > 2}$.

---

### Задача 13б: при каких $\lambda$ форма отрицательно определена

$q = \lambda x_1^2 - 2x_2^2 - 3x_3^2 + 2x_1x_2 - 2x_1x_3 + 2x_2x_3$.

Матрица:
$$B = \begin{pmatrix} \lambda & 1 & -1 \\ 1 & -2 & 1 \\ -1 & 1 & -3 \end{pmatrix}$$

$\Delta_1 = \lambda < 0 \Rightarrow \lambda < 0$.

$\Delta_2 = -2\lambda - 1 > 0 \Rightarrow \lambda < -\tfrac{1}{2}$.

$\Delta_3$:
$$\Delta_3 = \lambda((-2)(-3) - 1\cdot 1) - 1(1\cdot(-3) - 1\cdot(-1)) + (-1)(1\cdot 1 - (-2)(-1))$$
$$= \lambda(6-1) - 1(-3+1) + (-1)(1-2)$$
$$= 5\lambda + 2 + 1 = 5\lambda + 3.$$

Условие $\Delta_3 < 0$: $5\lambda + 3 < 0 \Rightarrow \lambda < -\tfrac{3}{5}$.

Пересечение условий $\lambda < 0$, $\lambda < -\tfrac{1}{2}$, $\lambda < -\tfrac{3}{5}$:

Ответ: $\boxed{\lambda < -\tfrac{3}{5}}$.

Проверка: при $\lambda = -1$: $\Delta_1 = -1 < 0$, $\Delta_2 = -2(-1)-1=1>0$, $\Delta_3 = 5(-1)+3=-2<0$ — знаки чередуются, начиная с минуса. Верно.

## Справочный лист

**Матрица квадратичной формы** $q = \sum_{i,j} b_{ij} x_i x_j$:
- диагональные элементы: $b_{ii}$ = коэффициент при $x_i^2$
- внедиагональные: $b_{ij} = b_{ji}$ = половина коэффициента при $x_i x_j$

**Угловые миноры:** $\Delta_0 = 1$, $\Delta_k = \det B_k$ (левый верхний блок $k\times k$).

**Метод Якоби (все $\Delta_k \neq 0$):**
$$q = \sum_{k=1}^{n} \frac{\Delta_k}{\Delta_{k-1}} y_k^2$$

**Критерий Сильвестра:**
- $q$ положительно определена $\Leftrightarrow$ все $\Delta_k > 0$
- $q$ отрицательно определена $\Leftrightarrow$ $(-1)^k \Delta_k > 0$ для всех $k$

**Инвариантность:** $\det B$, сигнатура $(r_+, r_-)$, ранг — не меняются при конгруэнтных преобразованиях. Знак $\det B$ сохраняется (умножается на $(\det C)^2 > 0$).

**Связь определителя и коэффициентов:** $\det B = \Delta_n = \lambda_1 \cdots \lambda_n$ при приведении методом Якоби.

**Сигнатура $\mathrm{tr}(X^2)$ на $M_n(\mathbb{R})$:**
$$r_+ = \frac{n(n+1)}{2}, \quad r_- = \frac{n(n-1)}{2}.$$

## Типичные ошибки семинара

1. **Матрица формы**: коэффициент при $x_i x_j$ ($i\neq j$) делится на два при записи в матрицу. Пример: при $4x_1x_2$ в форме в матрице стоит $b_{12} = b_{21} = 2$.

2. **Знак $\Delta_0$**: $\Delta_0 = 1$, а не $0$ — необходимо для формулы $\lambda_1 = \Delta_1/\Delta_0$.

3. **Критерий отрицательной определённости**: чередование знаков начинается с $\Delta_1 < 0$, а не с $\Delta_1 > 0$.

4. **Нулевой угловой минор**: если $\Delta_k = 0$, метод Якоби неприменим; из неотрицательности $\Delta_k$ не следует неотрицательная определённость формы.

5. **Блочная матрица**: для блочно-диагональной матрицы $\det B = \det B_1 \cdot \det B_2$ — можно использовать для быстрого вычисления $\Delta_n$.

6. **Форма на специальном пространстве**: $q(X) = \mathrm{tr}(X^2)$ — это именно квадратичная форма (не просто след); $\mathrm{tr}(E_{ij}^2) = 0$ при $i \neq j$, что меняет вид матрицы.

7. **Прямая сумма**: положительная определённость на обоих слагаемых не гарантирует положительную определённость на всём пространстве — перекрёстные члены могут нарушить знак.
