## Методы и подходы

### 1. Вычисление функции от оператора методом остатка от деления на минимальный многочлен

**Когда применять:** оператор $A$ задан матрицей в ортонормированном базисе; требуется вычислить $f(A)$ для некоторой функции $f$ (корень, синус, экспонента и т.п.); хотим избежать явного диагонализирующего базиса и матрицы перехода.

**Алгоритм:**

1. Найти характеристический многочлен $\chi_A(t) = \det(A - tI)$.
2. Найти минимальный многочлен $\mu_A(t)$. Если оператор самосопряжён и все собственные значения простые, то $\mu_A = \chi_A$.
3. Записать остаток $r(t)$ от деления $f(t)$ на $\mu_A(t)$: если $\deg \mu_A = 2$, то $r(t) = at + b$.
4. Составить систему, приравнивая $r$ и $f$ на каждом собственном значении:
$$f(\lambda_i) = a\lambda_i + b \quad \text{для каждого } \lambda_i \in \operatorname{Spec}(A).$$
5. Решить систему, найти $a$ и $b$.
6. Вычислить $f(A) = aA + bI$.

**Обоснование.** По теореме Гамильтона–Кэли $\mu_A(A) = 0$, поэтому $f(A) = r(A)$ при условии совпадения $f$ и $r$ на спектре.

**Пример (задача 13а листка ИИИ\_24 — разбиралась на семинаре).**

> восстановлено: точная матрица из условия задачи 13а. Транскрипция содержит только промежуточные числа; полные данные восстановлены по листку задач.

Оператор задан матрицей $A = \begin{pmatrix} 2 & 5 \\ 1 & 10 \end{pmatrix}$ в ортонормированном базисе.

**Шаг 1.** $A^*A = A^TA = \begin{pmatrix}2&1\\5&10\end{pmatrix}\begin{pmatrix}2&5\\1&10\end{pmatrix} = \begin{pmatrix}5&20\\20&125\end{pmatrix}.$

**Шаг 2.** Характеристический многочлен: $\chi(t) = (t-25)(t-225)$; собственные значения $\lambda_1 = 25$, $\lambda_2 = 225$.

Согласно транскрипции, именно эти собственные значения были получены на семинаре.

**Шаг 3.** Ищем $R = \sqrt{A^*A}$, т.е. $f(t) = \sqrt{t}$. Остаток $r(t) = at + b$.

**Шаг 4–5.** Берём арифметические корни (т.к. $R$ должна быть положительно определённой):
$$\begin{cases} 25a + b = 5, \\ 225a + b = 15. \end{cases}$$

Вычитаем: $200a = 10 \Rightarrow a = \tfrac{1}{20}$.
Подставляем: $b = 5 - \tfrac{25}{20} = \tfrac{15}{4}$.

**Шаг 6.** $R = \tfrac{1}{20} A^TA + \tfrac{15}{4} I.$

По данным транскрипции, после вычислений получается матрица вида $\tfrac{1}{20}\begin{pmatrix}104&28\\28&296\end{pmatrix}$:
$$R = \frac{1}{20}\begin{pmatrix}104 & 28 \\ 28 & 296\end{pmatrix}.$$

**Шаг 7.** Находим $U = AR^{-1}$.

$\det(20R) = 104 \cdot 296 - 28^2 = 30784 - 784 = 30000$, поэтому $\det R = \tfrac{30000}{400} = 75$.

$$R^{-1} = \frac{1}{1500}\begin{pmatrix}296 & -28 \\ -28 & 104\end{pmatrix}.$$

(Главные элементы поменяли местами, побочные получили знак минус; общий знаменатель $= 400 \cdot 75 / 20 = 1500$.)

Матрица $U = AR^{-1}$ ортогональна: её столбцы образуют ортонормированную систему, $U^TU = I$.

По транскрипции преподаватель указал: ортогональная матрица $2\times2$ — это либо матрица поворота, либо суперпозиция поворота и отражения.

---

**Частный случай: диагональная матрица (задача 13б).**

Если $A^*A$ диагональна, $R$ тоже диагональна: корни извлекаются поэлементно.

По транскрипции, в пункте б получились собственные значения $\lambda_1 = 2$, $\lambda_2 = 8$:
$$R = \begin{pmatrix}\sqrt{2}&0\\0&2\sqrt{2}\end{pmatrix}, \qquad R^{-1} = \begin{pmatrix}\tfrac{1}{\sqrt{2}}&0\\0&\tfrac{1}{2\sqrt{2}}\end{pmatrix}.$$

Обратная к диагональной матрице — диагональная с обратными элементами на диагонали.

**ЛОВУШКИ:**

- При извлечении корня берут только арифметический (положительный) корень, иначе $R$ не будет положительно определённой.
- Если из $R$ вынесен множитель $\tfrac{1}{k}$, определитель уменьшается в $k^n$ раз — необходимо правильно вычислить $\det R$.
- При записи $R^{-1}$ для матрицы $2\times2$: главные элементы меняются местами, побочные — меняют знак, и всё делится на $\det R$.

**ПРИЗНАКИ ПРИМЕНИМОСТИ:**

- Задан оператор в ортонормированном базисе, требуется полярное разложение $A = UR$.
- Характеристический многочлен $A^*A$ имеет небольшую степень, собственные значения вычислимы вручную.
- Хочется избежать нахождения собственного базиса и матрицы перехода.

### 2. Вывод канонического уравнения эллипса методом последовательного возведения в квадрат

**Когда применять:** кривая задана как геометрическое место точек через расстояния, нужно вывести её алгебраическое уравнение.

**Алгоритм:**

1. Ввести координатную систему: начало $O$ в центре, фокусы $F_1(-c,0)$, $F_2(c,0)$.
2. Записать определяющее условие в координатах: $|F_1X| + |F_2X| = 2a$.
3. Перенести один корень в правую часть.
4. Возвести в квадрат (оба члена неотрицательны — модуль не нужен).
5. Упростить, оставить корень в одной части.
6. Снова возвести в квадрат.
7. Привести подобные слагаемые, перегруппировать по $x^2$ и $y^2$, ввести $b^2 = a^2 - c^2$.
8. Разделить на $a^2 b^2$.
9. Проверить: убедиться, что посторонних точек нет.

**Пример (задача 1 листка ИИИ\_25, полностью разобрана на семинаре).**

Фокусы $F_1(-c,0)$, $F_2(c,0)$; условие: $|F_1X| + |F_2X| = 2a$, $a > c > 0$, $b^2 = a^2 - c^2 > 0$.

**Шаг 1.** Точка $X = (x, y)$:
$$\sqrt{(x+c)^2+y^2} + \sqrt{(x-c)^2+y^2} = 2a.$$

**Шаг 2.** Переносим:
$$\sqrt{(x+c)^2+y^2} = 2a - \sqrt{(x-c)^2+y^2}.$$

**Шаг 3.** Возводим в квадрат:
$$(x+c)^2 + y^2 = 4a^2 - 4a\sqrt{(x-c)^2+y^2} + (x-c)^2 + y^2.$$

Раскрываем и сокращаем $y^2$, $x^2$, $c^2$:
$$4cx = 4a^2 - 4a\sqrt{(x-c)^2+y^2}.$$

Делим на $4$ и собираем корень:
$$a\sqrt{(x-c)^2+y^2} = a^2 - cx.$$

**Шаг 4.** Возводим в квадрат снова:
$$a^2\bigl((x-c)^2+y^2\bigr) = (a^2-cx)^2.$$

Раскрываем:
$$a^2x^2 - 2a^2cx + a^2c^2 + a^2y^2 = a^4 - 2a^2cx + c^2x^2.$$

Члены $-2a^2cx$ сокращаются:
$$x^2(a^2-c^2) + a^2y^2 = a^2(a^2-c^2).$$
$$b^2x^2 + a^2y^2 = a^2b^2.$$

**Шаг 5.** Делим на $a^2b^2$:
$$\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1.$$

**Шаг 6 — проверка отсутствия посторонних точек.** Берём произвольную точку $(x,y)$ на кривой. Из уравнения: $y^2 = \tfrac{b^2(a^2-x^2)}{a^2}$.

Вычисляем $|F_2X|^2 = (x-c)^2 + y^2$:
$$(x-c)^2 + \frac{b^2(a^2-x^2)}{a^2} = x^2 - 2cx + c^2 + (a^2-c^2) - \frac{(a^2-c^2)x^2}{a^2} = \frac{c^2x^2}{a^2} - 2cx + a^2 = \left(\frac{cx}{a}-a\right)^2.$$

Поэтому $|F_2X| = \left|\varepsilon x - a\right|$, где $\varepsilon = c/a$.

Поскольку $x \in [-a, a]$ и $\varepsilon < 1$, имеем $\varepsilon x < a$, т.е. $\varepsilon x - a < 0$:
$$|F_2X| = a - \varepsilon x.$$

Аналогично $|F_1X| = a + \varepsilon x$. Тогда:
$$|F_1X| + |F_2X| = 2a. \quad \checkmark$$

Посторонних точек нет. Уравнение $\tfrac{x^2}{a^2} + \tfrac{y^2}{b^2} = 1$ — в точности каноническое уравнение эллипса.

**Анализ эксцентриситета $\varepsilon = c/a$.**

- Всегда $\varepsilon \in (0,1)$ (так как $0 < c < a$).
- При $\varepsilon \to 0$: $c \to 0$, $b \to a$ — эллипс вырождается в **окружность**.
- При $\varepsilon \to 1$: $c \to a$, $b \to 0$ — эллипс вырождается в **отрезок**.
- $\varepsilon$ — мера сплющенности: чем ближе к $1$, тем сильнее вытянут эллипс.

**ЛОВУШКИ:**

- Первое возведение в квадрат законно, так как сумма неотрицательных чисел остаётся неотрицательной.
- На шаге $a\sqrt{\ldots} = a^2 - cx$ правая часть должна быть неотрицательной — это выполняется для всех точек настоящего эллипса, но требует проверки при «обратном» шаге.
- Для гиперболы: $b^2 = c^2 - a^2 > 0$, знак при $y^2$ меняется.

### 3. Доказательство директориального свойства эллипса

**Когда применять:** нужно показать, что отношение расстояния от точки эллипса до фокуса к расстоянию до соответствующей директрисы постоянно и равно $\varepsilon$.

**Алгоритм:**

1. Записать директрисы: $x = \pm \tfrac{a}{\varepsilon}$.
2. Выписать фокальные радиусы: $|F_1X| = a + \varepsilon x$, $|F_2X| = a - \varepsilon x$ (из задачи 1).
3. Вычислить расстояние до соответствующей директрисы.
4. Разделить фокальный радиус на это расстояние и упростить.

**Пример (задача 2 листка ИИИ\_25, разобрана на семинаре).**

Рассмотрим правый фокус $F_2(c,0)$ и правую директрису $x = \tfrac{a}{\varepsilon}$.

Точка $(x,y)$ на эллипсе. Поскольку $x \leq a$ и $a < \tfrac{a}{\varepsilon}$ (т.к. $\varepsilon < 1$):
$$d = \frac{a}{\varepsilon} - x > 0.$$

Фокальный радиус: $|F_2X| = a - \varepsilon x$.

Отношение:
$$\frac{|F_2X|}{d} = \frac{a - \varepsilon x}{\dfrac{a}{\varepsilon} - x} = \frac{a - \varepsilon x}{\dfrac{a - \varepsilon x}{\varepsilon}} = \varepsilon. \quad \checkmark$$

Для левого фокуса и левой директрисы: аналогично, $\tfrac{|F_1X|}{x + a/\varepsilon} = \varepsilon$.

**Следствие.** Альтернативное определение эллипса: ГМТ, для которых отношение расстояния до фиксированной точки (фокуса) к расстоянию до фиксированной прямой (директрисы) постоянно и равно $\varepsilon \in (0,1)$. При $\varepsilon = 1$ — парабола, при $\varepsilon > 1$ — гипербола.

**ЛОВУШКИ:**

- «Соответствующая директриса»: правому фокусу — правая директриса, левому — левая.
- При раскрытии выражения $\tfrac{a}{\varepsilon} - x$ убедиться в его положительности: $x \leq a$, $\tfrac{a}{\varepsilon} > a$ при $\varepsilon < 1$.

### 4. Теорема о конических сечениях и сферы Данделена

**Когда применять:** нужно доказать, почему эллипс, гипербола и парабола называются коническими сечениями, и объяснить геометрический смысл директрис.

**Конструкция (задача 7 листка ИИИ\_25, разобрана на семинаре).**

Рассматриваем прямой круговой конус. Сечём его плоскостью $\alpha$ (не через вершину, не параллельно образующей и не параллельно оси).

Вписываем **две сферы Данделена** $S_1$ и $S_2$:
- каждая касается конуса вдоль окружности ($\omega_1$ и $\omega_2$),
- $S_1$ касается $\alpha$ в точке $F_1$, $S_2$ касается $\alpha$ в точке $F_2$.

Для произвольной точки $X$ сечения проводим образующую конуса через $X$. Она пересекает $\omega_1$ в $P_1$ и $\omega_2$ в $P_2$.

**Ключевое равенство.** Из $X$ проведены касательные к сфере $S_1$: $XF_1$ (через точку касания с $\alpha$) и $XP_1$ (вдоль образующей). Все касательные из одной точки к сфере равны по длине:
$$|XF_1| = |XP_1|, \qquad |XF_2| = |XP_2|.$$

Следовательно:
$$|XF_1| + |XF_2| = |XP_1| + |XP_2| = |P_1P_2|.$$

$|P_1P_2|$ — длина отрезка образующей между $\omega_1$ и $\omega_2$, **постоянна** для всех образующих. Это в точности определение эллипса с фокусами $F_1$, $F_2$.

**Директрисы.** Плоскости, проходящие через $\omega_1$ и $\omega_2$, пересекают плоскость $\alpha$ по двум прямым — это директрисы эллипса.

**Остальные случаи:**
- Плоскость параллельна одной образующей $\Rightarrow$ **парабола**.
- Плоскость пересекает обе полости конуса $\Rightarrow$ **гипербола** (разность $|XF_1| - |XF_2|$ постоянна, асимптоты совпадают с образующими).

Подробная иллюстрация: https://etudes.ru/etudes/Dandelin-spheres/

**ЛОВУШКИ:**

- Конус рассматривается как двуполостный.
- Случаи «плоскость через вершину» и «плоскость параллельна основанию» исключены из теоремы.

### Задача 12 (листок ИИИ\_24): Матрица поворота и сопряжённый оператор

**По транскрипции.** В начале семинара разбиралась задача 12 — нахождение сопряжённого оператора к повороту на угол $\varphi$ евклидовой плоскости в ортонормированном базисе $e_1, e_2$.

Матрица поворота на угол $\varphi = \pi/2$ (след = $2\cos\varphi + 1 = 1$ при $\varphi = \pi/2$):
$$A_\varphi = \begin{pmatrix} 0 & -1 \\ 1 & 0 \end{pmatrix}.$$

Для ортонормированного базиса матрица сопряжённого оператора $A^*$ совпадает с транспонированной: $A^* = A^T$.

Связь матриц оператора и сопряжённого в произвольном (не ортонормированном) базисе:
$$[A^*] = G^{-1}[A]^T G,$$
где $G$ — матрица Грама.

**Принцип записи строк матрицы оператора в ортонормированном базисе:**

Если $(e_1^*, e_2^*)$ — сопряжённый базис (для ортонормированного $e_i^* = e_i$ по скалярному произведению), то $j$-я строка матрицы $A$ — это координаты образа $A e_j$ в базисе, т.е. $[A]_{ij} = \langle e_i, Ae_j \rangle$.

## Справочный лист

### Эллипс

$$\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1, \quad a > b > 0, \quad b^2 = a^2 - c^2.$$

| Элемент | Формула |
|---|---|
| Фокусы | $(\pm c,\, 0)$ |
| Эксцентриситет | $\varepsilon = c/a \in (0,1)$ |
| Директрисы | $x = \pm\, a/\varepsilon$ |
| Фокальные радиусы | $r_{1,2} = a \pm \varepsilon x$ |
| Директориальное свойство | $\|FX\| / d(X,\, \text{дир.}) = \varepsilon$ |
| Вершины | $(\pm a,0)$ и $(0, \pm b)$ |
| Сумма фокальных радиусов | $r_1 + r_2 = 2a$ |

### Гипербола (дом. задание 3–5)

$$\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1, \quad b^2 = c^2 - a^2, \quad \varepsilon = c/a > 1.$$

Асимптоты: $y = \pm\tfrac{b}{a}\,x$.

### Парабола (дом. задание 4–5)

$$y^2 = 2px, \quad F = (p/2,\, 0), \quad \text{директриса } x = -p/2, \quad \varepsilon = 1.$$

### Полярное разложение

$$A = UR, \quad R = \sqrt{A^*A} \geq 0, \quad U = AR^{-1} \text{ (ортог.)}.$$

Метод остатка ($\deg \mu_A = 2$, $f(t) = \sqrt{t}$):
$$r(t) = at + b, \quad \begin{cases} f(\lambda_1) = a\lambda_1 + b, \\ f(\lambda_2) = a\lambda_2 + b, \end{cases} \quad R = aA^*A + bI.$$

Матрица сопряжённого в произвольном базисе: $[A^*] = G^{-1}[A]^T G$.

В ортонормированном базисе: $[A^*] = [A]^T$.

## Типичные ошибки семинара

1. **Знак корня.** Для положительно определённого $R$ берут только арифметические корни из собственных значений $A^*A$.

2. **Потеря множителя при определителе.** Если из $R$ вынесен $1/k$, то $\det R$ уменьшается в $k^n$ раз; $R^{-1}$ надо умножить на $k^n$, а не на $k$.

3. **Формула $R^{-1}$ для $2\times 2$.** Главные элементы меняются местами, побочные — меняют знак, всё делится на $\det R$.

4. **Посторонние точки при возведении в квадрат.** Обязательна проверка (шаг 9 алгоритма вывода уравнения эллипса).

5. **$b^2$ для эллипса и гиперболы.** Для эллипса $b^2 = a^2 - c^2 > 0$; для гиперболы $b^2 = c^2 - a^2 > 0$.

6. **Директриса и фокус.** Правому фокусу соответствует правая директриса; смешение приводит к неверному знаменателю дроби.

7. **Диагональный случай.** Если $A^*A$ диагональна, нет нужды в методе остатка — корни на диагонали извлекаются поэлементно.
