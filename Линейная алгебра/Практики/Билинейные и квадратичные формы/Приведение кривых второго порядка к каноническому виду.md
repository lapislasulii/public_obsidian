# Приведение кривых второго порядка к каноническому виду

## Обзор семинара

Семинар посвящён задаче 9 из листка ИИИ.25 — приведению кривых второго порядка, заданных в прямоугольной декартовой системе координат, к каноническому виду. Отрабатываются три случая: эллипс (9а), гипербола (9в), парабола (9б). Метод во всех задачах единый: выделить квадратичную часть уравнения, найти собственные значения и собственные векторы соответствующего самосопряжённого оператора, совершить ортогональный поворот, после чего выделением полных квадратов найти центр (или вершину для параболы) и привести уравнение к каноническому виду. Затем в канонической системе находятся полуоси, фокусы, директрисы, эксцентриситет и всё это пересчитывается в исходную систему координат.

---

## Методы и подходы

### 1. Метод ортогонального приведения квадратичной формы к каноническому виду

**Когда применять:** кривая второго порядка задана уравнением $f(x,y) = 0$, содержащим смешанный член $xy$. Необходимо устранить смешанный член путём поворота системы координат.

**Алгоритм:**

1. Выписать матрицу квадратичной части: если квадратичная часть есть $Ax^2 + 2Bxy + Cy^2$, то матрица $Q = \begin{pmatrix} A & B \\ B & C \end{pmatrix}$.
2. Найти собственные значения $\lambda_1, \lambda_2$ из характеристического уравнения $\det(Q - \lambda I) = 0$.
3. Для каждого $\lambda_i$ найти собственный вектор $v_i$ из $(Q - \lambda_i I)v = 0$.
4. Нормировать собственные векторы: $e_i = v_i / \|v_i\|$.
5. Составить матрицу перехода $T = (e_1 \mid e_2)$ (столбцы — нормированные собственные векторы). Матрица $T$ ортогональна, $T^{-1} = T^{\top}$.
6. Выразить старые координаты через новые: $\begin{pmatrix} x \\ y \end{pmatrix} = T \begin{pmatrix} \tilde{x} \\ \tilde{y} \end{pmatrix}$.
7. Подставить в уравнение кривой. Смешанный член исчезнет. Получится уравнение вида $\lambda_1 \tilde{x}^2 + \lambda_2 \tilde{y}^2 + \text{линейные члены} + \text{const} = 0$.
8. Выделить полные квадраты по $\tilde{x}$ и $\tilde{y}$ (если $\lambda_i \ne 0$).
9. Заменой $\tilde{\tilde{x}} = \tilde{x} - x_0$, $\tilde{\tilde{y}} = \tilde{y} - y_0$ (параллельный перенос) убрать линейные члены. Получить каноническое уравнение.
10. Определить тип кривой и её характеристики в канонической системе.
11. Пересчитать фокусы и директрисы в исходную систему координат.

**ЛОВУШКИ:**
- Матрица $Q$ должна быть симметричной: коэффициент при $xy$ делится на 2 (так как $\beta(x,y) = B\cdot x_1 y_2 + B\cdot x_2 y_1$, поэтому в матрице стоит $B$, а не $2B$).
- При нормировке собственных векторов легко перепутать знаки; нужно проверить, что $\det T = +1$ (правая тройка) или допустим $-1$ (тоже ортогональная матрица, просто отражение, но при вещественном случае это не страшно).
- Собственные значения различны $\Rightarrow$ собственные векторы ортогональны автоматически (теорема о самосопряжённом операторе в евклидовом пространстве).
- Один из собственных значений равен 0 — кривая не центральная (парабола); в этом случае полный квадрат по соответствующей переменной выделить нельзя, и линейный член этой переменной задаёт «параболическое направление».

**ПРИЗНАКИ ПРИМЕНИМОСТИ:** в уравнении кривой второго порядка присутствует смешанный член $xy$.

### 2. Выделение полных квадратов и параллельный перенос

**Когда применять:** после поворота в уравнении остались линейные члены; нужно найти центр кривой (для эллипса/гиперболы) или вершину (для параболы).

**Алгоритм (центральные кривые: эллипс, гипербола):**

1. После поворота уравнение имеет вид $\lambda_1 \tilde{x}^2 + \lambda_2 \tilde{y}^2 + p\tilde{x} + q\tilde{y} + r = 0$ (оба $\lambda_i \ne 0$).
2. Вынести $\lambda_1$ за скобку при $\tilde{x}$: $\lambda_1\!\left(\tilde{x} + \frac{p}{2\lambda_1}\right)^2 - \frac{p^2}{4\lambda_1} + \ldots$
3. Вынести $\lambda_2$ за скобку при $\tilde{y}$ аналогично.
4. Ввести $\tilde{\tilde{x}} = \tilde{x} + \frac{p}{2\lambda_1}$, $\tilde{\tilde{y}} = \tilde{y} + \frac{q}{2\lambda_2}$.
5. Привести уравнение к виду $\lambda_1 \tilde{\tilde{x}}^2 + \lambda_2 \tilde{\tilde{y}}^2 = C$ и разделить на $C$.

**Алгоритм (парабола, одно $\lambda_i = 0$):**

1. После поворота: $0 \cdot \tilde{x}^2 + \lambda_2 \tilde{y}^2 + p\tilde{x} + q\tilde{y} + r = 0$.
2. Выделить полный квадрат только по $\tilde{y}$: $\lambda_2\!\left(\tilde{y} + \frac{q}{2\lambda_2}\right)^2 - \frac{q^2}{4\lambda_2} + p\tilde{x} + r = 0$.
3. Ввести $\tilde{\tilde{y}} = \tilde{y} + \frac{q}{2\lambda_2}$.
4. Уравнение принимает вид $\lambda_2 \tilde{\tilde{y}}^2 + p\tilde{x} + r' = 0$, откуда $\tilde{\tilde{y}}^2 = -\frac{p}{\lambda_2}\tilde{x} - \frac{r'}{\lambda_2}$.
5. Ввести $\tilde{\tilde{x}} = \tilde{x} + \frac{r'}{p}$ (ещё один параллельный перенос). Получить каноническое уравнение параболы $\tilde{\tilde{y}}^2 = 2p_{\text{фок}} \tilde{\tilde{x}}$.

**ЛОВУШКИ:**
- Слагаемое, которое «выпрыгивает» при выделении полного квадрата, надо вычитать из правой части.
- Для параболы выделение полного квадрата по $\tilde{x}$ невозможно; «лишний» линейный член по $\tilde{x}$ убирается вторым параллельным переносом.

### 3. Нахождение характеристик кривых в канонической и исходной системах координат

**Эллипс** $\dfrac{\tilde{\tilde{x}}^2}{a^2} + \dfrac{\tilde{\tilde{y}}^2}{b^2} = 1$, $a > b > 0$:
- $c^2 = a^2 - b^2$, $c > 0$
- Фокусы в канонической системе: $(\pm c,\, 0)$
- Директрисы: $\tilde{\tilde{x}} = \pm \dfrac{a}{\varepsilon}$, где $\varepsilon = \dfrac{c}{a} < 1$
- Эксцентриситет: $\varepsilon = \dfrac{c}{a}$

**Гипербола** $\dfrac{\tilde{\tilde{x}}^2}{a^2} - \dfrac{\tilde{\tilde{y}}^2}{b^2} = 1$:
- $c^2 = a^2 + b^2$
- Фокусы: $(\pm c,\, 0)$
- Эксцентриситет: $\varepsilon = \dfrac{c}{a} > 1$
- Асимптоты: $\tilde{\tilde{y}} = \pm\dfrac{b}{a}\tilde{\tilde{x}}$

**Парабола** $\tilde{\tilde{y}}^2 = 2p_{\text{фок}}\tilde{\tilde{x}}$:
- Фокус: $\left(\dfrac{p_{\text{фок}}}{2},\, 0\right)$
- Директриса: $\tilde{\tilde{x}} = -\dfrac{p_{\text{фок}}}{2}$

**Пересчёт в исходную систему:**

Если $\begin{pmatrix} x \\ y \end{pmatrix} = T\begin{pmatrix} \tilde{x} \\ \tilde{y} \end{pmatrix}$ (поворот) и $\begin{pmatrix} \tilde{x} \\ \tilde{y} \end{pmatrix} = \begin{pmatrix} \tilde{\tilde{x}} \\ \tilde{\tilde{y}} \end{pmatrix} + \begin{pmatrix} x_0 \\ y_0 \end{pmatrix}$ (перенос), то фокус с координатами $(f_1, f_2)$ в канонической системе имеет в исходной системе координаты $T\left(\begin{pmatrix} f_1 \\ f_2 \end{pmatrix} + \begin{pmatrix} x_0 \\ y_0 \end{pmatrix}\right)$.

Директриса $\tilde{\tilde{x}} = d$ в канонической системе — это прямая. Чтобы записать её уравнение в исходной системе, нужно подставить $\tilde{x} = d + x_0$ (и произвольное $\tilde{y}$) в выражение $\begin{pmatrix} x \\ y \end{pmatrix} = T\begin{pmatrix} \tilde{x} \\ \tilde{y} \end{pmatrix}$, исключить $\tilde{y}$ и получить линейное уравнение на $x, y$.

## Разбор задач

### Задача 9а: $9x^2 - 4xy + 6y^2 + 16x - 8y - 2 = 0$

#### Шаг 1. Матрица квадратичной части

Квадратичная часть: $9x^2 - 4xy + 6y^2$.

Матрица:
$$Q = \begin{pmatrix} 9 & -2 \\ -2 & 6 \end{pmatrix}$$

(коэффициент при $xy$ равен $-4$, поэтому в матрицу ставим $-4/2 = -2$).

#### Шаг 2. Собственные значения

$$\det(Q - \lambda I) = (9-\lambda)(6-\lambda) - 4 = \lambda^2 - 15\lambda + 50 = 0$$

$$\lambda_{1,2} = \frac{15 \pm \sqrt{225 - 200}}{2} = \frac{15 \pm 5}{2}$$

$$\lambda_1 = 5, \quad \lambda_2 = 10$$

#### Шаг 3. Собственные векторы

**Для $\lambda_1 = 5$:**
$$(Q - 5I)v = \begin{pmatrix} 4 & -2 \\ -2 & 1 \end{pmatrix}v = 0 \implies v_1 = \begin{pmatrix} 1 \\ 2 \end{pmatrix}$$

**Для $\lambda_2 = 10$:**
$$(Q - 10I)v = \begin{pmatrix} -1 & -2 \\ -2 & -4 \end{pmatrix}v = 0 \implies v_2 = \begin{pmatrix} -2 \\ 1 \end{pmatrix}$$

Проверка ортогональности: $v_1 \cdot v_2 = (1)(-2) + (2)(1) = 0$. ✓

#### Шаг 4. Нормировка

$$\|v_1\| = \|v_2\| = \sqrt{1^2 + 2^2} = \sqrt{5}$$

$$e_1 = \frac{1}{\sqrt{5}}\begin{pmatrix} 1 \\ 2 \end{pmatrix}, \quad e_2 = \frac{1}{\sqrt{5}}\begin{pmatrix} -2 \\ 1 \end{pmatrix}$$

#### Шаг 5. Матрица поворота и замена координат

$$T = \frac{1}{\sqrt{5}}\begin{pmatrix} 1 & -2 \\ 2 & 1 \end{pmatrix}, \quad \begin{pmatrix} x \\ y \end{pmatrix} = T\begin{pmatrix} \tilde{x} \\ \tilde{y} \end{pmatrix} = \frac{1}{\sqrt{5}}\begin{pmatrix} \tilde{x} - 2\tilde{y} \\ 2\tilde{x} + \tilde{y} \end{pmatrix}$$

#### Шаг 6. Подстановка в уравнение

Квадратичная часть после замены даёт $5\tilde{x}^2 + 10\tilde{y}^2$.

Линейная часть $16x - 8y$:
$$16 \cdot \frac{\tilde{x} - 2\tilde{y}}{\sqrt{5}} - 8 \cdot \frac{2\tilde{x} + \tilde{y}}{\sqrt{5}} = \frac{16\tilde{x} - 32\tilde{y} - 16\tilde{x} - 8\tilde{y}}{\sqrt{5}} = \frac{-40\tilde{y}}{\sqrt{5}} = -8\sqrt{5}\,\tilde{y}$$

> восстановлено: линейная часть по $\tilde{x}$ обнулилась, что согласуется со словами преподавателя «сократить» на временном отрезке [09:43].

Уравнение в новых координатах:
$$5\tilde{x}^2 + 10\tilde{y}^2 - 8\sqrt{5}\,\tilde{y} - 2 = 0$$

#### Шаг 7. Выделение полных квадратов

По $\tilde{x}$ уже полный квадрат (нет линейного члена).

По $\tilde{y}$:
$$10\!\left(\tilde{y}^2 - \frac{8\sqrt{5}}{10}\tilde{y}\right) = 10\!\left(\tilde{y} - \frac{4}{\sqrt{5} \cdot \sqrt{5}} \cdot \sqrt{5}\right)^2 - \ldots$$

Точнее: вынесем 10:
$$10\!\left(\tilde{y} - \frac{4\sqrt{5}}{10}\right)^2 - 10 \cdot \frac{16 \cdot 5}{100} = 10\!\left(\tilde{y} - \frac{2}{\sqrt{5}}\right)^2 - 8$$

Подставляем в уравнение:
$$5\tilde{x}^2 + 10\!\left(\tilde{y} - \frac{2}{\sqrt{5}}\right)^2 - 8 - 2 = 0$$

$$5\tilde{x}^2 + 10\!\left(\tilde{y} - \frac{2}{\sqrt{5}}\right)^2 = 10$$

#### Шаг 8. Новые координаты (параллельный перенос)

$$\tilde{\tilde{x}} = \tilde{x}, \quad \tilde{\tilde{y}} = \tilde{y} - \frac{2}{\sqrt{5}}$$

Центр в системе $(\tilde{x}, \tilde{y})$: точка $\left(0,\; \dfrac{2}{\sqrt{5}}\right)$.

#### Шаг 9. Каноническое уравнение

$$\frac{\tilde{\tilde{x}}^2}{2} + \tilde{\tilde{y}}^2 = 1$$

Это **эллипс** с $a^2 = 2$, $b^2 = 1$, то есть $a = \sqrt{2}$, $b = 1$.

Большая полуось — вдоль $\tilde{\tilde{x}}$ (направление $e_1$), малая — вдоль $\tilde{\tilde{y}}$ (направление $e_2$).

#### Шаг 10. Характеристики эллипса

$$c^2 = a^2 - b^2 = 2 - 1 = 1, \quad c = 1$$

**Эксцентриситет:**
$$\varepsilon = \frac{c}{a} = \frac{1}{\sqrt{2}} = \frac{\sqrt{2}}{2} \approx 0{,}707$$

**Фокусы в канонической системе** $(\tilde{\tilde{x}}, \tilde{\tilde{y}})$:
$$F_1 = (-1, 0), \quad F_2 = (1, 0)$$

**Директрисы в канонической системе:**
$$\tilde{\tilde{x}} = \pm\frac{a}{\varepsilon} = \pm\frac{\sqrt{2}}{1/\sqrt{2}} = \pm 2$$

#### Шаг 11. Пересчёт в исходную систему координат

Цепочка замен:
$$\begin{pmatrix} x \\ y \end{pmatrix} = T\begin{pmatrix} \tilde{x} \\ \tilde{y} \end{pmatrix} = T\begin{pmatrix} \tilde{\tilde{x}} \\ \tilde{\tilde{y}} + \frac{2}{\sqrt{5}} \end{pmatrix}$$

**Фокусы.** Для $F_1 = (-1, 0)$ в канонической системе: $\tilde{x} = -1$, $\tilde{y} = 0 + \dfrac{2}{\sqrt{5}} = \dfrac{2}{\sqrt{5}}$.

$$\begin{pmatrix} x \\ y \end{pmatrix} = \frac{1}{\sqrt{5}}\begin{pmatrix} -1 - 2 \cdot \frac{2}{\sqrt{5}} \\ 2 \cdot (-1) + \frac{2}{\sqrt{5}} \end{pmatrix} = \frac{1}{\sqrt{5}}\begin{pmatrix} -1 - \frac{4}{\sqrt{5}} \\ -2 + \frac{2}{\sqrt{5}} \end{pmatrix}$$

Аналогично для $F_2 = (1, 0)$: $\tilde{x} = 1$, $\tilde{y} = \dfrac{2}{\sqrt{5}}$.

**Директриса $\tilde{\tilde{x}} = 2$**, т.е. $\tilde{x} = 2$. В исходных координатах ($\tilde{y}$ — произвольный параметр $t$):
$$x = \frac{1}{\sqrt{5}}(2 - 2t), \quad y = \frac{1}{\sqrt{5}}(4 + t)$$

Из первого уравнения $t = 1 - \dfrac{x\sqrt{5}}{2}$. Подставляем в второе:

$$y = \frac{4}{\sqrt{5}} + \frac{1}{\sqrt{5}}\!\left(1 - \frac{x\sqrt{5}}{2}\right) = \frac{4}{\sqrt{5}} + \frac{1}{\sqrt{5}} - \frac{x}{2} = \frac{5}{\sqrt{5}} - \frac{x}{2} = \sqrt{5} - \frac{x}{2}$$

Уравнение директрисы в исходной системе:
$$-\frac{x}{2} - y + \sqrt{5} = 0, \quad \text{или} \quad x + 2y = 2\sqrt{5}$$

> восстановлено: на семинаре преподаватель вывел уравнение вида $-\frac{x}{2} - y + \frac{5}{\sqrt{5}}$ для одной из директрис (временной отрезок [27:59]–[28:30]); вторая директриса получается аналогично.

### Задача 9в: $5x^2 + 12xy - 22x - 12y - 19 = 0$

#### Шаг 1. Матрица квадратичной части

Квадратичная часть: $5x^2 + 12xy$.

$$Q = \begin{pmatrix} 5 & 6 \\ 6 & 0 \end{pmatrix}$$

#### Шаг 2. Собственные значения

$$\det(Q - \lambda I) = (5-\lambda)(0-\lambda) - 36 = -\lambda(5-\lambda) - 36 = \lambda^2 - 5\lambda - 36 = 0$$

$$\lambda = \frac{5 \pm \sqrt{25 + 144}}{2} = \frac{5 \pm 13}{2}$$

$$\lambda_1 = 9, \quad \lambda_2 = -4$$

Собственные значения разных знаков — ожидаем **гиперболу**.

#### Шаг 3. Собственные векторы

**Для $\lambda_1 = 9$:**
$$(Q - 9I)v = \begin{pmatrix} -4 & 6 \\ 6 & -9 \end{pmatrix}v = 0 \implies 2v_1 = 3v_2 \implies v_1 = \begin{pmatrix} 3 \\ 2 \end{pmatrix}$$

**Для $\lambda_2 = -4$:**
$$(Q + 4I)v = \begin{pmatrix} 9 & 6 \\ 6 & 4 \end{pmatrix}v = 0 \implies 3v_1 = -2v_2 \implies v_2 = \begin{pmatrix} -2 \\ 3 \end{pmatrix}$$

Проверка: $v_1 \cdot v_2 = -6 + 6 = 0$. ✓

#### Шаг 4. Нормировка

$$\|v_1\| = \|v_2\| = \sqrt{9 + 4} = \sqrt{13}$$

$$e_1 = \frac{1}{\sqrt{13}}\begin{pmatrix} 3 \\ 2 \end{pmatrix}, \quad e_2 = \frac{1}{\sqrt{13}}\begin{pmatrix} -2 \\ 3 \end{pmatrix}$$

#### Шаг 5. Матрица поворота и замена

$$T = \frac{1}{\sqrt{13}}\begin{pmatrix} 3 & -2 \\ 2 & 3 \end{pmatrix}, \quad \begin{pmatrix} x \\ y \end{pmatrix} = \frac{1}{\sqrt{13}}\begin{pmatrix} 3\tilde{x} - 2\tilde{y} \\ 2\tilde{x} + 3\tilde{y} \end{pmatrix}$$

#### Шаг 6. Подстановка линейной части

Линейная часть $-22x - 12y$:
$$-22 \cdot \frac{3\tilde{x} - 2\tilde{y}}{\sqrt{13}} - 12 \cdot \frac{2\tilde{x} + 3\tilde{y}}{\sqrt{13}} = \frac{-66\tilde{x} + 44\tilde{y} - 24\tilde{x} - 36\tilde{y}}{\sqrt{13}} = \frac{-90\tilde{x} + 8\tilde{y}}{\sqrt{13}}$$

> восстановлено: на временном отрезке [36:47] преподаватель называет коэффициенты «минус 22 3x с волной минус 2y с волной на корень из 13» и «плюс 8y с волной на корень из 13», что соответствует вычислению выше.

Уравнение после поворота:
$$9\tilde{x}^2 - 4\tilde{y}^2 - \frac{90}{\sqrt{13}}\tilde{x} + \frac{8}{\sqrt{13}}\tilde{y} - 19 = 0$$

#### Шаг 7. Выделение полных квадратов

По $\tilde{x}$ (коэффициент $\lambda_1 = 9$):
$$9\!\left(\tilde{x} - \frac{5}{\sqrt{13}}\right)^2 - \frac{225}{13}$$

По $\tilde{y}$ (коэффициент $\lambda_2 = -4$):
$$-4\!\left(\tilde{y} - \frac{1}{\sqrt{13}}\right)^2 + \frac{4}{13}$$

Подставляем:
$$9\!\left(\tilde{x} - \frac{5}{\sqrt{13}}\right)^2 - \frac{225}{13} - 4\!\left(\tilde{y} - \frac{1}{\sqrt{13}}\right)^2 + \frac{4}{13} - 19 = 0$$

$$9\!\left(\tilde{x} - \frac{5}{\sqrt{13}}\right)^2 - 4\!\left(\tilde{y} - \frac{1}{\sqrt{13}}\right)^2 = 19 + \frac{225}{13} - \frac{4}{13} = 19 + \frac{221}{13} = 19 + 17 = 36$$

> восстановлено: числовые значения правой части восстановлены из структуры задачи; в транскрипции на отрезке [40:33]–[42:22] идёт обсуждение выделения полных квадратов с подтверждением участниками.

#### Шаг 8. Каноническое уравнение

$$\tilde{\tilde{x}} = \tilde{x} - \frac{5}{\sqrt{13}}, \quad \tilde{\tilde{y}} = \tilde{y} - \frac{1}{\sqrt{13}}$$

$$\frac{\tilde{\tilde{x}}^2}{4} - \frac{\tilde{\tilde{y}}^2}{9} = 1$$

Это **гипербола** с $a^2 = 4$, $b^2 = 9$, $a = 2$ (вещественная полуось), $b = 3$ (мнимая полуось).

#### Шаг 9. Характеристики гиперболы

$$c^2 = a^2 + b^2 = 4 + 9 = 13, \quad c = \sqrt{13}$$

**Эксцентриситет:**
$$\varepsilon = \frac{c}{a} = \frac{\sqrt{13}}{2} > 1$$

**Фокусы в канонической системе:** $(\pm\sqrt{13},\, 0)$.

**Асимптоты в канонической системе:** $\tilde{\tilde{y}} = \pm\dfrac{3}{2}\tilde{\tilde{x}}$.

**Построение гиперболы:** строится прямоугольник со сторонами $2a = 4$ вдоль $\tilde{\tilde{x}}$ и $2b = 6$ вдоль $\tilde{\tilde{y}}$; диагонали этого прямоугольника дают асимптоты. Гипербола пересекает ось $\tilde{\tilde{x}}$ в точках $(\pm 2, 0)$ и приближается к асимптотам.

**Фокусы в исходной системе:** $\tilde{x} = \pm\sqrt{13} + \dfrac{5}{\sqrt{13}}$, $\tilde{y} = \dfrac{1}{\sqrt{13}}$.

$$\begin{pmatrix} x \\ y \end{pmatrix} = \frac{1}{\sqrt{13}}\begin{pmatrix} 3\tilde{x} - 2\tilde{y} \\ 2\tilde{x} + 3\tilde{y} \end{pmatrix}$$

### Задача 9б: $x^2 - 2xy + y^2 - 10x - 6y + 25 = 0$

#### Шаг 1. Матрица квадратичной части

Квадратичная часть: $x^2 - 2xy + y^2$.

$$Q = \begin{pmatrix} 1 & -1 \\ -1 & 1 \end{pmatrix}$$

#### Шаг 2. Собственные значения

$$\det(Q - \lambda I) = (1-\lambda)^2 - 1 = \lambda^2 - 2\lambda = \lambda(\lambda - 2) = 0$$

$$\lambda_1 = 0, \quad \lambda_2 = 2$$

Одно из собственных значений равно нулю — ожидаем **параболу**.

#### Шаг 3. Собственные векторы

**Для $\lambda_1 = 0$:**
$$Qv = 0 \implies v_1 - v_2 = 0 \implies v_1 = \begin{pmatrix} 1 \\ 1 \end{pmatrix}$$

**Для $\lambda_2 = 2$:**
$$(Q - 2I)v = \begin{pmatrix} -1 & -1 \\ -1 & -1 \end{pmatrix}v = 0 \implies v_2 = \begin{pmatrix} -1 \\ 1 \end{pmatrix}$$

Проверка: $v_1 \cdot v_2 = -1 + 1 = 0$. ✓

#### Шаг 4. Нормировка

$$e_1 = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 \\ 1 \end{pmatrix}, \quad e_2 = \frac{1}{\sqrt{2}}\begin{pmatrix} -1 \\ 1 \end{pmatrix}$$

#### Шаг 5. Матрица поворота и замена

$$T = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 & -1 \\ 1 & 1 \end{pmatrix}, \quad \begin{pmatrix} x \\ y \end{pmatrix} = \frac{1}{\sqrt{2}}\begin{pmatrix} \tilde{x} - \tilde{y} \\ \tilde{x} + \tilde{y} \end{pmatrix}$$

> восстановлено: на отрезке [52:56]–[54:12] преподаватель указывает, что одно из собственных значений 0 (легко видеть: сумма строк матрицы равна 0), второе равно 2.

#### Шаг 6. Подстановка в уравнение

Квадратичная часть: $0 \cdot \tilde{x}^2 + 2\tilde{y}^2 = 2\tilde{y}^2$.

Линейная часть $-10x - 6y$:
$$-10 \cdot \frac{\tilde{x} - \tilde{y}}{\sqrt{2}} - 6 \cdot \frac{\tilde{x} + \tilde{y}}{\sqrt{2}} = \frac{-10\tilde{x} + 10\tilde{y} - 6\tilde{x} - 6\tilde{y}}{\sqrt{2}} = \frac{-16\tilde{x} + 4\tilde{y}}{\sqrt{2}} = -8\sqrt{2}\,\tilde{x} + 2\sqrt{2}\,\tilde{y}$$

Уравнение после поворота:
$$2\tilde{y}^2 - 8\sqrt{2}\,\tilde{x} + 2\sqrt{2}\,\tilde{y} + 25 = 0$$

> восстановлено: на отрезке [57:45]–[59:44] преподаватель называет «минус 16x с волной делить на корень из 2» (что равно $-8\sqrt{2}\,\tilde{x}$) и «минус 4y с волной на корень из нуля» (ошибка в транскрипции; из структуры уравнения следует $+2\sqrt{2}\,\tilde{y}$).

#### Шаг 7. Выделение полного квадрата по $\tilde{y}$

$$2\!\left(\tilde{y}^2 + \sqrt{2}\,\tilde{y}\right) = 2\!\left(\tilde{y} + \frac{\sqrt{2}}{2}\right)^2 - 2 \cdot \frac{1}{2} = 2\!\left(\tilde{y} + \frac{1}{\sqrt{2}}\right)^2 - 1$$

Уравнение:
$$2\!\left(\tilde{y} + \frac{1}{\sqrt{2}}\right)^2 - 1 - 8\sqrt{2}\,\tilde{x} + 25 = 0$$

$$2\!\left(\tilde{y} + \frac{1}{\sqrt{2}}\right)^2 = 8\sqrt{2}\,\tilde{x} - 24$$

$$2\!\left(\tilde{y} + \frac{1}{\sqrt{2}}\right)^2 = 8\sqrt{2}\!\left(\tilde{x} - \frac{24}{8\sqrt{2}}\right) = 8\sqrt{2}\!\left(\tilde{x} - \frac{3}{\sqrt{2}}\right)$$

> восстановлено: точные коэффициенты восстановлены из структуры; в транскрипции на отрезке [59:44]–[1:01:42] обсуждается выбор базиса (собственный вектор для нулевого значения) и итоговый вид уравнения с «минус 4y волной», что согласуется с выделением полного квадрата.

#### Шаг 8. Новые координаты

$$\tilde{\tilde{y}} = \tilde{y} + \frac{1}{\sqrt{2}}, \quad \tilde{\tilde{x}} = \tilde{x} - \frac{3}{\sqrt{2}}$$

Уравнение принимает вид:
$$2\tilde{\tilde{y}}^2 = 8\sqrt{2}\,\tilde{\tilde{x}}$$

$$\tilde{\tilde{y}}^2 = 4\sqrt{2}\,\tilde{\tilde{x}}$$

#### Шаг 9. Каноническое уравнение параболы

$$\tilde{\tilde{y}}^2 = 2p_{\text{фок}} \cdot \tilde{\tilde{x}}, \quad 2p_{\text{фок}} = 4\sqrt{2}, \quad p_{\text{фок}} = 2\sqrt{2}$$

Это **парабола** с фокальным параметром $p_{\text{фок}} = 2\sqrt{2}$.

#### Шаг 10. Характеристики параболы

**Фокус в канонической системе:**
$$F = \left(\frac{p_{\text{фок}}}{2},\; 0\right) = \left(\sqrt{2},\; 0\right)$$

**Директриса в канонической системе:**
$$\tilde{\tilde{x}} = -\frac{p_{\text{фок}}}{2} = -\sqrt{2}$$

Фокальный параметр $p_{\text{фок}}$ равен расстоянию от фокуса до директрисы, а также половине длины фокальной хорды (хорды, проходящей через фокус перпендикулярно оси параболы).

**Фокус и директриса в исходной системе** получаются через ту же цепочку замен поворот + два параллельных переноса.

> восстановлено: на отрезке [1:01:42]–[1:03:42] преподаватель говорит «получился $\tilde{\tilde{y}}^2$ равно 2 по x» и объясняет, что канонические уравнения параболы имеют вид $y^2 = 2px$, где $p$ — фокальный параметр, фокус в точке $(p/2, 0)$.

## Справочный лист

**Матрица квадратичной части** $Ax^2 + 2Bxy + Cy^2$:
$$Q = \begin{pmatrix} A & B \\ B & C \end{pmatrix}$$

**Тип кривой по знакам $\lambda_1, \lambda_2$:**
- $\lambda_1, \lambda_2 > 0$: эллипс (или пустое множество/точка при вырожденных случаях)
- $\lambda_1, \lambda_2 < 0$: эллипс (мнимый)
- $\lambda_1 > 0, \lambda_2 < 0$ (или наоборот): гипербола
- один $\lambda_i = 0$: парабола

**Нормировка:** $e_i = v_i / \|v_i\|$; $T = (e_1 \mid e_2)$ — матрица поворота.

**Формулы для эллипса** $x^2/a^2 + y^2/b^2 = 1$, $a \ge b > 0$:
$$c = \sqrt{a^2 - b^2}, \quad \varepsilon = c/a, \quad F_{1,2} = (\pm c, 0), \quad \text{директрисы: } x = \pm a/\varepsilon$$

**Формулы для гиперболы** $x^2/a^2 - y^2/b^2 = 1$:
$$c = \sqrt{a^2 + b^2}, \quad \varepsilon = c/a > 1, \quad F_{1,2} = (\pm c, 0), \quad \text{асимптоты: } y = \pm(b/a)x$$

**Формулы для параболы** $y^2 = 2px$:
$$F = (p/2, 0), \quad \text{директриса: } x = -p/2, \quad \varepsilon = 1$$

**Пересчёт в исходную систему:**
$$\begin{pmatrix} x \\ y \end{pmatrix} = T\begin{pmatrix} \tilde{\tilde{x}} + x_0 \\ \tilde{\tilde{y}} + y_0 \end{pmatrix}$$

где $(x_0, y_0)$ — смещение центра в системе $(\tilde{x}, \tilde{y})$.

## Типичные ошибки семинара

1. **Ошибка в матрице квадратичной формы:** коэффициент при $xy$ нужно разделить на 2 перед занесением в матрицу. Например, член $-4xy$ даёт $B = -2$.

2. **Перепутаны формулы для эллипса и гиперболы:** для эллипса $c^2 = a^2 - b^2$, для гиперболы $c^2 = a^2 + b^2$. Путать их — принципиальная ошибка; на семинаре этот момент был специально разобран (отрезок [30:50]–[32:48]).

3. **Забыть, что «выпрыгивает» при выделении полного квадрата:** при выделении $\lambda(\tilde{x} - x_0)^2$ из $\lambda\tilde{x}^2 + p\tilde{x}$ нужно отнять $\lambda x_0^2$ от константы.

4. **Неправильный выбор «большой» и «малой» полуосей:** большая полуось $a$ — наибольшее из $a, b$; фокусы лежат на большой оси.

5. **Подставлять координаты фокусов без пересчёта:** фокусы найдены в канонической системе; в исходную систему их нужно переводить через полную цепочку замен (поворот и оба параллельных переноса).

6. **Для параболы пытаться выделить полный квадрат по обеим переменным:** по переменной с нулевым собственным значением полный квадрат не выделяется — соответствующий линейный член остаётся и убирается вторым параллельным переносом.

7. **Ошибка в знаке при подстановке линейных членов:** после замены $x = (a_{11}\tilde{x} + a_{12}\tilde{y})/\sqrt{k}$ в выражение $\alpha x + \beta y$ нужно аккуратно расписать оба слагаемых, чтобы не потерять знак.
