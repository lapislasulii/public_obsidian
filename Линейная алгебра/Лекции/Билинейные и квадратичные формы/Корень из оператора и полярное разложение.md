# Корень из оператора и полярное разложение

## 1. Обзор лекции

Лекция посвящена завершению теории нормальных операторов в вещественных евклидовых пространствах и введению двух ключевых конструкций: **квадратного корня из положительно определённого оператора** и **полярного разложения** произвольного невырожденного оператора. Сначала доказывается теорема об ортогональности двумерных инвариантных подпространств нормального оператора, соответствующих несопряжённым собственным значениям комплексификации, — это обобщение хорошо известного факта об ортогональности собственных векторов для разных собственных значений. Затем вводятся положительно определённые и неотрицательно определённые самосопряжённые операторы по аналогии с положительными вещественными числами. Центральный результат — теорема о существовании и единственности квадратного корня из положительно определённого оператора. Завершает лекцию теорема о полярном разложении: любой невырожденный оператор единственным образом представляется как произведение ортогонального оператора и положительно определённого оператора — прямой аналог записи комплексного числа в тригонометрической форме $z = r e^{i\varphi}$.

---

## 2. Предварительные сведения

- **Нормальный оператор**: оператор $A$, коммутирующий со своим сопряжённым: $AA^* = A^*A$.
- **Самосопряжённый оператор**: $A = A^*$; частный случай нормального оператора.
- **Ортогональный оператор**: $UU^* = U^*U = I$, то есть $U^{-1} = U^*$.
- **Комплексификация**: вещественное пространство $V$ расширяется до $V_\mathbb{C} = V \oplus iV$; вещественный оператор $A$ действует на $V_\mathbb{C}$ покоординатно.
- **Инвариантное подпространство**: подпространство $W \subseteq V$, для которого $A(W) \subseteq W$.
- **Эрмитово скалярное произведение** в $V_\mathbb{C}$: $\langle u+iv,\, x+iy \rangle_\mathbb{C} = \langle u,x\rangle + \langle v,y\rangle + i(\langle v,x\rangle - \langle u,y\rangle)$.
- **Квадратичная форма** самосопряжённого оператора $A$: $q(x) = \langle Ax, x \rangle$.
- **Положительная определённость квадратичной формы**: $q(x) > 0$ для всех $x \neq 0$.
- **Теорема о нормальном операторе**: в комплексном пространстве нормальный оператор диагонализируем в ортонормированном базисе; собственные векторы, соответствующие разным собственным значениям, ортогональны.

## 3. Основное содержание

### 3.1. Ортогональность двумерных инвариантных подпространств нормального оператора

#### Контекст

Пусть $A$ — вещественный нормальный оператор на евклидовом пространстве $V$. При комплексификации $A$ становится нормальным оператором на $V_\mathbb{C}$. Его невещественные собственные значения приходят комплексно-сопряжёнными парами $\lambda, \bar\lambda$. Каждой паре соответствует двумерное вещественное инвариантное подпространство

$$V_\lambda = \mathrm{span}_\mathbb{R}(\mathbf{a}, \mathbf{b}),$$

где $\mathbf{a} + i\mathbf{b}$ — собственный вектор в $V_\mathbb{C}$ с собственным значением $\lambda$.

**Теорема 3.1** (Ортогональность двумерных инвариантных подпространств).

Пусть $A$ — вещественный нормальный оператор; $\lambda, \mu$ — невещественные собственные значения его комплексификации, причём $\lambda \neq \mu$ и $\lambda \neq \bar\mu$. Тогда соответствующие двумерные вещественные инвариантные подпространства $V_\lambda$ и $V_\mu$ ортогональны в $V$ (относительно вещественного скалярного произведения):

$$V_\lambda \perp V_\mu.$$

**Доказательство.**

Пусть $\mathbf{a}+i\mathbf{b} \in V_\mathbb{C}$ — собственный вектор с собственным значением $\lambda$, а $\mathbf{c}+i\mathbf{d} \in V_\mathbb{C}$ — собственный вектор с собственным значением $\mu$.

Тогда $\mathbf{a}-i\mathbf{b}$ — собственный вектор с собственным значением $\bar\lambda$, а $\mathbf{c}-i\mathbf{d}$ — с собственным значением $\bar\mu$.

**Шаг 1. Эрмитовы скалярные произведения равны нулю.**

Так как $A$ нормальный, собственные векторы при разных собственных значениях ортогональны в $V_\mathbb{C}$. По условию все четыре числа $\lambda, \bar\lambda, \mu, \bar\mu$ различны (из $\lambda \neq \mu$ и $\lambda \neq \bar\mu$ следует, что ни одна из четырёх пар не совпадает). Поэтому:

$$\langle \mathbf{a}+i\mathbf{b},\, \mathbf{c}+i\mathbf{d} \rangle_\mathbb{C} = 0, \tag{1}$$
$$\langle \mathbf{a}+i\mathbf{b},\, \mathbf{c}-i\mathbf{d} \rangle_\mathbb{C} = 0. \tag{2}$$

**Шаг 2. Раскрываем по определению.**

Из (1):
$$\langle \mathbf{a},\mathbf{c}\rangle + \langle \mathbf{b},\mathbf{d}\rangle + i\bigl(\langle \mathbf{b},\mathbf{c}\rangle - \langle \mathbf{a},\mathbf{d}\rangle\bigr) = 0.$$

Равенство нулю вещественной и мнимой частей:
$$\langle \mathbf{a},\mathbf{c}\rangle + \langle \mathbf{b},\mathbf{d}\rangle = 0, \tag{3}$$
$$\langle \mathbf{b},\mathbf{c}\rangle - \langle \mathbf{a},\mathbf{d}\rangle = 0. \tag{4}$$

Из (2):
$$\langle \mathbf{a},\mathbf{c}\rangle - \langle \mathbf{b},\mathbf{d}\rangle + i\bigl(\langle \mathbf{b},\mathbf{c}\rangle + \langle \mathbf{a},\mathbf{d}\rangle\bigr) = 0.$$

$$\langle \mathbf{a},\mathbf{c}\rangle - \langle \mathbf{b},\mathbf{d}\rangle = 0, \tag{5}$$
$$\langle \mathbf{b},\mathbf{c}\rangle + \langle \mathbf{a},\mathbf{d}\rangle = 0. \tag{6}$$

**Шаг 3. Выводим ортогональность базисных векторов.**

Складываем (3) и (5): $2\langle \mathbf{a},\mathbf{c}\rangle = 0$, то есть $\langle \mathbf{a},\mathbf{c}\rangle = 0$.

Вычитаем (5) из (3): $2\langle \mathbf{b},\mathbf{d}\rangle = 0$, то есть $\langle \mathbf{b},\mathbf{d}\rangle = 0$.

Складываем (4) и (6): $2\langle \mathbf{b},\mathbf{c}\rangle = 0$, то есть $\langle \mathbf{b},\mathbf{c}\rangle = 0$.

Вычитаем (6) из (4): $2\langle \mathbf{a},\mathbf{d}\rangle = 0$, то есть $\langle \mathbf{a},\mathbf{d}\rangle = 0$.

**Шаг 4. Заключение.**

Все попарные скалярные произведения базисных векторов $\{\mathbf{a},\mathbf{b}\}$ подпространства $V_\lambda$ и $\{\mathbf{c},\mathbf{d}\}$ подпространства $V_\mu$ равны нулю. Следовательно, $V_\lambda \perp V_\mu$. $\square$

**Следствие 3.1.** В случае $\lambda$ и $\bar\lambda$ одного и того же невещественного собственного значения: вещественное инвариантное подпространство $V_\lambda = \mathrm{span}(\mathbf{a},\mathbf{b})$ обладает тем свойством, что $\mathbf{a} \perp \mathbf{b}$ (то есть уже имеем ортогональный базис подпространства).

> восстановлено: Это следует из равенства $\langle \mathbf{a}+i\mathbf{b},\, \mathbf{a}-i\mathbf{b}\rangle_\mathbb{C} = 0$, что разворачивается в $\|\mathbf{a}\|^2 - \|\mathbf{b}\|^2 = 0$ и $2\langle \mathbf{a},\mathbf{b}\rangle = 0$. Из второго: $\mathbf{a}\perp\mathbf{b}$. Из первого: $\|\mathbf{a}\| = \|\mathbf{b}\|$. Таким образом, нормировав, получаем ортонормированный базис нужного двумерного подпространства.

МОТИВАЦИЯ: При описании структуры нормального оператора в вещественном пространстве вместо одномерных собственных подпространств появляются двумерные, и необходимо знать, что они тоже попарно ортогональны — иначе не удалось бы построить ортонормированный базис всего пространства.

ИНТУИЦИЯ: В четырёхмерном пространстве нормальный оператор может действовать как поворот в одной двумерной плоскости на угол $\varphi_1$ и как поворот в другой двумерной плоскости на угол $\varphi_2$. Теорема говорит: эти две плоскости ортогональны (как двумерные подпространства в $\mathbb{R}^4$), что является прямым аналогом ортогональности собственных векторов.

ТИПИЧНЫЕ ОШИБКИ:
- Забыть условие $\lambda \neq \bar\mu$: если $\mu = \bar\lambda$, то подпространства $V_\lambda$ и $V_\mu$ совпадают — утверждение неприменимо.
- Путать эрмитово и вещественное скалярное произведение при вычислениях.

### 3.2. Положительно определённые операторы

**Определение 3.2.** Самосопряжённый оператор $A$ в евклидовом пространстве называется:

- **положительно определённым** (и обозначается $A > 0$), если соответствующая квадратичная форма $q(x) = \langle Ax, x\rangle$ положительно определена:
$$\forall\, x \neq 0:\quad \langle Ax, x\rangle > 0;$$

- **неотрицательно определённым** (и обозначается $A \geq 0$), если:
$$\forall\, x:\quad \langle Ax, x\rangle \geq 0.$$

> восстановлено: Аналогично определяются отрицательно определённый ($A < 0$) и неположительно определённый ($A \leq 0$) операторы.

МОТИВАЦИЯ: Это прямой аналог положительных и неотрицательных вещественных чисел. Как у положительных чисел можно извлекать квадратный корень, так и у положительно определённых операторов — это будет показано ниже.

ИНТУИЦИЯ: Положительно определённый оператор — это «растягивающий» оператор без зеркальных отражений. Все направления растягиваются (скалярное произведение $\langle Ax, x\rangle > 0$ означает, что оператор «не переворачивает» вектор).

**Лемма 3.2** (Критерий через собственные значения).

Самосопряжённый оператор $A$ положительно определён тогда и только тогда, когда все его собственные значения положительны:
$$A > 0 \iff \forall\, i:\; \lambda_i > 0.$$

**Доказательство.**

Так как $A$ самосопряжён, он нормальный, и существует ортонормированный базис $e_1, \ldots, e_n$ из собственных векторов: $Ae_i = \lambda_i e_i$. Для произвольного $x = \sum_i x_i e_i$:

$$\langle Ax, x\rangle = \left\langle \sum_i \lambda_i x_i e_i,\, \sum_j x_j e_j \right\rangle = \sum_i \lambda_i x_i^2.$$

Если все $\lambda_i > 0$, то $\sum_i \lambda_i x_i^2 > 0$ при $x \neq 0$, то есть $A > 0$.

Обратно: если $A > 0$, то для собственного вектора $e_i$:
$$0 < \langle Ae_i, e_i\rangle = \lambda_i \langle e_i, e_i\rangle = \lambda_i,$$
то есть $\lambda_i > 0$. $\square$

ТИПИЧНЫЕ ОШИБКИ:
- Определение применимо только к самосопряжённым операторам; у несамосопряжённого оператора $\langle Ax,x\rangle$ может быть знакопостоянным, но это не делает его «положительно определённым» в смысле данного определения.

### 3.3. Квадратный корень из оператора

**Определение 3.3.** Для положительно определённого оператора $A > 0$ **квадратным корнем** (арифметическим) называется такой положительно определённый оператор $B > 0$, что

$$B^2 = A.$$

Обозначение: $B = \sqrt{A}$.

МОТИВАЦИЯ: Арифметический квадратный корень из числа $a > 0$ — это единственное $b > 0$ с $b^2 = a$. Здесь мы обобщаем это понятие на операторы, сохраняя условие «положительности» ($B > 0$), чтобы обеспечить единственность.

**Теорема 3.3** (Существование и единственность корня).

Для любого положительно определённого оператора $A > 0$ в евклидовом пространстве существует единственный квадратный корень $\sqrt{A} > 0$.

**Доказательство.**

**Существование.**

Так как $A$ самосопряжён, существует ортонормированный базис $e_1, \ldots, e_n$ такой, что матрица $A$ в этом базисе диагональна:

$$[A] = \mathrm{diag}(\lambda_1, \ldots, \lambda_n), \quad \lambda_i > 0.$$

Определим оператор $B$ матрицей в том же базисе:

$$[B] = \mathrm{diag}\!\left(\sqrt{\lambda_1}, \ldots, \sqrt{\lambda_n}\right).$$

Тогда:
1. $[B]^2 = \mathrm{diag}(\lambda_1, \ldots, \lambda_n) = [A]$, то есть $B^2 = A$.
2. Все $\sqrt{\lambda_i} > 0$, значит $B > 0$.

**Единственность.**

Пусть $B_1$ и $B_2$ — два положительно определённых корня: $B_1^2 = B_2^2 = A$.

Собственные значения $A$ суть $\lambda_1, \ldots, \lambda_n > 0$. Оператор $B_i$ тоже самосопряжён и положительно определён (по условию), поэтому диагонализируем в ортонормированном базисе. Единственное, что можно варьировать при построении корня — это порядок расстановки значений $\sqrt{\lambda_k}$ на диагонали, то есть выбор перестановки $\sigma$ собственных значений:

$$[B_\sigma] = \mathrm{diag}\!\left(\sqrt{\lambda_{\sigma(1)}}, \ldots, \sqrt{\lambda_{\sigma(n)}}\right).$$

Но собственные значения $B$ однозначно определяются как $\sqrt{\lambda_i}$ (из уравнения $B^2 = A$ и условия $B>0$): у положительно определённого оператора набор собственных значений фиксирован. Различные перестановки дают операторы с теми же собственными значениями, и соответствующий им оператор в исходном пространстве один и тот же (так как $B$ сам обязан быть нормальным и иметь те же инвариантные подпространства, что и $A$). Следовательно, $B_1 = B_2$. $\square$

> восстановлено: Более строгое доказательство единственности: если $B^2 = A$ и $B > 0$, то $B$ коммутирует с $A$ (так как $BA = B \cdot B^2 = B^3 = B^2 \cdot B = AB$). Операторы, коммутирующие с нормальным оператором, имеют те же инвариантные подпространства. Значит, $B$ диагонализируется в том же ортонормированном базисе, что и $A$, а на диагонали стоят положительные квадратные корни из $\lambda_i$ — единственные положительные числа с квадратом $\lambda_i$.

ИНТУИЦИЯ: Ровно как $\sqrt{4} = 2$ (а не $-2$), потому что мы выбираем положительный корень, здесь мы выбираем «положительно определённый» корень, что и обеспечивает единственность.

ТИПИЧНЫЕ ОШИБКИ:
- Без условия положительной определённости корень может не быть единственным (ср.: у единичного оператора $I$ есть корень $I$, но и оператор с $\pm 1$ на диагонали тоже даёт $I$ при возведении в квадрат).

### 3.4. Полярное разложение оператора

**Мотивирующая аналогия.**

Группа ненулевых комплексных чисел $\mathbb{C}^*$ изоморфна $\mathbb{R}_{>0} \times U(1)$: каждое $z \in \mathbb{C}^*$ единственным образом записывается как

$$z = r \cdot e^{i\varphi}, \quad r = |z| > 0,\; \varphi \in [0, 2\pi).$$

Здесь $r$ — «модуль» (вещественное положительное число), а $e^{i\varphi}$ — «поворот» (элемент окружности $U(1)$). Для операторов роль $r$ играет положительно определённый оператор, роль $e^{i\varphi}$ — ортогональный оператор.

**Теорема 3.4** (Полярное разложение).

Пусть $A$ — вещественный невырожденный оператор евклидова пространства. Тогда существуют единственные операторы $U$ и $B$ такие, что:
- $U$ — ортогональный оператор,
- $B$ — положительно определённый оператор,
- $A = U B$.

Это разложение называется **полярным разложением** оператора $A$.

**Доказательство.**

**Существование.**

Рассмотрим оператор $A^*A$. Он самосопряжён: $(A^*A)^* = A^*(A^*)^* = A^*A$. Кроме того, он положительно определён: для любого $x \neq 0$

$$\langle A^*Ax, x\rangle = \langle Ax, Ax\rangle = \|Ax\|^2 > 0,$$

так как $A$ невырожден, $Ax \neq 0$. По теореме 3.3 существует единственный $B = \sqrt{A^*A} > 0$.

Положим $U = AB^{-1}$. Оператор $B^{-1}$ существует, так как $B > 0$ невырожден. Проверим, что $U$ ортогонален:

$$U^*U = (AB^{-1})^*(AB^{-1}) = (B^{-1})^* A^* A B^{-1} = B^{-1} (A^*A) B^{-1}.$$

> восстановлено: Здесь использовано $(B^{-1})^* = (B^*)^{-1} = B^{-1}$, так как $B$ самосопряжён. Затем, так как $B^2 = A^*A$ и $B$ коммутирует со своими степенями: $B^{-1}(A^*A)B^{-1} = B^{-1} B^2 B^{-1} = I$.

Таким образом $U^*U = I$, то есть $U$ ортогональный. И $A = UB$.

**Единственность.**

Если $A = UB$ с $U$ ортогональным и $B > 0$, то

$$A^*A = (UB)^*(UB) = B^*U^*UB = B^*B = B^2,$$

так как $U^*U = I$ и $B^* = B$. Следовательно $B^2 = A^*A$, откуда $B = \sqrt{A^*A}$ единственным образом по теореме 3.3. Из $B$ находим $U = AB^{-1}$ единственным образом. $\square$

МОТИВАЦИЯ: Полярное разложение позволяет разделить геометрическое действие оператора на «изометрическую часть» (повороты/отражения, которые не меняют длин) и «растягивающую часть» (положительно определённый оператор, который только растягивает).

ИНТУИЦИЯ: В двумерном случае $A = UB$ означает: сначала растягиваем пространство (возможно, по-разному вдоль разных осей) — это $B$, затем поворачиваем/отражаем — это $U$. В одномерном случае: $A = u \cdot b$, где $u = \pm 1$ (знак), $b > 0$ (модуль). Полярное разложение — это точный многомерный аналог.

ТИПИЧНЫЕ ОШИБКИ:
- Знак перестановки имеет значение: $A = UB \neq BU$ в общем случае. Существует и «правое» полярное разложение $A = B'U'$, где $B' = \sqrt{AA^*}$.
- Невырожденность $A$ существенна: при $\det A = 0$ оператор $B = \sqrt{A^*A}$ всё ещё существует, но $U = AB^{-1}$ уже не определён (нет $B^{-1}$). Разложение обобщается на вырожденный случай через псевдообратные операторы.

## 4. Справочный лист

| Объект | Формула / характеристика |
|---|---|
| Положительно определённый оператор | $A = A^*$, $\langle Ax,x\rangle > 0$ $\forall x\neq 0$ |
| Критерий через собственные значения | $A > 0 \iff$ все $\lambda_i > 0$ |
| Квадратный корень | $B = \sqrt{A}$: $B^2 = A$, $B > 0$ |
| Конструкция корня | В базисе: $[B] = \mathrm{diag}(\sqrt{\lambda_1},\ldots,\sqrt{\lambda_n})$ |
| Полярное разложение | $A = UB$, $U$ ортогон., $B = \sqrt{A^*A} > 0$ |
| Единственность полярного разложения | $B = \sqrt{A^*A}$, $U = AB^{-1}$ — единственны |
| Ортогональность двумерных подпространств | $V_\lambda \perp V_\mu$ при $\lambda \neq \mu$, $\lambda \neq \bar\mu$ (нормальный оператор) |
| Базис внутри $V_\lambda$ | $\mathbf{a} \perp \mathbf{b}$, $\|\mathbf{a}\| = \|\mathbf{b}\|$ |

## 5. Концептуальное резюме

Лекция завершает описание структуры нормальных операторов в вещественных евклидовых пространствах: двумерные инвариантные подпространства, отвечающие несопряжённым собственным значениям, взаимно ортогональны, а внутри каждого такого подпространства оператор имеет вид блока поворота. Затем вводится новый класс объектов — положительно определённые операторы, являющиеся операторным аналогом положительных чисел; они обладают всеми ключевыми свойствами последних, в частности, из них можно однозначно извлекать квадратный корень. Кульминацией лекции является теорема о полярном разложении: любой невырожденный оператор единственным образом представляется как произведение ортогонального и положительно определённого операторов — это многомерное обобщение тригонометрической формы комплексного числа и открывает путь к понятию сингулярного разложения матриц.

## 6. Связи с более широкой математикой

- **Сингулярное разложение (SVD)**: полярное разложение $A = UB$ при $B = \sqrt{A^*A}$ напрямую связано с SVD: $A = U \Sigma V^T$, где $\Sigma$ — диагональная матрица сингулярных чисел (квадратные корни из собственных значений $A^*A$).
- **Матричный анализ**: для матриц понятие $f(A)$ при $f = \sqrt{\cdot}$ обобщается через спектральное разложение; корень из матрицы активно используется в теории дифференциальных уравнений и в статистике (метод Холецкого — факторизация $A = LL^T$ для положительно определённых матриц).
- **Алгебра Ли / теория Ли**: ортогональные операторы образуют группу Ли $O(n)$, положительно определённые — конус в линейном пространстве. Полярное разложение описывает топологию группы $GL_n(\mathbb{R})$ как прямое произведение $O(n) \times \{\text{положительно определённые}\}$.
- **Квантовая механика**: положительно определённые операторы (точнее, неотрицательно определённые) — это операторы плотности (матрицы состояний); корень из оператора используется при определении расстояния между квантовыми состояниями (расстояние Буреса).
- **Численные методы**: метод квадратного корня (метод Холецкого) для решения систем $Ax = b$ с положительно определённой матрицей $A$ — прямое приложение теории корня из оператора.
