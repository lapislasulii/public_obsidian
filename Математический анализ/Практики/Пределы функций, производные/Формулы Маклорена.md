## Разбор задач

### Задача №9 (a): Разложение функции $f(x) = x e^x$

**Условие** Запишите формулу Маклорена для функции $f(x) = x e^x$.

**Что требуется найти** Многочлен Тейлора (Маклорена) порядка $n$ с остаточным членом.

**Первые мысли и наблюдения**

- Функция представляет собой произведение $x$ на стандартную экспоненту.
- Можно использовать готовую формулу для $e^x$.
- Требуется представить ответ в общем виде, желательно через знак суммы ($\sum$).

**Идея решения** Основной прием — **почленное умножение** известного разложения на $x$. Это увеличивает степень каждого слагаемого на единицу.

**Подробное решение**

1. Вспоминаем стандартную формулу для $e^x$: $e^x = \sum_{k=0}^{n} \frac{x^k}{k!} + o(x^n)$.
2. Умножаем всё выражение на $x$: $f(x) = x \cdot \left( \sum_{k=0}^{n} \frac{x^k}{k!} + o(x^n) \right) = \sum_{k=0}^{n} \frac{x^{k+1}}{k!} + o(x^{n+1})$.
3. Преподаватель просит сделать так, чтобы конечная степень была $n$ или общий член был выражен через $x^n$.
4. Сделаем замену индекса: пусть $m = k + 1$. Если $k$ менялось от $0$ до $n-1$, то $m$ меняется от $1$ до $n$.
5. Получаем: $f(x) = \sum_{k=1}^{n} \frac{x^k}{(k-1)!} + o(x^n)$.

**Типичные ошибки**

- Забыть умножить остаточный член $o(x^n)$ на $x$ (он превращается в $o(x^{n+1})$).
- Ошибиться в начальном индексе суммы при сдвиге степеней.

**Ответ** $f(x) = \sum_{k=1}^{n} \frac{x^k}{(k-1)!} + o(x^n)$

### Задача №9 (b): Разложение $f(x) = x^2 \ln(3 + 2x)$

**Условие** Разложить функцию $f(x) = x^2 \ln(3 + 2x)$ по формуле Маклорена.

**Что требуется найти** Разложение до порядка $n$ включительно.

**Первые мысли и наблюдения**

- Внутри логарифма стоит $3 + 2x$, а стандартная формула требует вида $\ln(1 + \dots)$.
- Важно: "единичка выделяется именно **вынесением за скобки**", а не простым разбиением числа (например, $3 = 2+1$ писать нельзя, так как $2+x$ не стремится к нулю).

**Идея решения** Применить «трюк с вынесением константы», чтобы привести аргумент логарифма к стандартному виду $\ln(1 + t)$, где $t \to 0$.

**Подробное решение**

1. Преобразуем логарифм: $\ln(3 + 2x) = \ln\left(3 \left(1 + \frac{2}{3}x\right)\right) = \ln 3 + \ln\left(1 + \frac{2}{3}x\right)$.
2. Используем стандартную формулу $\ln(1 + t) = \sum_{k=1}^{n} \frac{(-1)^{k-1} t^k}{k} + o(t^n)$, где $t = \frac{2}{3}x$.
3. Подставляем: $\ln(3 + 2x) = \ln 3 + \sum_{k=1}^{n} \frac{(-1)^{k-1} (\frac{2}{3}x)^k}{k} + o(x^n)$.
4. Умножаем на $x^2$: $f(x) = x^2 \ln 3 + \sum_{k=1}^{n} \frac{(-1)^{k-1} 2^k x^{k+2}}{k \cdot 3^k} + o(x^{n+2})$.
5. Заметим, что слагаемое $x^2 \ln 3$ стоит отдельно, а сумма начинается сразу с $x^3$.

**Ответ** $f(x) = x^2 \ln 3 + \sum_{k=1}^{n} \frac{(-1)^{k-1} 2^k x^{k+2}}{k \cdot 3^k} + O(x^{n+3})$ (здесь использован остаток в форме $O$-большое для демонстрации точности).

### Задача №9 (d): Разложение $f(x) = (2x^2 + 5) \sin 3x$

**Условие** Разложить $f(x) = (2x^2 + 5) \sin 3x$ по формуле Маклорена.

**Идея решения** Раскрыть скобки и работать с двумя суммами: одна для $5 \sin 3x$, другая для $2x^2 \sin 3x$. Важно отследить четность степеней.

**Подробное решение**

1. Раскрываем скобки: $f(x) = 5 \sin 3x + 2x^2 \sin 3x$.
2. Расписываем $\sin 3x$: $\sin 3x = \sum_{k=1}^{n} \frac{(-1)^{k-1} (3x)^{2k-1}}{(2k-1)!} + o(x^{2n})$.
3. Первая часть: $5 \sum_{k=1}^{n} \frac{(-1)^{k-1} 3^{2k-1} x^{2k-1}}{(2k-1)!}$ — здесь только **нечетные** степени ($x^1, x^3, x^5...$).
4. Вторая часть: $2x^2 \sum \dots = \sum_{k=1}^{n} \frac{2 \cdot (-1)^{k-1} 3^{2k-1} x^{2k+1}}{(2k-1)!}$ — здесь только **четные** степени ($x^3, x^5, x^7...$ стоп, нет, $2k+1$ — это тоже нечетные!).
5. **Важное замечание преподавателя**: "Нельзя ли их сложить?". При раскрытии через многоточие видно, что подобные слагаемые будут при одинаковых нечетных степенях $x$.
6. Например, коэффициент при $x^3$:
    - От первой суммы: $5 \cdot \frac{(-1)^{1} 3^3}{3!}$ (при $k=2$)
    - От второй суммы: $2 \cdot \frac{(-1)^0 3^1}{1!}$ (при $k=1$).
7. Итоговый коэффициент собирается из двух частей для каждой нечетной степени.

**Ответ** Результат представляется в виде суммы, где коэффициенты при $x^{2n-1}$ пересчитаны с учетом вклада обеих частей.

### Задача №11 (b): Вычисление предела $\lim_{x\to0} \frac{\sqrt{1 + 3x} - \sqrt{1 + 2x}}{x^2}$

**Условие** Вычислите предел, используя разложения Маклорена.

**Что требуется найти** Значение предела при $x \to 0$.

**Первые мысли и наблюдения**

- "Здесь должно быть легкое дежавю после первого семестра".
- Если использовать только первые эквиваленты ($\sqrt{1+3x} \sim 1+x$ и $\sqrt{1+2x} \sim 1+x$), то в числителе получится $1+x - (1+x) = 0$, что дает неопределенность $o(x)/x^2$.
- **Вывод**: эквиваленты "не сработают", нужно расписывать до второго порядка ($x^2$), так как в знаменателе $x^2$.

**Идея решения** "Усилить эквивалент": расписать по формуле $(1+x)^\alpha = 1 + \alpha x + \frac{\alpha(\alpha-1)}{2}x^2 + o(x^2)$.

**Подробное решение**

1. Разложение для $\sqrt{1+3x}$:
    - $\alpha = 1/3, t = 3x$.
    - $(1+3x)^{1/3} = 1 + \frac{1}{3}(3x) + \frac{\frac{1}{3}(\frac{1}{3}-1)}{2}(3x)^2 + o(x^2)$
    - $= 1 + x + \frac{\frac{1}{3}(-\frac{2}{3})}{2} \cdot 9x^2 = 1 + x - x^2 + o(x^2)$.
2. Разложение для $\sqrt{1+2x}$:
    - $\alpha = 1/2, t = 2x$.
    - $(1+2x)^{1/2} = 1 + \frac{1}{2}(2x) + \frac{\frac{1}{2}(\frac{1}{2}-1)}{2}(2x)^2 + o(x^2)$
    - $= 1 + x + \frac{\frac{1}{2}(-\frac{1}{2})}{2} \cdot 4x^2 = 1 + x - \frac{1}{2}x^2 + o(x^2)$.
3. Подставляем в предел:
    - $\lim_{x\to0} \frac{(1 + x - x^2) - (1 + x - \frac{1}{2}x^2) + o(x^2)}{x^2}$.
4. Сокращаем единицы и иксы:
    - $\lim_{x\to0} \frac{-x^2 + \frac{1}{2}x^2 + o(x^2)}{x^2} = \lim_{x\to0} \frac{-0.5x^2 + o(x^2)}{x^2}$.
5. Результат: $-1/2$.

**Ответ** $-1/2$

## Методы и техники, использованные на занятии

### Метод выделения единицы в логарифме

- **Когда применять**: Если аргумент логарифма не вида $(1+x)$.
- **Алгоритм**: Вынести константу за скобки: $\ln(a+bx) = \ln a + \ln(1 + \frac{b}{a}x)$.
- **Важно**: $a$ должна быть положительной, а $\frac{b}{a}x \to 0$.

### Техника "Умного перемножения" скобок

- **Когда применять**: При перемножении двух разложений (например, $\sqrt{1+x^2} \cdot \cos x$).
- **Алгоритм**: Перемножать только те слагаемые, произведение которых дает степень не выше требуемой (например, $x^4$). Все остальные "съедаются" остаточным членом $o(x^4)$.
- **В задачах**: №11(c).

## Полезные приемы и "фишки"

- **Определение порядка разложения**: "Знаменатель указывает, до какого порядка раскладывать". Если внизу $x^4$, числитель нужно тянуть до $x^4$.
- **Свойство $o$-малого**: $x^2 \cdot o(x^n) = o(x^{n+2})$.
- **Визуализация**: Чем выше степень многочлена Тейлора, тем на более широком промежутке он "прилипает" к графику исходной функции (особенно наглядно для периодических функций типа $\sin x$ и $\cos x$).

## Вопросы студентов и ответы

- **Вопрос**: Можно ли использовать правило Лопиталя в №11?
- **Ответ**: Можно, но придется дифференцировать несколько раз (например, 3-4 раза), что гораздо более трудоемко и ведет к ошибкам. Тейлор в таких случаях эффективнее.
- **Вопрос**: Зачем переписывать суммы через $k-1$ или $k-2$?
- **Ответ**: Чтобы собрать все коэффициенты при одинаковых степенях $x^k$ в одну скобку и получить красивый общий вид многочлена.
