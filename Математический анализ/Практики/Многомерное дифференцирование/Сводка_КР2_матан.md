# Сводка по КР №2 (матан, 2 семестр)

**Темы:** экстремум ФНП, определённый интеграл, площади и длины, несобственный интеграл.

## 1. Экстремум функции нескольких переменных

### Алгоритм

1. **Найти стационарные точки** — решить систему $\nabla f = 0$:
   $$\begin{cases} f_x'(x, y) = 0, \\ f_y'(x, y) = 0. \end{cases}$$

2. **Вычислить матрицу Гессе** в каждой точке:
   $$H = \begin{pmatrix} f_{xx}'' & f_{xy}'' \\ f_{yx}'' & f_{yy}'' \end{pmatrix}.$$

3. **Применить критерий Сильвестра** к $d^2 f$.

### Критерий Сильвестра (для $n = 2$)

| Условие | Заключение |
|---|---|
| $\Delta_1 > 0,\ \Delta_2 > 0$ | строгий минимум |
| $\Delta_1 < 0,\ \Delta_2 > 0$ | строгий максимум |
| $\Delta_2 < 0$ | седло (нет экстремума) |
| $\Delta_2 = 0$ | критерий **не работает**, нужен другой метод |

Здесь $\Delta_1 = f_{xx}''$, $\Delta_2 = \det H = f_{xx}'' f_{yy}'' - (f_{xy}'')^2$.

### Особые случаи

- **$\Delta_2 < 0$, но $\Delta_1 = 0$.** Критерий Сильвестра формально «не подходит» под шаблон, но для $n = 2$ работает следствие: $\det H < 0 \Rightarrow$ форма знакопеременна $\Rightarrow$ седло.
- **$\det H = 0$ (полуопределённая форма).** Достаточное условие через $d^2 f$ не даёт ответа. Нужен прямой анализ: смотреть знак приращения $\Delta f$ на конкретных направлениях, использовать старшие дифференциалы или разложение Тейлора.
- **Знакопеременность через прямую проверку:** взять два вектора $h_1, h_2 \neq 0$ так, чтобы $d^2 f(h_1) > 0$, $d^2 f(h_2) < 0$ — этого достаточно для вывода о седле.

### Пример

$f(x, y) = x^3 + 8y^3 - 6xy + 5$.

Стационарные точки: $\begin{cases} 3x^2 - 6y = 0 \\ 24y^2 - 6x = 0 \end{cases} \Rightarrow (0, 0),\ (1, 1/2)$.

Гессиан: $f_{xx} = 6x$, $f_{yy} = 48y$, $f_{xy} = -6$.

- $(0, 0)$: $\det H = -36 < 0$ — седло.
- $(1, 1/2)$: $\Delta_1 = 6 > 0$, $\Delta_2 = 6 \cdot 24 - 36 = 108 > 0$ — минимум, $f_{\min} = 4$.

## 2. Определённый интеграл

### Основные приёмы

**Замена переменной:** если делаешь $x = \varphi(t)$, то:
- $dx = \varphi'(t)\, dt$;
- пределы пересчитываются: $a \to \varphi^{-1}(a)$, $b \to \varphi^{-1}(b)$;
- **главная ошибка** — забыть пересчитать пределы.

**Интегрирование по частям:** $\int_a^b u\, dv = uv\Big|_a^b - \int_a^b v\, du$.

### Тригонометрические замены

| Подынтегральное выражение | Замена |
|---|---|
| $\sqrt{a^2 - x^2}$ | $x = a\sin t$ или $x = a\cos t$ |
| $\sqrt{a^2 + x^2}$ | $x = a\,\mathrm{tg}\, t$ |
| $\sqrt{x^2 - a^2}$ | $x = \dfrac{a}{\cos t}$ |

При замене $x = a\sin t$: пределы из $[-a, a]$ переходят в $[-\pi/2, \pi/2]$, $\arcsin$ возрастает.

При замене $x = a\cos t$: пределы **переворачиваются** ($\arccos$ убывает) — это нормально, либо оставляешь $\int_\beta^\alpha$, либо разворачиваешь со знаком минус.

### Пример

$\int_0^1 \dfrac{x^2}{\sqrt{4-x^2}}\, dx$. Замена $x = 2\sin t$, $dx = 2\cos t\, dt$, $\sqrt{4-x^2} = 2\cos t$.

Пределы: $x = 0 \to t = 0$, $x = 1 \to t = \pi/6$.

$$\int_0^{\pi/6} \frac{4\sin^2 t}{2\cos t} \cdot 2\cos t\, dt = 4\int_0^{\pi/6} \sin^2 t\, dt = 2\int_0^{\pi/6} (1 - \cos 2t)\, dt = \frac{\pi}{3} - \frac{\sqrt{3}}{2}.$$

## 3. Площади

### Декартовы координаты

**Формула:** $S = \displaystyle\int_a^b (g(x) - f(x))\, dx$, где $f \leqslant g$ на $[a, b]$.

**Алгоритм:**
1. Найти точки пересечения границ.
2. Определить, кто сверху, кто снизу.
3. Если несколько кусков — считать отдельно.

**Когда интегрировать по $dy$:** когда $x$ выражается через $y$ проще, чем наоборот. Признаки:
- в уравнении есть $y^2$ (или степень $y$ выше первой);
- по $x$ пришлось бы извлекать корень и работать с двумя ветвями.

Тогда: $S = \displaystyle\int_c^d (\varphi_{\text{прав}}(y) - \varphi_{\text{лев}}(y))\, dy$.

### Полярные координаты

**Формула:** $S = \dfrac{1}{2}\displaystyle\int_\alpha^\beta \rho^2(\varphi)\, d\varphi$.

**Алгоритм определения пределов:**
1. Найти, где $\rho(\varphi) = 0$ — это границы «лепестков».
2. $\rho \geqslant 0$ на $[\alpha, \beta]$.
3. Проверить, что фигура обходится **один раз**, не два.

**Типовые случаи:**

| Кривая | Пределы | Площадь |
|---|---|---|
| Кардиоида $\rho = a(1 \pm \cos\varphi)$ | $[0, 2\pi]$ | $\tfrac{3}{2}\pi a^2$ |
| Окружность $\rho = a\cos\varphi$ | $[-\pi/2, \pi/2]$ | $\tfrac{\pi a^2}{4}$ |
| Роза $\rho = a\cos(k\varphi)$, $k$ нечётно | один лепесток на $[-\pi/(2k), \pi/(2k)]$, всего $k$ лепестков | — |
| Роза $\rho = a\cos(k\varphi)$, $k$ чётно | $2k$ лепестков | — |

### Параметрические кривые

Если кривая задана $x = x(t),\ y = y(t)$, $t \in [\alpha, \beta]$, ограничивает фигуру с осью $Ox$:
$$S = \left|\int_\alpha^\beta y(t)\, x'(t)\, dt\right|.$$

### Пример

$y^2 = 2x + 1$ и $y = x - 1$. Точки пересечения: $(0, -1)$ и $(4, 3)$.

Удобнее по $dy$: парабола $x = \tfrac{y^2-1}{2}$ слева, прямая $x = y+1$ справа на $y \in [-1, 3]$.

$$S = \int_{-1}^{3} \left((y+1) - \frac{y^2-1}{2}\right) dy = \frac{1}{2}\int_{-1}^{3} (-y^2 + 2y + 3)\, dy = \frac{16}{3}.$$

## 4. Длины дуг

### Базовый принцип

Длина дуги — интеграл от «скорости» в подходящих координатах.

### Декартовы координаты $y = f(x)$

$$L = \int_a^b \sqrt{1 + (f'(x))^2}\, dx.$$

**Главный приём:** функция почти всегда подобрана так, чтобы $1 + (f')^2$ свернулось в полный квадрат.

Типичный паттерн: $f' = A - B$, тогда $(f')^2 = A^2 - 2AB + B^2$, и если $2AB = 1$, то $1 + (f')^2 = (A + B)^2$. Средний член меняет знак с $-$ на $+$.

**Главная ловушка:** потерять полный квадрат, забыв сменить знак среднего члена.

### Параметрические кривые

$$L = \int_\alpha^\beta \sqrt{(x'(t))^2 + (y'(t))^2}\, dt.$$

### Полярные координаты

$$L = \int_\alpha^\beta \sqrt{\rho^2 + (\rho')^2}\, d\varphi.$$

### Типовые примеры

| Кривая | Длина |
|---|---|
| Циклоида $x = a(t-\sin t),\ y = a(1-\cos t)$, одна арка | $8a$ |
| Кардиоида $\rho = a(1-\cos\varphi)$ | $8a$ |
| Астроида $x = a\cos^3 t,\ y = a\sin^3 t$ | $6a$ |
| Окружность $\rho = a\cos\varphi$ | $\pi a$ |
| Лог. спираль $x = e^t\cos t,\ y = e^t\sin t$, $t \in [0, T]$ | $\sqrt{2}(e^T - 1)$ |

### Пример

$y = \tfrac{x^3}{3} + \tfrac{1}{4x}$ на $[1, 2]$.

$y' = x^2 - \tfrac{1}{4x^2}$. $(y')^2 = x^4 - \tfrac{1}{2} + \tfrac{1}{16x^4}$.

$1 + (y')^2 = x^4 + \tfrac{1}{2} + \tfrac{1}{16x^4} = \left(x^2 + \tfrac{1}{4x^2}\right)^2$.

$$L = \int_1^2 \left(x^2 + \frac{1}{4x^2}\right) dx = \left[\frac{x^3}{3} - \frac{1}{4x}\right]_1^2 = \frac{59}{24}.$$

## 5. Несобственный интеграл — вычисление

### Эталоны (нужно знать наизусть)

**На бесконечности:**
$$\int_1^{+\infty} \frac{dx}{x^\alpha} \text{ сходится} \iff \alpha > 1.$$

**В нуле:**
$$\int_0^1 \frac{dx}{x^\alpha} \text{ сходится} \iff \alpha < 1.$$

**Эталон с логарифмом:**
$$\int_2^{+\infty} \frac{dx}{x \ln^\beta x} \text{ сходится} \iff \beta > 1.$$

### Алгоритм вычисления

1. Найти все особенности — особые точки и/или бесконечные пределы.
2. Если особенностей больше одной — **разбить** интеграл по обычной точке.
3. Каждый кусок — заменой/по частям свести к обычному интегралу с параметром $\omega$, затем перейти к пределу.

### Правило разбиения

**Одна особенность на кусок.** Точки разбиения — обычные (не особые).

| Ситуация | Стандартное разбиение |
|---|---|
| $\int_0^{+\infty}$ с особенностями в $0$ и $+\infty$ | $\int_0^1 + \int_1^{+\infty}$ |
| Особенность $x_0$ внутри $[a, b]$ | $\int_a^{x_0} + \int_{x_0}^{b}$ |

Интеграл сходится $\iff$ оба куска сходятся.

### Пример

$\int_1^{+\infty} \dfrac{\ln x}{x^2}\, dx$. По частям: $u = \ln x$, $dv = \tfrac{dx}{x^2}$, $du = \tfrac{dx}{x}$, $v = -\tfrac{1}{x}$.

$$\int_1^{+\infty} \frac{\ln x}{x^2}\, dx = \left[-\frac{\ln x}{x}\right]_1^{+\infty} + \int_1^{+\infty} \frac{dx}{x^2} = 0 + 1 = 1.$$

При $x \to +\infty$: $\tfrac{\ln x}{x} \to 0$ (логарифм медленнее любой степени, можно доказать через правило Лопиталя).

## 6. Несобственный интеграл — исследование сходимости

### Когда использовать какой признак

| Тип функции | Метод |
|---|---|
| Знакопостоянная | **Признак сравнения** с эталоном |
| Осциллирующая ($\sin$, $\cos$ в произведении с убывающей функцией) | **Признак Дирихле/Абеля** |
| Произведение знакопостоянной и осциллирующей | Дирихле |

**Ключевое правило:** Дирихле — про осцилляции. Где осцилляций нет — там сравнение.

### Признак сравнения

Для функций $0 \leqslant f(x) \leqslant g(x)$:
- $\int g$ сходится $\Rightarrow$ $\int f$ сходится;
- $\int f$ расходится $\Rightarrow$ $\int g$ расходится.

В предельной форме: $f \sim g$ при $x \to b^- \Rightarrow$ интегралы сходятся/расходятся одновременно.

**Главная ошибка:** оценить **расходящимся** эталоном сверху — ничего не доказывает.

### Признак Дирихле

$\int_a^b f(x) g(x)\, dx$ сходится, если:
1. **Первообразная $f$ ограничена** на $[a, b)$: $\left|\int_a^A f\, dx\right| \leqslant M$ для всех $A \in [a, b)$;
2. $g$ **монотонна** и $g(x) \to 0$ при $x \to b^-$.

**Подвох:** первообразная $f$ ограничена, а не сама $f$. Поэтому Дирихле работает для $f = \sin x$, $\cos x$, $\cos kx$, но **не работает** для $f = |\sin x|$ — у модуля первообразная неограничена.

### Признак Абеля

$\int_a^b f(x) g(x)\, dx$ сходится, если:
1. $\int_a^b f\, dx$ **сходится**;
2. $g$ **монотонна и ограничена**.

Слабее Дирихле, но иногда удобен (например, когда $g = \mathrm{arctg}\, x$ — ограниченная и монотонная).

### Абсолютная и условная сходимость

- **Абсолютная сходимость:** сходится $\int |f|\, dx$. Из абсолютной следует обычная.
- **Условная:** $\int f\, dx$ сходится, но $\int |f|\, dx$ расходится.

### Шаблон для исследования осциллирующего интеграла

Типа $\int_a^{+\infty} \tfrac{\sin x}{x^\alpha}\, dx$ или $\int_a^{+\infty} \tfrac{\cos x}{\ln x}\, dx$.

**Шаг 1: Сходимость.** Применить Дирихле:
- $\sin x$ или $\cos x$ — первообразные ограничены;
- $\tfrac{1}{x^\alpha}$ или $\tfrac{1}{\ln x}$ — монотонно убывает к $0$.

**Шаг 2: Абсолютная сходимость.** Стандартный приём через $\sin^2$ или $\cos^2$:

$$\frac{|\sin x|}{x^\alpha} \geqslant \frac{\sin^2 x}{x^\alpha} = \frac{1 - \cos 2x}{2x^\alpha} = \frac{1}{2x^\alpha} - \frac{\cos 2x}{2x^\alpha}.$$

- Первое слагаемое — эталон $\int \tfrac{dx}{x^\alpha}$;
- Второе — сходится по Дирихле.

Разность «расходящееся $-$ сходящееся» расходится $\Rightarrow$ $\int \tfrac{\sin^2 x}{x^\alpha}\, dx$ расходится $\Rightarrow$ $\int \tfrac{|\sin x|}{x^\alpha}\, dx$ — тем более.

**Шаг 3: Вывод.**
- Если первое слагаемое расходится — модуль не интегрируем, **условная сходимость**.
- Если первое слагаемое сходится (например, $\alpha > 1$) — для абсолютной сходимости использовать прямую оценку $\tfrac{|\sin x|}{x^\alpha} \leqslant \tfrac{1}{x^\alpha}$.

### Сводная таблица для $\int_1^{+\infty} \tfrac{\sin x}{x^\alpha}\, dx$

| $\alpha$ | Поведение |
|---|---|
| $\alpha > 1$ | абсолютно сходится |
| $0 < \alpha \leqslant 1$ | условно сходится |
| $\alpha = 0$ | расходится (нет предела) |
| $\alpha < 0$ | расходится |

### Полезные замены для исследования

| Исходный вид | Замена |
|---|---|
| $\sin(x^2)$, $\cos(x^2)$ (Френель) | $t = x^2$ |
| $f(\ln x) \cdot \tfrac{dx}{x}$ | $t = \ln x$ |
| $f(e^x) \cdot e^x\, dx$ | $t = e^x$ |

### Раскладка степеней тригонометрии

$$\sin^2 x = \frac{1 - \cos 2x}{2}, \quad \cos^2 x = \frac{1 + \cos 2x}{2}.$$

$$\sin^3 x = \frac{3\sin x - \sin 3x}{4}, \quad \cos^3 x = \frac{3\cos x + \cos 3x}{4}.$$

Используется, чтобы свести интегралы вида $\int \tfrac{\sin^n x}{(\cdot)}\, dx$ к линейной комбинации стандартных $\int \tfrac{\sin kx}{(\cdot)}\, dx$, к каждому из которых применим Дирихле.

### Пример

$\int_1^{+\infty} \dfrac{\sin x}{\sqrt{x}}\, dx$.

**Сходимость:** по Дирихле — первообразная $\sin x$ ограничена, $\tfrac{1}{\sqrt{x}}$ монотонно $\searrow 0$. Сходится.

**Абсолютная:** $\tfrac{|\sin x|}{\sqrt{x}} \geqslant \tfrac{\sin^2 x}{\sqrt{x}} = \tfrac{1}{2\sqrt{x}} - \tfrac{\cos 2x}{2\sqrt{x}}$.

Первое — расходится ($\alpha = 1/2 \leqslant 1$). Второе — сходится по Дирихле. Разность расходится $\Rightarrow$ модуль не интегрируем.

**Итог:** условная сходимость.

## Главные ловушки (то, на чём чаще всего ошибаются)

1. **Экстремум ФНП.** Использовать Сильвестр при $\det H = 0$ — критерий не работает. При $\det H < 0$ и любых $\Delta_1$ (включая $\Delta_1 = 0$) — седло.

2. **Замена в опр. интеграле.** Забыть пересчитать пределы. Лайфхак: сразу после записи замены написать новые пределы на полях.

3. **Полярная площадь.** Забыть про $\tfrac{1}{2}$. Неправильно выбрать пределы — двойной обход даёт удвоенную площадь.

4. **Длина дуги.** Потерять полный квадрат под корнем — арифметическая ошибка, при которой ответ получается «правдоподобный», но неверный.

5. **Несобственный интеграл.** Думать, что «функция $\to 0$» = «интеграл сходится». Контрпример: $\int_1^{+\infty} \tfrac{dx}{\sqrt{x}}$ расходится, а $\tfrac{1}{\sqrt{x}} \to 0$.

6. **Сходимость.** Использовать оценку расходящимся эталоном сверху — ничего не доказывает.

7. **Дирихле.** Применять к модулю $|\sin x|$ или $|\cos x|$ — у них первообразная неограниченна, признак не работает.

8. **Особенности.** Не разбить интеграл на куски при наличии нескольких особенностей — формально признаки применять нельзя.
