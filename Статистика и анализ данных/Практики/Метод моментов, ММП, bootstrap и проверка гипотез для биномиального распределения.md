# Метод моментов, ММП, bootstrap и проверка гипотез для биномиального распределения

## Обзор семинара

На семинаре рассматривается задача оценки параметра $p$ биномиального распределения $\mathrm{Bin}(m, p)$, где $m$ фиксировано и известно, а $p$ неизвестно. Разбираются два метода точечного оценивания: метод моментов и метод максимального правдоподобия. Затем строится доверительный интервал для $p$ методом bootstrap. Наконец, формулируется и проверяется статистический критерий для параметра $p$: двусторонний, правосторонний и левосторонний, с разбором понятия p-value.

---

## Методы и подходы

### 1. Метод моментов (ММ)

**Когда применять:** когда нужно оценить параметры распределения, приравнивая теоретические моменты к выборочным. Особенно прост, если теоретический момент выражается через параметр явной формулой.

**Алгоритм:**
1. Выписать теоретическое математическое ожидание (или нужный момент) как функцию неизвестного параметра.
2. Заменить теоретический момент его выборочным аналогом (выборочное среднее $\bar{x}$).
3. Решить уравнение относительно параметра — получить оценку $\hat{\theta}$.

**Пример (биномиальное распределение $\mathrm{Bin}(m, p)$):**

Теоретическое математическое ожидание:
$$\mathbb{E}[X] = mp.$$

Приравниваем к выборочному среднему:
$$\bar{x} = m\hat{p}.$$

Решаем относительно $\hat{p}$:
$$\hat{p}_{\mathrm{MM}} = \frac{\bar{x}}{m} = \frac{1}{nm}\sum_{i=1}^{n} x_i.$$

**ЛОВУШКИ:**
- Не забыть разделить именно на $m$ (параметр биномиального распределения), а не на $n$ (объём выборки).
- Метод моментов даёт одно уравнение, когда неизвестен один параметр — если параметров два, нужны два момента.

**ПРИЗНАКИ ПРИМЕНИМОСТИ:**
- Параметр входит явно в формулу для момента.
- Уравнение, связывающее момент и параметр, легко решается аналитически.

---

### 2. Метод максимального правдоподобия (ММП)

**Когда применять:** универсальный метод для параметрического оценивания; как правило, даёт более эффективные оценки, чем метод моментов.

**Алгоритм:**
1. Записать функцию правдоподобия $L(p) = \prod_{i=1}^{n} P(X = x_i \mid p)$.
2. Перейти к логарифму правдоподобия $\ell(p) = \ln L(p)$ (произведение заменяется суммой).
3. Упростить $\ell(p)$, выделив части, зависящие от $p$, и константы.
4. Взять производную $\ell'(p)$, приравнять нулю.
5. Решить уравнение относительно $p$ — получить $\hat{p}_{\mathrm{MLE}}$.

**Пример (биномиальное распределение $\mathrm{Bin}(m, p)$):**

Вероятность одного наблюдения:
$$P(X = k) = \binom{m}{k} p^k (1-p)^{m-k}.$$

Функция правдоподобия по независимой выборке $x_1, \ldots, x_n$:
$$L(p) = \prod_{i=1}^{n} \binom{m}{x_i} p^{x_i} (1-p)^{m - x_i}.$$

Логарифм правдоподобия:
$$\ell(p) = \sum_{i=1}^{n} \left[ \ln \binom{m}{x_i} + x_i \ln p + (m - x_i) \ln(1-p) \right].$$

Член $\sum_{i=1}^{n} \ln \binom{m}{x_i}$ не зависит от $p$ — это константа, её производная равна нулю.

Берём производную по $p$:
$$\ell'(p) = \sum_{i=1}^{n} \frac{x_i}{p} - \sum_{i=1}^{n} \frac{m - x_i}{1-p}.$$

Приравниваем нулю:
$$\frac{\sum_{i=1}^{n} x_i}{p} = \frac{\sum_{i=1}^{n}(m - x_i)}{1-p}.$$

Раскрываем:
$$(1-p)\sum_{i=1}^{n} x_i = p \sum_{i=1}^{n}(m - x_i) = p\left(mn - \sum_{i=1}^{n} x_i\right).$$

$$\sum_{i=1}^{n} x_i = p \cdot mn.$$

$$\hat{p}_{\mathrm{MLE}} = \frac{\sum_{i=1}^{n} x_i}{mn} = \frac{\bar{x}}{m}.$$

**Вывод:** для биномиального распределения $\hat{p}_{\mathrm{MM}} = \hat{p}_{\mathrm{MLE}} = \dfrac{\bar{x}}{m}$ — оценки совпадают.

**ЛОВУШКИ:**
- Не забыть, что $\ln \binom{m}{x_i}$ — константа относительно $p$; её не нужно дифференцировать.
- При упрощении $\ell'(p)$: $\sum_{i=1}^{n} m$ выносится как $mn$, поскольку $m$ не зависит от $i$.
- Знак: производная $\ln(1-p)$ по $p$ равна $-\frac{1}{1-p}$; минус на минус даёт плюс перед суммой.

**ПРИЗНАКИ ПРИМЕНИМОСТИ:**
- Есть параметрическая модель с известной функцией вероятностей/плотности.
- Функция правдоподобия дифференцируема по параметру.

### 3. Bootstrap доверительный интервал

**Когда применять:** когда аналитическое распределение оценки неизвестно или труднодоступно, и нужно построить доверительный интервал эмпирически.

**Алгоритм (параметрический bootstrap для дискретного распределения):**
1. По исходной выборке $x_1, \ldots, x_n$ построить эмпирическое распределение: для каждого значения $v$ вероятность $\hat{P}(X = v) = \frac{\#\{i: x_i = v\}}{n}$.
2. Зафиксировать seed для воспроизводимости.
3. Сгенерировать $B$ bootstrap-выборок объёма $n$ из этого эмпирического распределения.
4. Для каждой bootstrap-выборки вычислить оценку $\hat{\theta}^{*(b)}$ (в нашем случае $\hat{p}^{*(b)} = \bar{x}^{*(b)} / m$).
5. Построить вспомогательную выборку:
$$z^{*(b)} = \sqrt{n}\left(\hat{\theta}^{*(b)} - \hat{\theta}\right), \quad b = 1, \ldots, B,$$
где $\hat{\theta}$ — оценка по исходной выборке.
6. Найти выборочные квантили вспомогательной выборки: $q_{\alpha/2}$ и $q_{1-\alpha/2}$.
7. Доверительный интервал уровня $1-\alpha$:
$$\left[\hat{\theta} - \frac{q_{1-\alpha/2}}{\sqrt{n}},\quad \hat{\theta} - \frac{q_{\alpha/2}}{\sqrt{n}}\right].$$

**Пример (код на Python):**

```python
import numpy as np
from scipy.stats import rv_discrete

# Параметры задачи
m = 5       # параметр биномиального распределения (фиксирован)
p_real = 0.52
sample_size = 100
np.random.seed(42)

# Исходная выборка
sample = np.random.binomial(m, p_real, size=sample_size)

# Исходный эстиматор
theta_hat = np.mean(sample) / m

# Построение эмпирического распределения
values, counts = np.unique(sample, return_counts=True)
probs = counts / sample_size

emp_dist = rv_discrete(values=(values, probs))

# Bootstrap
bootstrap_size = 1000
bootstrap_samples = emp_dist.rvs(size=(bootstrap_size, sample_size))

# Bootstrap-эстиматоры
bootstrap_estimators = np.mean(bootstrap_samples, axis=1) / m

# Вспомогательная выборка
temp_sample = np.sqrt(sample_size) * (bootstrap_estimators - theta_hat)

# Квантили
significance_level = 0.05
q_left  = np.quantile(temp_sample, significance_level / 2)
q_right = np.quantile(temp_sample, 1 - significance_level / 2)

# Доверительный интервал
left  = theta_hat - q_right / np.sqrt(sample_size)
right = theta_hat - q_left  / np.sqrt(sample_size)

print(f"Доверительный интервал уровня {1 - significance_level:.0%}: [{left:.3f}, {right:.3f}]")
print(f"Реальное p = {p_real}")
```

**Пояснение вспомогательной выборки:** мы знаем, что $\sqrt{n}(\hat{\theta} - \theta) \xrightarrow{d} \mathcal{M}$ для некоторого предельного распределения $\mathcal{M}$, которое нам неизвестно. Метод bootstrap заменяет неизвестное распределение его эмпирической оценкой: мы берём реализации $\sqrt{n}(\hat{\theta}^* - \hat{\theta})$ и по ним оцениваем квантили.

**ЛОВУШКИ:**
- Границы интервала: $\hat{\theta} - q_{1-\alpha/2}/\sqrt{n}$ (левая) и $\hat{\theta} - q_{\alpha/2}/\sqrt{n}$ (правая). Знаки минус стоят в **обоих** концах, но квантили разные: правый квантиль вычитается слева, левый — справа. Легко перепутать.
- `np.unique` с `return_counts=True` возвращает только значения, которые встретились в выборке. Если нолик ни разу не попался — его вероятность равна нулю и он просто не войдёт в список значений.
- Seed задаётся один раз перед генерацией, чтобы результаты были воспроизводимы.

**ПРИЗНАКИ ПРИМЕНИМОСТИ:**
- Распределение оценки трудно найти аналитически.
- Объём выборки достаточно велик для качественной эмпирической аппроксимации.

### 4. Проверка гипотезы о параметре $p$ биномиального распределения

**Когда применять:** когда нужно проверить, равен ли параметр $p$ некоторому конкретному значению $p_0$.

**Постановка задачи:**

$$H_0: p = p_0$$

против одной из трёх альтернатив:
- $H_1^{(1)}: p \neq p_0$ (двусторонняя),
- $H_1^{(2)}: p > p_0$ (правосторонняя),
- $H_1^{(3)}: p < p_0$ (левосторонняя).

**Статистика критерия:**

$$T = \sum_{i=1}^{n} x_i.$$

При истинности $H_0$ каждая $x_i \sim \mathrm{Bin}(m, p_0)$, и они независимы. Сумма независимых биномиальных случайных величин с одинаковым $p$ снова биномиальна:

$$T \sim \mathrm{Bin}(mn,\; p_0).$$

**Критические области и области принятия:**

| Альтернатива | Критическая область | Область принятия $H_0$ |
|---|---|---|
| $p \neq p_0$ (двусторонняя) | $T < q_{\alpha/2}$ или $T > q_{1-\alpha/2}$ | $[q_{\alpha/2},\; q_{1-\alpha/2}]$ |
| $p > p_0$ (правосторонняя) | $T > q_{1-\alpha}$ | $[0,\; q_{1-\alpha}]$ |
| $p < p_0$ (левосторонняя) | $T < q_{\alpha}$ | $[q_\alpha,\; mn]$ |

где квантили берутся из распределения $\mathrm{Bin}(mn, p_0)$.

**Алгоритм:**
1. Задать $p_0$ и уровень значимости $\alpha$.
2. Вычислить наблюдаемое значение статистики: $T_{\mathrm{obs}} = \sum_{i=1}^{n} x_i$.
3. Найти квантили $\mathrm{Bin}(mn, p_0)$ нужных порядков.
4. Проверить, попало ли $T_{\mathrm{obs}}$ в область принятия.
5. Вычислить p-value (см. ниже).
6. Сформулировать вывод.

**Пример (код на Python):**

```python
from scipy.stats import binom

p0 = 0.6
alpha = 0.05

# Наблюдаемое значение статистики
T_obs = np.sum(sample)
N = m * sample_size  # параметр n для Bin(mn, p0)

# Квантили распределения Bin(mn, p0)
q_a2       = binom.ppf(alpha / 2,       N, p0)
q_1_a2     = binom.ppf(1 - alpha / 2,   N, p0)
q_a        = binom.ppf(alpha,            N, p0)
q_1_a      = binom.ppf(1 - alpha,       N, p0)

print(f"T_obs = {T_obs}")
print(f"\nАльтернатива p != p0 (двусторонняя):")
print(f"  Область принятия: [{q_a2}, {q_1_a2}]")
print(f"  Вывод: {'принимаем H0' if q_a2 <= T_obs <= q_1_a2 else 'отвергаем H0'}")

print(f"\nАльтернатива p > p0 (правосторонняя):")
print(f"  Область принятия: [0, {q_1_a}]")
print(f"  Вывод: {'принимаем H0' if T_obs <= q_1_a else 'отвергаем H0'}")

print(f"\nАльтернатива p < p0 (левосторонняя):")
print(f"  Область принятия: [{q_a}, {N}]")
print(f"  Вывод: {'принимаем H0' if T_obs >= q_a else 'отвергаем H0'}")

# p-value
pval_two   = 2 * min(binom.cdf(T_obs, N, p0), 1 - binom.cdf(T_obs - 1, N, p0))
pval_right = 1 - binom.cdf(T_obs - 1, N, p0)
pval_left  = binom.cdf(T_obs, N, p0)

print(f"\np-value (двусторонний):    {pval_two:.4f}")
print(f"p-value (правосторонний):  {pval_right:.4f}")
print(f"p-value (левосторонний):   {pval_left:.4f}")
```

**ЛОВУШКИ:**
- Для правостороннего критерия квантиль порядка $1 - \alpha$ (не $1 - \alpha/2$). Легко перепутать со двусторонним.
- Область принятия для левосторонней альтернативы находится **справа**: $[q_\alpha, mn]$, а критическая область слева. Это контринтуитивно: если мы подозреваем, что $p$ маленькое, то большие значения суммы типичны для $H_0$ и принимаются.
- Максимальное возможное значение статистики — $mn$ (все испытания успешны), а не $n$.
- p-value — **не** вероятность того, что $H_0$ верна. Это вероятность при истинности $H_0$ получить наблюдение **столь же или более экстремальное**, чем фактическое $T_{\mathrm{obs}}$.

**ПРИЗНАКИ ПРИМЕНИМОСТИ:**
- Выборка состоит из независимых наблюдений из $\mathrm{Bin}(m, p)$ с известным $m$.
- Нужно проверить конкретное значение параметра $p$.

## Разбор задач

### Задача. Оценивание параметра $p$ биномиального распределения

**Условие:** Дана выборка $x_1, \ldots, x_n$ из биномиального распределения $\mathrm{Bin}(m, p)$, где $m$ фиксировано и известно, $p$ неизвестно. Найти оценки $p$ методом моментов (а) и методом максимального правдоподобия (б).

**Пункт (а): Метод моментов**

Теоретическое математическое ожидание:
$$\mathbb{E}[X] = mp.$$

Теоретическое соотношение (приравниваем к выборочному среднему):
$$\bar{x} = m\hat{p}.$$

Откуда:
$$\hat{p}_{\mathrm{MM}} = \frac{\bar{x}}{m} = \frac{1}{nm}\sum_{i=1}^{n} x_i.$$

**Пункт (б): Метод максимального правдоподобия**

Вероятность одного наблюдения $x_i$:
$$P(X_i = x_i) = \binom{m}{x_i} p^{x_i}(1-p)^{m-x_i}.$$

Функция правдоподобия (наблюдения независимы):
$$L(p) = \prod_{i=1}^{n} \binom{m}{x_i} p^{x_i}(1-p)^{m-x_i}.$$

Логарифмическая функция правдоподобия:
$$\ell(p) = \sum_{i=1}^{n} \ln\binom{m}{x_i} + \ln p \cdot \sum_{i=1}^{n} x_i + \ln(1-p) \cdot \sum_{i=1}^{n}(m - x_i).$$

Первое слагаемое — константа (не зависит от $p$). Берём производную:
$$\ell'(p) = \frac{\sum_{i=1}^{n} x_i}{p} - \frac{\sum_{i=1}^{n}(m - x_i)}{1-p} = \frac{\sum_{i=1}^{n} x_i}{p} - \frac{mn - \sum_{i=1}^{n} x_i}{1-p}.$$

Приравниваем нулю:
$$(1-p)\sum_{i=1}^{n} x_i = p\left(mn - \sum_{i=1}^{n} x_i\right).$$

$$\sum_{i=1}^{n} x_i - p\sum_{i=1}^{n} x_i = pmn - p\sum_{i=1}^{n} x_i.$$

$$\sum_{i=1}^{n} x_i = pmn.$$

$$\hat{p}_{\mathrm{MLE}} = \frac{\sum_{i=1}^{n} x_i}{mn} = \frac{\bar{x}}{m}.$$

**Ответ:** $\hat{p}_{\mathrm{MM}} = \hat{p}_{\mathrm{MLE}} = \dfrac{\bar{x}}{m}$.

Оба метода дают совпадающие оценки. Это результат особой структуры биномиального распределения.

### Задача. Bootstrap доверительный интервал для $p$

**Условие:** По той же выборке построить 95%-й доверительный интервал для $p$ методом bootstrap.

**Решение (схема):**

1. Вычислить $\hat{p} = \bar{x}/m$.
2. Построить эмпирическое дискретное распределение на наблюдённых значениях $\{v_j\}$ с вероятностями $\hat{P}(X = v_j) = n_j/n$, где $n_j$ — число вхождений $v_j$.
3. Сгенерировать $B = 1000$ bootstrap-выборок объёма $n$.
4. Для каждой вычислить $\hat{p}^{*(b)} = \bar{x}^{*(b)}/m$.
5. Построить вспомогательную выборку: $z^{*(b)} = \sqrt{n}(\hat{p}^{*(b)} - \hat{p})$.
6. Найти $q_{0.025}$ и $q_{0.975}$ вспомогательной выборки.
7. Интервал: $\left[\hat{p} - \dfrac{q_{0.975}}{\sqrt{n}},\quad \hat{p} - \dfrac{q_{0.025}}{\sqrt{n}}\right]$.

### Задача. Проверка гипотезы $H_0: p = 0{,}6$

**Условие:** Дана выборка из $\mathrm{Bin}(5, p)$ объёма $n = 100$. Проверить $H_0: p = 0{,}6$ против трёх альтернатив при $\alpha = 0{,}05$. Реальное $p = 0{,}52$.

**Статистика:** $T = \sum_{i=1}^{n} x_i$. При $H_0$ верна: $T \sim \mathrm{Bin}(500, 0{,}6)$.

Матожидание и дисперсия при $H_0$:
$$\mathbb{E}[T \mid H_0] = 500 \cdot 0{,}6 = 300, \quad \mathrm{Var}[T \mid H_0] = 500 \cdot 0{,}6 \cdot 0{,}4 = 120.$$

Реальное матожидание (при $p = 0{,}52$):
$$\mathbb{E}[T] = 500 \cdot 0{,}52 = 260.$$

Поэтому $T_{\mathrm{obs}} \approx 260$ — значительно ниже 300.

**Двусторонняя альтернатива** ($p \neq 0{,}6$): наблюдаемое значение $\approx 260$ попадает в критическую область (слишком далеко от 300). $H_0$ **отвергается**. p-value $\approx 0$.

**Правосторонняя альтернатива** ($p > 0{,}6$): область принятия $[0, q_{0{,}95}]$. Поскольку $T_{\mathrm{obs}} \approx 260$ мало относительно 300, оно попадает в область принятия. $H_0$ **не отвергается**. p-value $\approx 1$.

Интерпретация: мы проверяли, не меньше ли $p$, чем $p_0 = 0{,}6$. Наши данные ($p_{\mathrm{real}} = 0{,}52 < 0{,}6$) действительно не дают оснований думать, что $p > 0{,}6$, — поэтому логично принять $H_0$.

**Левосторонняя альтернатива** ($p < 0{,}6$): область принятия $[q_{0{,}05}, 500]$. $T_{\mathrm{obs}} \approx 260$ попадает в критическую область (она слева). $H_0$ **отвергается**. p-value $\approx 0$.

Интерпретация: реальное $p = 0{,}52 < 0{,}6$, поэтому тест корректно обнаруживает, что данные несовместимы с $H_0$ в пользу $p < p_0$.

## Справочный лист

**Оценки $p$ для $\mathrm{Bin}(m, p)$:**
$$\hat{p} = \frac{\bar{x}}{m} = \frac{1}{nm}\sum_{i=1}^{n} x_i.$$

**Матожидание и дисперсия $\mathrm{Bin}(m,p)$:**
$$\mathbb{E}[X] = mp, \quad \mathrm{Var}[X] = mp(1-p).$$

**Сумма биномиальных с одним и тем же $p$:**
$$X_i \sim \mathrm{Bin}(m, p),\; \text{независимы} \Rightarrow \sum_{i=1}^{n} X_i \sim \mathrm{Bin}(mn, p).$$

**Логарифм правдоподобия для $\mathrm{Bin}(m,p)$:**
$$\ell(p) = \mathrm{const} + \left(\sum_{i=1}^{n} x_i\right)\ln p + \left(mn - \sum_{i=1}^{n} x_i\right)\ln(1-p).$$

**Bootstrap доверительный интервал (двусторонний, уровень $1-\alpha$):**
$$\hat{\theta} - \frac{q_{1-\alpha/2}^*}{\sqrt{n}} \;\leq\; \theta \;\leq\; \hat{\theta} - \frac{q_{\alpha/2}^*}{\sqrt{n}},$$
где $q_{\alpha/2}^*, q_{1-\alpha/2}^*$ — квантили вспомогательной выборки $\{\sqrt{n}(\hat{\theta}^{*(b)} - \hat{\theta})\}$.

**Критические области для $T \sim \mathrm{Bin}(mn, p_0)$:**

| Альтернатива | Критическая область | p-value |
|---|---|---|
| $p \neq p_0$ | $T < q_{\alpha/2}$ или $T > q_{1-\alpha/2}$ | $2\min(F(T_{\mathrm{obs}}),\, 1 - F(T_{\mathrm{obs}}-1))$ |
| $p > p_0$ | $T > q_{1-\alpha}$ | $1 - F(T_{\mathrm{obs}} - 1)$ |
| $p < p_0$ | $T < q_{\alpha}$ | $F(T_{\mathrm{obs}})$ |

где $F$ — функция распределения $\mathrm{Bin}(mn, p_0)$.

## Типичные ошибки семинара

1. **Деление не на то число.** При вычислении $\hat{p}$ делить нужно на $m$ (параметр распределения), а не на $n$ (объём выборки). Размерность: $x_i$ — число успехов из $m$ испытаний, значит $\hat{p} = \bar{x}/m$.

2. **Потеря константы в логарифме правдоподобия.** Член $\sum \ln\binom{m}{x_i}$ не зависит от $p$, его производная ноль — он исчезает при дифференцировании. Не нужно пытаться его дифференцировать.

3. **Знаки в Bootstrap-интервале.** Обе границы вычисляются как $\hat{\theta}$ минус квантиль, делённый на $\sqrt{n}$. Левая граница: вычитаем **правый** квантиль ($q_{1-\alpha/2}$); правая граница: вычитаем **левый** квантиль ($q_{\alpha/2}$).

4. **Максимум статистики критерия.** Максимальное возможное значение $T = \sum x_i$ равно $mn$, а не $n$.

5. **Путаница сторон критической области.** Для правосторонней альтернативы ($p > p_0$) критическая область — **справа** ($T > q_{1-\alpha}$), область принятия — **слева** ($[0, q_{1-\alpha}]$). Для левосторонней — наоборот.

6. **Интерпретация p-value.** p-value — не вероятность истинности $H_0$. Это вероятность получить значение статистики, **столь же или более экстремальное**, чем наблюдаемое, при условии что $H_0$ верна. Большое p-value означает, что данные типичны для $H_0$; маленькое — что данные нетипичны.

7. **«Принять $H_0$» не означает доказать её.** Принятие нулевой гипотезы означает лишь, что данные не противоречат ей, — аналогия с судом: «не виновен» не означает «заведомо невиновен».
