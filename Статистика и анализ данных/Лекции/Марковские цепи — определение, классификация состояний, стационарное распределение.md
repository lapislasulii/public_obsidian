# Марковские цепи — определение, классификация состояний, стационарное распределение

## Обзор лекции

Лекция посвящена введению в теорию марковских цепей — одного из центральных объектов современной теории вероятностей и математической статистики. Ключевой идеей является марковское свойство: будущее состояние системы определяется только её настоящим состоянием, но не историей. На этой основе строится аппарат переходных матриц и векторов распределения, позволяющий вычислять распределение в произвольный момент времени через степень матрицы. Далее вводится классификация состояний: достижимость, сообщаемость, существенность/несущественность, периодичность, возвратность/невозвратность, нулевость. Изучается блочная структура матрицы перехода, теорема солидарности для неразложимых цепей. В конце рассматриваются стационарное распределение, среднее время первого возврата и понятие эргодичности, включая критерии сильной эргодичности и эргодическую теорему.

---

## Предварительные сведения

- **Случайная величина**: функция на вероятностном пространстве, принимающая числовые значения.
- **Условная вероятность**: $P(A \mid B) = P(A \cap B)/P(B)$.
- **Формула полной вероятности**: $P(A) = \sum_i P(A \mid H_i) P(H_i)$ по полной группе гипотез.
- **Матричное умножение**: $(AB)_{ij} = \sum_k A_{ik} B_{kj}$.
- **НОД (GCD)**: наибольший общий делитель множества натуральных чисел.
- **Отношение эквивалентности**: рефлексивное, симметричное, транзитивное отношение.

## 1. Марковское свойство и определение цепи Маркова

### Определение 1.1 (Цепь Маркова)

Пусть $\{X_n\}_{n \geq 0}$ — последовательность случайных величин, принимающих значения из не более чем счётного множества состояний

$$E = \{1, 2, \ldots, N\} \quad (\text{или бесконечного: } E = \{1, 2, \ldots\}).$$

Последовательность $\{X_n\}$ называется **цепью Маркова**, если для любого $n \geq 1$ и любых $i_{n+1}, i_n, \ldots, i_0 \in E$ выполнено **марковское свойство**:

$$P(X_{n+1} = i_{n+1} \mid X_n = i_n,\ X_{n-1} = i_{n-1},\ \ldots,\ X_0 = i_0) = P(X_{n+1} = i_{n+1} \mid X_n = i_n).$$

МОТИВАЦИЯ: Большинство реальных систем обладают «короткой памятью»: следующее состояние определяется только текущим. Это позволяет математически описывать броуновское движение, очереди, генетические модели, алгоритмы ранжирования веб-страниц.

ИНТУИЦИЯ: Индекс $n$ трактуется как дискретное время. Момент $n$ — «настоящее», момент $n+1$ — «будущее», все предыдущие — «прошлое». Марковское свойство говорит: **будущее зависит только от настоящего**, история не важна.

## 2. Переходная матрица и вектор распределения

### Определение 2.1 (Переходная матрица)

Для цепи Маркова в момент времени $n$ определим **переходную матрицу** $P(n)$ с элементами

$$p_{ij}(n) = P(X_n = j \mid X_{n-1} = i), \quad i, j \in E.$$

Таким образом, $p_{ij}(n)$ — вероятность перейти из состояния $i$ в состояние $j$ на шаге $n$.

**Ключевое свойство переходной матрицы.** Поскольку условная вероятность является вероятностью, сумма по строке равна 1:

$$\sum_{j \in E} p_{ij}(n) = 1 \quad \text{для любого } i.$$

Такая матрица называется **стохастической**.

### Определение 2.2 (Вектор распределения)

**Вектором распределения** в момент времени $n$ называется вектор-столбец

$$\mathbf{p}(n) = \begin{pmatrix} P(X_n = 1) \\ P(X_n = 2) \\ \vdots \\ P(X_n = N) \end{pmatrix}.$$

### Теорема 2.1 (Рекуррентная формула для распределения)

$$\mathbf{p}(n) = P(n)^\top \mathbf{p}(n-1).$$

Доказательство. По формуле полной вероятности:

$$P(X_n = j) = \sum_{i \in E} P(X_n = j \mid X_{n-1} = i)\, P(X_{n-1} = i) = \sum_{i \in E} p_{ij}(n)\, p_i(n-1).$$

Это в точности $j$-я компонента вектора $P(n)^\top \mathbf{p}(n-1)$, так как мы суммируем $j$-й столбец матрицы $P(n)$ (т.е. $j$-ю строку $P(n)^\top$) скалярно с вектором $\mathbf{p}(n-1)$. $\square$

## 3. Однородные цепи Маркова

### Определение 3.1 (Однородная цепь Маркова)

Цепь Маркова называется **однородной**, если переходная матрица не зависит от времени:

$$P(n) = P \quad \text{для всех } n.$$

Для однородной цепи рекуррентная формула принимает вид $\mathbf{p}(n) = P^\top \mathbf{p}(n-1)$. Раскрывая по индукции:

$$\mathbf{p}(n) = (P^\top)^n\, \mathbf{p}(0).$$

МОТИВАЦИЯ: Однородность — естественное предположение для стационарных систем, где «правила игры» не меняются со временем. Математически это резко упрощает анализ: нужно научиться возводить одну матрицу в степень.

ИНТУИЦИЯ: Вся динамика цепи закодирована в единственной матрице $P$. Её $n$-я степень накапливает информацию о переходах за $n$ шагов.

### Теорема 3.1 (Уравнение Колмогорова–Чэпмена)

Введём обозначение $p_{ij}^{(n)} = P(X_n = j \mid X_0 = i)$ — вероятность попасть из $i$ в $j$ за ровно $n$ шагов. Тогда для любых $n, m \geq 0$:

$$p_{ij}^{(n+m)} = \sum_{k \in E} p_{ik}^{(n)}\, p_{kj}^{(m)}.$$

Доказательство. Заметим, что элемент $(i,j)$ матрицы $P^n$ равен $p_{ij}^{(n)}$ (доказывается индукцией по $n$: база $n=1$ очевидна, шаг следует из определения матричного произведения и формулы полной вероятности). Тогда из матричного тождества $P^{n+m} = P^n \cdot P^m$ по определению матричного умножения:

$$(P^{n+m})_{ij} = \sum_{k} (P^n)_{ik} (P^m)_{kj} = \sum_k p_{ik}^{(n)} p_{kj}^{(m)}. \quad \square$$

ИНТУИЦИЯ: Чтобы попасть из $i$ в $j$ за $n+m$ шагов, нужно промежуточно оказаться в каком-то состоянии $k$ ровно через $n$ шагов. Уравнение суммирует вклады всех таких «транзитных» состояний.

## 4. Графическая интерпретация. Классификация состояний

### Определение 4.1 (Граф переходов)

Однородной цепи Маркова сопоставляется **ориентированный граф**, в котором:
- вершины соответствуют состояниям $E$;
- ориентированное ребро $i \to j$ проводится тогда и только тогда, когда $p_{ij} > 0$, и помечается весом $p_{ij}$.

По графу можно однозначно восстановить матрицу перехода и наоборот.

### Определение 4.2 (Достижимость и сообщаемость)

- Состояние $j$ **достижимо** из $i$ (обозначение: $i \to j$), если $\exists\, n \geq 1$ такое, что $p_{ij}^{(n)} > 0$.  
  На языке теории графов: существует ориентированный путь из $i$ в $j$.

- Состояния $i$ и $j$ **сообщаются** (обозначение: $i \leftrightarrow j$), если $i \to j$ и $j \to i$.

МОТИВАЦИЯ: Классификация состояний позволяет понять долгосрочное поведение цепи: в какие состояния система «застрянет», а из каких «убежит».

Отношение $\leftrightarrow$ является **отношением эквивалентности** (рефлексивность, симметричность и транзитивность проверяются непосредственно), поэтому все состояния разбиваются на **классы эквивалентности**.

### Определение 4.3 (Открытый и замкнутый класс)

- Класс $C$ называется **замкнутым**, если нельзя уйти из него в другие классы: $i \in C,\ i \to j \Rightarrow j \in C$.
- Класс $C$ называется **открытым**, если из него можно уйти.

### Определение 4.4 (Поглощающее состояние)

Замкнутый класс, состоящий из одного состояния $i$ (т.е. $p_{ii} = 1$), называется **поглощающим состоянием**.

### Определение 4.5 (Существенные и несущественные состояния)

Состояние $i$ называется **несущественным**, если существует состояние $j$ такое, что $i \to j$, но $j \not\to i$ (можно уйти, но не вернуться).

Состояние $i$ называется **существенным**, если для любого $j$: либо $i \not\to j$, либо $j \to i$.

ИНТУИЦИЯ: Несущественные состояния лежат в открытых классах (из них можно «улететь насовсем»), существенные — в замкнутых (система в них «остаётся»).

### Блочная структура матрицы перехода

Если перенумеровать состояния так, что сначала идут вершины открытого класса, а затем — вершины замкнутых классов $C_1, C_2, \ldots, C_r$, то матрица перехода принимает блочную форму:

$$P = \begin{pmatrix} Q & R_1 & R_2 & \cdots & R_r \\ 0 & P_1 & 0 & \cdots & 0 \\ 0 & 0 & P_2 & \cdots & 0 \\ \vdots & & & \ddots & \vdots \\ 0 & 0 & 0 & \cdots & P_r \end{pmatrix},$$

где $Q$ — подматрица переходов внутри открытого класса, $P_k$ — подматрица переходов внутри $k$-го замкнутого класса, а нули отражают невозможность вернуться из замкнутых классов в открытый.

## 5. Периодические и апериодические состояния

### Определение 5.1 (Период состояния)

Для состояния $i$ определим **период**:

$$d_i = \gcd\{n \geq 1 : p_{ii}^{(n)} > 0\}.$$

Если $d_i = 1$, состояние называется **апериодическим**; если $d_i > 1$ — **периодическим** с периодом $d_i$.

ИНТУИЦИЯ: Рассмотрим все циклы, проходящие через $i$ (т.е. все пути из $i$ в $i$). Если НОД длин всех таких циклов равен 1, состояние апериодично — цепь может вернуться в него в любое достаточно большое время. Если НОД равен $d > 1$, возвращение возможно лишь на шагах, кратных $d$.

ТИПИЧНЫЕ ОШИБКИ: Апериодичность не означает, что система возвращается в $i$ каждый шаг — она лишь означает, что НОД длин всех возможных циклов равен 1.

### Теорема 5.1 (Циклическая структура периодического замкнутого класса)

Пусть $C$ — замкнутый класс, и все его состояния имеют период $d > 1$. Тогда $C$ можно разбить на $d$ непересекающихся подклассов $C_0, C_1, \ldots, C_{d-1}$ таких, что из вершин $C_k$ можно попасть только в вершины $C_{k+1 \pmod{d}}$.

ИНТУИЦИЯ: Состояния замкнутого класса «циклически сменяются»: система последовательно обходит группы $C_0 \to C_1 \to \cdots \to C_{d-1} \to C_0 \to \cdots$

## 6. Возвратные и невозвратные состояния

### Определение 6.1 (Вероятность первого возврата)

Обозначим

$$f_{ii}^{(n)} = P(X_n = i,\ X_{n-1} \neq i,\ \ldots,\ X_1 \neq i \mid X_0 = i)$$

— вероятность **впервые** вернуться в $i$ ровно за $n$ шагов. Полная вероятность возврата:

$$f_{ii} = \sum_{n=1}^{\infty} f_{ii}^{(n)}.$$

### Определение 6.2 (Возвратное и невозвратное состояние)

- Состояние $i$ **возвратно** (рекуррентно), если $f_{ii} = 1$ (с вероятностью 1 вернёмся).
- Состояние $i$ **невозвратно** (транзиентно), если $f_{ii} < 1$.

### Определение 6.3 (Нулевое и ненулевое состояние)

Состояние $i$ называется **нулевым**, если $p_{ii}^{(n)} \to 0$ при $n \to \infty$.

Состояние $i$ называется **ненулевым**, если $p_{ii}^{(n)} \not\to 0$.

### Теорема 6.1 (Критерий возвратности)

Состояние $i$ **возвратно** тогда и только тогда, когда ряд $\displaystyle B_i = \sum_{n=1}^{\infty} p_{ii}^{(n)}$ **расходится**.

При этом $f_{ii} = \dfrac{B_i}{1 + B_i}$ (если $B_i < \infty$), откуда в случае невозвратности $f_{ii} < 1$.

Доказательство (идея). Введём производящие функции $F(z) = \sum_{n \geq 1} f_{ii}^{(n)} z^n$ и $U(z) = \sum_{n \geq 0} p_{ii}^{(n)} z^n$. По формуле полной вероятности $p_{ii}^{(n)} = \sum_{k=1}^n f_{ii}^{(k)} p_{ii}^{(n-k)}$, что даёт $U(z) = 1 + F(z) U(z)$, откуда $F(z) = 1 - 1/U(z)$. При $z \nearrow 1$: если $U(1) = \infty$ (ряд расходится), то $F(1) = f_{ii} = 1$, т.е. состояние возвратно; если $U(1) < \infty$, то $f_{ii} < 1$, т.е. невозвратно. $\square$

### Следствие 6.1 (Невозвратное $\Rightarrow$ нулевое; ненулевое $\Rightarrow$ возвратное)

Пусть состояние $i$ невозвратно. Тогда ряд $\sum p_{ii}^{(n)}$ сходится, и по необходимому условию сходимости ряда $p_{ii}^{(n)} \to 0$, т.е. $i$ нулевое.

Контрапозитивно: если $i$ ненулевое, то $p_{ii}^{(n)} \not\to 0$, ряд расходится, и $i$ возвратно.

Таким образом, для общих цепей:

$$\text{невозвратное} \Rightarrow \text{нулевое}, \qquad \text{ненулевое} \Rightarrow \text{возвратное}.$$

Для счётных цепей верны и обратные импликации.

### Теорема 6.2 (Несущественное $\Rightarrow$ невозвратное)

Если состояние $i$ несущественно, то оно невозвратно.

Доказательство (идея). По определению несущественности, существует $j$ такое, что $i \to j$ и $j \not\to i$. Значит, попав в $j$, система уже никогда не вернётся в $i$, а это происходит с положительной вероятностью. Поэтому вероятность никогда не вернуться в $i$ строго положительна, откуда $f_{ii} < 1$. $\square$

Цепочка следствий:

$$\text{несущественное} \Rightarrow \text{невозвратное} \Rightarrow \text{нулевое}.$$

Контрапозитивно:

$$\text{ненулевое} \Rightarrow \text{возвратное} \Rightarrow \text{существенное}.$$

## 7. Неразложимые цепи

### Определение 7.1 (Неразложимая цепь)

Цепь Маркова называется **неразложимой**, если все состояния сообщаются: $i \leftrightarrow j$ для любых $i, j \in E$ (цепь состоит из одного класса эквивалентности).

### Теорема 7.1 (Солидарность)

Пусть цепь Маркова неразложима. Тогда:

а) Если одно состояние **возвратно**, то все состояния возвратны.

б) Если одно состояние **нулевое**, то все состояния нулевые.

в) Если одно состояние имеет **период** $d$, то все состояния имеют период $d$.

Доказательство (идея). В неразложимой цепи из любого $i$ можно попасть в $j$ и наоборот. Пусть $i \to j$ за $m$ шагов и $j \to i$ за $l$ шагов. Тогда $p_{jj}^{(n)} \geq p_{ji}^{(l)} p_{ii}^{(n)} p_{ij}^{(m)}$. Если $\sum p_{ii}^{(n)} = \infty$, то и $\sum p_{jj}^{(n)} = \infty$, т.е. $j$ тоже возвратно. Аналогичные рассуждения применимы к нулевости и периоду. $\square$

### Теорема 7.2 (Конечная неразложимая цепь)

В **конечной неразложимой** цепи Маркова все состояния **ненулевые** (а значит, возвратные и существенные).

Следствие. Для конечных цепей понятия «существенное», «возвратное» и «ненулевое» равносильны.

## 8. Стационарное распределение

### Определение 8.1 (Стационарное распределение)

Вектор вероятностей $\boldsymbol{\pi} = (\pi_1, \pi_2, \ldots)^\top$, $\pi_i \geq 0$, $\sum_i \pi_i = 1$, называется **стационарным распределением** цепи Маркова, если

$$P^\top \boldsymbol{\pi} = \boldsymbol{\pi},$$

то есть $\boldsymbol{\pi}$ — собственный вектор матрицы $P^\top$, отвечающий собственному значению 1.

МОТИВАЦИЯ: Если начальное распределение совпадает со стационарным ($\mathbf{p}(0) = \boldsymbol{\pi}$), то $\mathbf{p}(n) = \boldsymbol{\pi}$ для всех $n$ — распределение не меняется со временем.

ИНТУИЦИЯ: Представьте, что система «гуляет» по цепи очень долго. Может случиться, что распределение вероятностей стабилизируется. Это равновесное («стационарное») распределение.

### Теорема 8.1 (Существование стационарного распределения)

У любой **конечной** цепи Маркова существует хотя бы одно стационарное распределение.

Доказательство (идея). Матрица $P^\top$ является стохастической (её строки суммируются в 1). По теореме Брауэра о неподвижной точке (или теореме Перрона–Фробениуса для неотрицательных матриц), у неё есть собственное значение 1, а соответствующий собственный вектор с неотрицательными компонентами, нормированный, и даёт $\boldsymbol{\pi}$. $\square$

## 9. Среднее время первого возврата

### Определение 9.1

Пусть $\tau_i = \min\{n \geq 1 : X_n = i\}$ — **момент первого посещения** состояния $i$.

Введём обозначение

$$\mu_{ij} = E[\tau_j \mid X_0 = i]$$

— **среднее время первого прихода** из $i$ в $j$. При $i = j$ это **среднее время первого возврата** в $i$.

### Теорема 9.1 (Связь среднего времени возврата со стационарным распределением)

Пусть цепь эргодична (имеет единственное стационарное распределение). Тогда

$$\mu_{ii} = \frac{1}{\pi_i}.$$

Доказательство. Запишем рекуррентное соотношение для $\mu_{ij}$:

$$\mu_{ij} = \sum_{k \neq j} p_{ik} (\mu_{kj} + 1) + p_{ij} \cdot 1.$$

(По формуле полного математического ожидания: с вероятностью $p_{ik}$ делаем первый шаг в $k \neq j$ и дальше ждём среднее время $\mu_{kj}$; с вероятностью $p_{ij}$ сразу попадаем в $j$ за 1 шаг.)

Умножим это соотношение на $\pi_i$ и просуммируем по всем $i$:

$$\sum_i \pi_i \mu_{ij} = \sum_i \pi_i \sum_{k \neq j} p_{ik} (\mu_{kj} + 1) + \sum_i \pi_i p_{ij}.$$

Правая часть преобразуется:

$$= \sum_{k \neq j} \mu_{kj} \underbrace{\sum_i \pi_i p_{ik}}_{= \pi_k} + \sum_{k \neq j} \underbrace{\sum_i \pi_i p_{ik}}_{= \pi_k} + \underbrace{\sum_i \pi_i p_{ij}}_{= \pi_j}.$$

Здесь мы использовали условие стационарности: $\sum_i \pi_i p_{ik} = \pi_k$. Поэтому:

$$\sum_i \pi_i \mu_{ij} = \sum_{k \neq j} \pi_k \mu_{kj} + \sum_{k \neq j} \pi_k + \pi_j.$$

Перенесём $\sum_{k \neq j} \pi_k \mu_{kj}$ влево:

$$\pi_j \mu_{jj} = \sum_{k \neq j} \pi_k + \pi_j = \sum_k \pi_k = 1.$$

Откуда $\mu_{jj} = \dfrac{1}{\pi_j}$. $\square$

ИНТУИЦИЯ: Чем больше $\pi_i$, тем чаще система бывает в состоянии $i$, и тем короче среднее время ожидания возврата в него. Формула $\mu_{ii} = 1/\pi_i$ точно отражает эту обратно пропорциональную зависимость.

## 10. Эргодические цепи и критерии сильной эргодичности

### Определение 10.1 (Эргодическая цепь)

Цепь Маркова называется **эргодической**, если существует единственное стационарное распределение $\boldsymbol{\pi}$.

### Определение 10.2 (Сильно эргодическая цепь)

Цепь называется **сильно эргодической**, если

$$\lim_{n \to \infty} p_{ij}^{(n)} = \pi_j^* > 0 \quad \text{для всех } i, j \in E.$$

То есть независимо от начального состояния, через достаточно большое время вероятность оказаться в $j$ стремится к одному и тому же положительному числу $\pi_j^*$.

Замечание. Можно доказать, что $\boldsymbol{\pi}^* = \boldsymbol{\pi}$ — предел совпадает с (единственным) стационарным распределением.

### Теорема 10.1 (Критерии сильной эргодичности)

Следующие утверждения являются критериями сильной эргодичности конечной цепи Маркова:

**Критерий а.** Конечная цепь Маркова **сильно эргодична** тогда и только тогда, когда она **неразложима и апериодична**.

**Критерий б.** Конечная цепь Маркова сильно эргодична тогда и только тогда, когда существует $n_0$ такое, что для всех $i, j$:

$$p_{ij}^{(n_0)} > 0.$$

То есть матрица $P^{n_0}$ имеет все строго положительные элементы.

**Критерий в.** Цепь Маркова (в том числе счётная) сильно эргодична, если она **неразложима, апериодична, и все состояния ненулевые**.

**Критерий г.** Цепь Маркова сильно эргодична тогда и только тогда, когда она **апериодична и эргодична** (т.е. апериодична и имеет единственное стационарное распределение $\boldsymbol{\pi}$ с $\pi_j > 0$ для всех $j$).

СВЯЗИ: Критерий б (полная связность за $n_0$ шагов) является самым «прямым» и часто удобен для проверки на практике. Критерий а более структурен и требует проверки двух свойств. Критерии в и г обобщают на счётный случай.

### Утверждение 10.1

Если состояние $i$ является **нулевым**, то $\pi_i = 0$.

ИНТУИЦИЯ: Нулевое состояние обладает нулевой долей в стационарном распределении, ведь вероятность возврата в него убывает к нулю, и «в среднем» система всё меньше времени проводит в $i$.

## 11. Эргодическая теорема

### Теорема 11.1 (Эргодическая теорема)

Пусть цепь Маркова эргодична. Введём случайную величину

$$V_i(N) = \sum_{k=1}^{N} \mathbf{1}[X_k = i]$$

— число посещений состояния $i$ за время $N$. Тогда

$$P\!\left(\lim_{N \to \infty} \frac{V_i(N)}{N} = \pi_i\right) = 1.$$

МОТИВАЦИЯ: Теорема даёт статистический смысл стационарному распределению: $\pi_i$ — это **доля времени**, которую система проводит в состоянии $i$ на длинном горизонте.

ИНТУИЦИЯ: Запустите частицу гулять по цепи. Посчитайте, сколько раз за $N$ шагов она была в состоянии $i$. При $N \to \infty$ это число, делённое на $N$, сходится почти наверное к $\pi_i$. Таким образом, стационарное распределение можно «измерить» экспериментально.

СВЯЗИ: Аналог закона больших чисел для марковских цепей. Применяется в методе Монте-Карло по Маркову (MCMC), лежащем в основе байесовской статистики и машинного обучения.

## Концептуальное резюме

Марковские цепи — модель случайных процессов с «короткой памятью», где вся история сворачивается в текущее состояние. Однородность упрощает анализ до изучения степеней одной матрицы. Состояния делятся по достижимости на классы эквивалентности, среди которых выделяют замкнутые (существенные, возвратные) и открытые (несущественные, невозвратные, нулевые). Периодичность характеризует «ритм» возвращения в состояние. Ключевой объект долгосрочного анализа — стационарное распределение $\boldsymbol{\pi}$, к которому сходится система при эргодичности; оно одновременно является обратной величиной среднего времени первого возврата и предельной долей времени в каждом состоянии.

## Связи с более широкой математикой

- **Линейная алгебра**: переходная матрица — стохастическая, её анализ опирается на теорему Перрона–Фробениуса; возведение в степень через жорданову нормальную форму даёт явные формулы для $p_{ij}^{(n)}$.
- **Теория графов**: классификация состояний — это теория сильной связности ориентированных графов.
- **Численные ряды**: критерий возвратности выражается через сходимость/расходимость ряда $\sum p_{ii}^{(n)}$.
- **MCMC**: алгоритмы Метрополиса–Гастингса, Гиббса используют марковские цепи для сэмплирования из сложных распределений.
- **PageRank**: алгоритм Google — стационарное распределение специальной марковской цепи на графе веб-страниц.
- **Теория массового обслуживания**: цепи Маркова моделируют очереди, смертность-рождаемость и другие прикладные системы.
