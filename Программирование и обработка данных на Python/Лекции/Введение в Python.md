Python — это **интерпретируемый, динамически типизированный скриптовый язык**, разработанный в 1989 году как "шуточный проект" энтузиастами. Его философия заключается в максимальном упрощении синтаксиса для облегчения работы и автоматизации "скучных вещей", делая его доступным даже для новичков. Он считается **самым простым языком для изучения** среди языков программирования.

Вот основные аспекты Python:

- **Природа языка**
    
    - **Интерпретируемый язык**: Код Python выполняется построчно интерпретатором, а не компилируется целиком перед запуском, как в C++ или Java. Эта особенность позволяет работать в интерактивном режиме (REPL), где можно вводить код построчно и сразу получать результат.
    - **Динамически типизированный**: Python автоматически определяет и управляет типами данных во время выполнения программы, не требуя явного объявления типа переменной. Это даёт гибкость, но может замедлять выполнение, так как расчёт выделения памяти происходит "на лету".
    - **Развитие сообществом**: Развитие Python управляется его сообществом, без какого-либо "патрона" или "большого брата", что позволило ему занять уникальную нишу.
    - **Версии**: В рамках курса будет использоваться **Python 3**, так как Python 2 считается устаревшим.
- **Работа с кодом и среда разработки**
    
    - **Интерактивный режим (REPL)**: Можно открыть терминал, набрать `python` и вводить код построчно для немедленного получения результатов. Например, `2 + 2` сразу выведет `4` без использования `print()`. Этот режим будет часто использоваться для проверки функций и внутренней логики.
    - **Среда разработки**: Для написания кода на лекциях будет использоваться PyCharm, хотя можно использовать и VS Code.
- **Базовый синтаксис и операции**
    
    - **Выражения (Expressions)**: Любая операция, выполненная с помощью математических операторов, например, `2 + 2`, называется выражением. Даже отдельное значение, например `2`, может быть выражением.
    - **Значения (Values) и Операторы (Operators)**: Выражения состоят из значений (числа, строки) и операторов (`+`, `-`, `*` и т.д.).
    - **Математические операторы**:
        - Сложение (`+`)
        - Вычитание (`-`)
        - Умножение (`*`)
        - Деление (`/`)
        - Взятие остатка от деления (`%`)
        - Возведение в степень (`**`)
        - Целочисленное деление (`//`)
    - **Порядок выполнения операций**: Python соблюдает привычный математический порядок операций, и для изменения приоритета можно использовать скобки.
    - **Вывод информации**: Функция `print()` используется для вывода текста в консоль, например, `print("Hello World")`.
    - **Пробелы и табуляция**: Python **чувствителен к табуляции (отступам)** для определения блоков кода (например, внутри `if` или циклов), но **нечувствителен к пробелам внутри выражений** (например, `2+2` и `2 + 2` равнозначны). Строгие правила табуляции призваны обеспечить читаемость и "красоту" кода.
- **Примитивные типы данных**
    
    - **Числа**:
        - **Целые (Integers)**: Числа без десятичной точки.
        - **Числа с плавающей точкой (Floats)**: Числа с десятичной точкой.
    - **Строки (Strings)**: Последовательности символов, заключённые в одинарные или двойные кавычки. Даже один символ считается строкой.
        - **Конкатенация**: Оператор `+` используется для объединения строк (например, `"Hello" + " World"` даст `"Hello World"`).
        - **Умножение**: Строку можно умножить на целое число для её повторения (например, `"Hello" * 4` даст `"HelloHelloHelloHello"`).
        - **Совместимость типов**: Нельзя напрямую складывать строку и число без предварительного преобразования числа в строку.
    - **Булевы значения (Booleans)**: `True` или `False`, используемые в условных конструкциях. Они могут быть присвоены переменным и неявно преобразуются в `1` или `0` соответственно.
- **Переменные**
    
    - **Объявление и присваивание**: Переменные создаются и им присваиваются значения с помощью оператора `=` (например, `a = 5`).
    - **Организация памяти**: Память в Python состоит из "стека" (где хранятся имена переменных) и "кучи" (где хранятся их значения).
        - Присваивание переменной значения (например, `b = a`) часто создаёт новую ссылку на уже существующее значение в "куче", а не дублирует его, что оптимизирует использование памяти.
        - Для оптимизации производительности, значения от 1 до 100 предварительно выделяются и постоянно хранятся в "куче" при запуске интерпретатора.
    - **Соглашения по именованию**: Переменные следует называть осмысленно. Рекомендуется использовать **Snake Case** (например, `my_variable_name`), где слова разделяются нижним подчёркиванием, вместо Camel Case (например, `myVariableName`). Важно следовать общим правилам структурирования кода (например, из репозитория Pythons).
- **Управляющие конструкции**
    
    - **Условные операторы (Ветвления)**:
        - **`if`**: Выполняет блок кода, если условие истинно (`True`).
        - **`elif`** (сокращение от "else if"): Предоставляет дополнительные условия для проверки, если предыдущие `if` или `elif` были ложными.
        - **`else`**: Выполняет блок кода, если все предыдущие `if` и `elif` условия были ложными.
        - **Синтаксис**: `if условие: # блок кода`, `elif другое_условие: # блок кода`, `else: # блок кода`. Блоки кода определяются отступами.
    - **Операторы сравнения**: Используются в условиях для сравнения значений: `==` (равно), `!=` (не равно), `>` (больше), `<` (меньше), `>=` (больше или равно), `<=` (меньше или равно).
    - **Циклы (Повторения)**:
        - **`while` цикл**: Выполняет блок кода многократно, пока заданное условие остаётся истинным (`True`). Например, `while условие: # блок кода`. `while True:` создаст бесконечный цикл.
        - **`for` цикл**: Предназначен для итерации по элементам **коллекции (итерируемого объекта)**.
            - Он присваивает элементы из коллекции переменной цикла (например, `i`) на каждом шаге.
            - Функция **`range()`** часто используется для генерации последовательности чисел (например, `range(10)` генерирует числа от 0 до 9), чтобы цикл выполнился определённое количество раз.
            - **Синтаксис**: `for переменная_цикла in коллекция: # блок кода`.
