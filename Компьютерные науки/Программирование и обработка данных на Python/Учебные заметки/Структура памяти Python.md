Есть [[stack]] и [[heap]].
В куче находятся значения, которые мы даем переменным. 
По умолчанию в куче находятся числа от 0 до 100
Когда мы присваиваем значение переменной, создается ссылка на значение в куче.

```python
a = 1
b = a
```

a и b ссылаются на 1 объект(значение) в памяти компьютера
