### **Построение интерполяционного многочлена Лагранжа**

**Суть метода:** Интерполяция заключается в приближении сложной функции многочленом так, чтобы в заданных точках (**узлах интерполяции**) значения многочлена и функции полностью совпадали. Между узлами значения могут расходиться, и эта разность называется ошибкой или остаточным членом $R_n(x)$.

#### **Задача: Построение многочлена для $f(x) = \cos(x) \cdot \sin(x) \cdot x^2$**

**Условия:**

- **Функция:** $f(x) = \cos(x) \sin(x) x^2 = \frac{1}{2} \sin(2x) \cdot x^2$.
- **Отрезок:** $[-1, 2]$.
- **Количество узлов:** $n=4$ (равномерная сетка).

#### **Шаг 1: Определение узлов сетки**

Для 4-х точек на отрезке $[-1, 2]$ с равным шагом узлы будут следующими:

- $x_0 = -1$
- $x_1 = 0$
- $x_2 = 1$
- $x_3 = 2$

#### **Шаг 2: Вычисление значений функции в узлах**

Вычисляем $y_k = f(x_k)$:

1. $f(-1) = \cos(-1) \sin(-1) (-1)^2 = -\cos(1)\sin(1) = \mathbf{-0.5 \sin(2)}$.
2. $f(0) = \cos(0) \sin(0) (0)^2 = \mathbf{0}$.
3. $f(1) = \cos(1) \sin(1) (1)^2 = \mathbf{0.5 \sin(2)}$.
4. $f(2) = \cos(2) \sin(2) (2)^2 = 0.5 \sin(4) \cdot 4 = \mathbf{2 \sin(4)}$.

#### **Шаг 3: Построение базисных полиномов $L_k(x)$**

Общая формула базисного полинома: $L_k(x) = \frac{\omega_k(x)}{\omega_k(x_k)}$, где $\omega_k$ — произведение разностей $(x - x_i)$ для всех $i \neq k$.

- Для **$x_0 = -1$**: $\omega_0(x) = x(x-1)(x-2)$. Знаменатель: $(-1)(-1-1)(-1-2) = -6$. $L_0(x) = \frac{x(x-1)(x-2)}{-6}$.
- Для **$x_1 = 0$**: $L_1$ не потребуется в итоговой сумме, так как $f(x_1)=0$.
- Для **$x_2 = 1$**: $\omega_2(x) = (x+1)x(x-2)$. Знаменатель: $(1+1)(1)(1-2) = -2$. $L_2(x) = \frac{(x+1)x(x-2)}{-2}$.
- Для **$x_3 = 2$**: $\omega_3(x) = (x+1)x(x-1)$. Знаменатель: $(2+1)(2)(2-1) = 6$. $L_3(x) = \frac{(x+1)x(x-1)}{6}$.

#### **Шаг 4: Сборка многочлена Лагранжа $Q_3(x)$**

Многочлен представляет собой сумму $Q_3(x) = \sum_{k=0}^{3} f(x_k) L_k(x)$: $$Q_3(x) = \left( \frac{x(x-1)(x-2)}{-6} \right) \cdot (-0.5 \sin 2) + 0 + \left( \frac{(x+1)x(x-2)}{-2} \right) \cdot (0.5 \sin 2) + \left( \frac{(x+1)x(x-1)}{6} \right) \cdot (2 \sin 4)$$.

#### **Шаг 5: Проверка значения в точке $x = 4/3$**

Для оценки точности приближения на семинаре было предложено подставить значение $x = 4/3 \approx 1.33$ в полученный многочлен и сравнить с реальным значением функции $f(4/3)$.

- При расчете для 4-х точек значение многочлена составило примерно **0.389** (или 0.406 по разным прикидкам), в то время как реальное значение функции — около **0.18**.
- **Вывод:** Использование всего 4-х точек на таком отрезке дает существенную погрешность. Увеличение количества узлов (например, до 7 или 8) должно улучшить точность, что и предлагается проверить в рамках домашнего задания.

**Дополнительное замечание:** В конце занятия обсуждалось, что **неравномерная сетка (узлы Чебышёва)** зачастую позволяет минимизировать ошибку интерполяции эффективнее, чем равномерная, особенно при увеличении степени многочлена.
